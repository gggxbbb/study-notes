## 一、呼吸系统

### 1. 疾病鉴别诊断

| 疾病 | 核心特征 | 关键线索 |
|------|----------|----------|
| 气管异物 | 突发吸气性呼吸困难、三凹征、发绀、哮鸣音、呼吸音减弱 | 异物吸入史 |
| 自发性气胸 | 突发呼吸困难、胸痛、胸廓隆起、三凹征、发绀、鼓音 | 突发胸痛 |
| 支气管哮喘 | 反复发作性胸闷、呼气性呼吸困难、夜间/凌晨加重、哮鸣音 | 幼年过敏史、皮疹史 |
| 左心衰肺水肿 | 活动后呼吸困难、端坐呼吸、粉红泡沫痰、双肺湿罗音 | 心脏病史 |
| 肺栓塞 | 突发胸痛、咯血、呼吸困难 | 长期卧床史 |
| 咳嗽变异性哮喘 | 仅有咳嗽、抗生素治疗无效、检查无异常 | 夜间加重、气道高反应性 |

### 2. 呼吸困难类型与气道阻塞部位

- **吸气性呼吸困难**：大气道阻塞
  - 吸气时气体先经过大气道，大气道阻塞导致吸气困难
- **呼气性呼吸困难**：小气道阻塞
  - 呼气时气体先经过小气道，小气道阻塞导致呼气困难

### 3. 听诊体征

- **湿罗音** — 气道分泌物
- **哮鸣音** — 平滑肌收缩

### 4. 哮喘发病机制

#### 本质、机制、病理
- **本质**：气道慢性炎症
- **机制**：气道高反应性
- **病理**：气道重构

#### 细胞学基础

| 细胞类型 | 关联类型 | 机制/学说 |
|----------|----------|-----------|
| 嗜碱性粒细胞 | 早发性哮喘 | 变态反应学说：与抗体结合，释放组胺 |
| 嗜酸性粒细胞 | 迟发性哮喘 | 哮喘本质 — 慢性气道炎症 |
| 中性粒细胞 | COPD | COPD 的主要效应细胞 |

#### 治疗原则（记忆口诀）
- **控制/最有效/最佳** → 激素（糖皮质激素）
- **急性发作** → 沙丁胺醇（β2受体激动剂）

#### 诊断与检查分层

| 检查项目 | 用途 | 临床意义 |
|----------|------|----------|
| 肺功能检查 | 金标准 | 用于鉴别诊断 |
| 动脉血气分析 | 重症评估 | 用于分级治疗 |

**检查选择逻辑（老贺口诀）**
- 有舒张选舒张（支气管舒张试验）
- 没舒张选激发（支气管激发试验）
- 意识障碍选血气分析

**两种试验的用途与鉴别逻辑**
- 有病（已有症状）→ 选支气管舒张试验 → 测定气道可逆性受限 → 鉴别哮喘与 COPD（最大不同：可逆性）
- 无病（无典型症状）→ 选支气管激发试验 → 测定气道高反应性 → 确诊哮喘

**临床快速判断**
- 哮喘患者神志清醒 → 可排除二氧化碳潴留（二氧化碳潴留会导致意识障碍）


#### 不同阶段酸碱失衡

| 阶段 | 酸碱状态 | 机制 |
|------|----------|------|
| 早期 | 呼吸性碱中毒（呼碱） | 紧张+呼吸频率代偿性增快（气道痉挛→潮气量↓），CO₂扩散快、呼出过多 |
| 重症 | 呼吸性酸中毒（呼酸） | 气道过度痉挛，通气严重受阻，CO₂潴留 |
| 晚期 | 代谢性酸中毒（代酸） | 缺氧时间长，酸性代谢产物堆积 |

#### 哮喘严重程度分级

| 分级 | 临床表现 | 关键体征/指标 |
|------|----------|---------------|
| 轻度 | 上楼困难 | — |
| 中度 | 心率 100~120 次/分，三凹征，奇脉 | — |
| 重度 | 心率＞120 次/分，端坐呼吸，奇脉，Ⅱ型呼衰 | — |
| 危重 | 意识障碍，胸腹矛盾运动，沉默肺，Ⅱ型呼衰 | 沉默肺 = 气道极度痉挛，气流几乎无法通过 |

#### 哮喘治疗药物（抗炎类药物口诀）

**抗炎类药物记忆口诀：白糖酮色**
- 白三烯
- 糖皮质激素
- 酮替芬
- 色甘酸钠

**药物特点与记忆窍门**
- **预防色狼** → 色甘酸钠，用于预防发作
- **季节交替** → 酮替芬，对季节性发作有效

### 5. 肺功能检查

**阻塞性通气障碍**
- 本质：气道问题，通不了气
- 机制：小气道提前关闭
- 表现：呼气性呼吸困难
- 见于：慢阻肺、支气管哮喘
- 特点：带 F 的指标均↓，例外 VC 也↓，其余↑
- **记忆窍门**：F = 流速，阻塞时流速下降，所以有 F 的指标均下降

**FVC（用力肺活量）定义**：尽力最大吸气后，尽力尽快呼气所能呼出的最大气量

**限制性通气障碍**
- 本质：胸廓问题，肺扩张受限
- 机制：肺部外部受压不能完全舒张
- 见于：气胸、胸腔积液、胸廓病变、肌肉/神经病变、肺纤维化
- 特点：两个比值↑或正常，其他均↓

### 6. 氧疗原则

| 疾病/情况 | 氧疗方式 |
|-----------|----------|
| 慢性支气管炎、COPD | 持续低流量吸氧 |
| ARDS | 呼气末正压通气（PEEP） |
| 气性坏疽、CO 中毒 | 高压氧疗 |
| 肺炎 | 无控制给氧 |

### 7. 肺功能检查指标解读

| 检查/指标 | 结果 | 临床意义 |
|-----------|------|----------|
| 支气管舒张试验 | 阳性 | 提示存在可逆性气道阻塞 |
| 功能残气量（FRC） | 显著增加 | 表明肺过度充气，非特异性 |
| 支气管激发试验 | 阳性 | 提示存在气道高反应性 |
| 限制性通气障碍 + 气体交换障碍 | 阳性 | 间质性肺疾病的特征性表现 |

### 8. 病理生理与体征机制

#### 吸气位与呼吸活动度
- 机制：呼气困难 → 气体难以快速呼出 → 胸廓呈吸气后饱满扩张状态
- 表现：呼吸动度变小，呈吸气位，可有三凹征

#### 吸气性 vs 呼气性呼吸困难
- 仅支气管异物 → 吸气性呼吸困难
- 其余疾病 → 呼气性呼吸困难
- **支哮记忆**：呼不出来气（呼气困难）就是哮喘

#### 重症哮喘三凹征机制
- 气道痉挛 → 呼气困难 → 呼气延长
- 缺氧 → 呼吸频率加快 → 吸气缩短
- 吸气肌过度用力 → 三凹征

#### 语音震颤（语音共振）
- 增强：肺实变（大叶性肺炎）
- 减弱：气体积滞（呼气困难）、口诀「气水堵厚弱」
  - 气：气胸
  - 水：胸腔积液
  - 堵：气道阻塞
  - 厚：胸膜增厚
  - 弱：肺气肿/肺组织减弱

#### 强迫体位
- 端坐呼吸：膈肌下降 → 肺通气功能增加

#### 反常运动鉴别
- **胸腹反常运动** → 重度哮喘（呼吸肌疲劳，胸腹运动不协调）
- **胸壁反常运动** → 多根多处肋骨骨折（局部胸壁软化，吸气时内陷、呼气时外凸）

### 9. 叩诊音

| 情况 | 叩诊音 |
|------|--------|
| 正常肺脏 | 清音 |
| 肺炎 | 浊音或实音 |
| 气胸 | 鼓音 |
| 哮喘 | 过清音 |

**记忆窍门**
- 肺内气多 → 过清音
- 肺外气多 → 鼓音
- 肺内水多 → 浊音
- 肺外水多 → 实音

### 11. 疾病诊断公式

#### 急性上呼吸道感染
- 咽痛 + 咳嗽 + 发热

#### 肺炎

| 类型 | 诊断公式 | 关键特征 |
|------|----------|----------|
| 大叶性肺炎 | 成人 + 受凉 + 高热 + 咳铁锈色痰 | 铁锈色痰 |
| 克雷白杆菌肺炎 | 咳砖红色痰 + X线片空洞 | 砖红色痰 |
| 支原体肺炎 | 儿童 + 刺激性干咳 + 关节痛 + 抗生素无效 | 刺激性干咳 |
| 支气管肺炎 | 婴幼儿 + 发热 + 呼吸困难（鼻翼煽动、三凹征） | 婴幼儿 |
| 金黄色葡萄球菌肺炎 | 高热 + 胸痛 + 脓血痰 + X线片状影 | 脓血痰 |

#### 结核病

| 类型 | 诊断公式 | 关键特征 |
|------|----------|----------|
| 肺结核 | 青壮年 + 咯血 + 午后低热 + 夜间盗汗 + 抗生素无效 | 午后低热、盗汗 |
| 结核性胸膜炎 | 结核 + 胸膜积液（胸痛 + 语颤消失 + 叩诊实音/呼吸音消失） | 胸膜积液体征 |
| 结核性心包炎 | 结核 + 心包积液（心前区痛 + 呼吸困难 + 上腹部闷胀 + 下肢浮肿） | 心包积液 |
| 肠结核 | 结核 + 腹部症状（腹痛、腹泻、右下腹肿块） | 右下腹肿块 |
| 结核性腹膜炎 | 结核 + 腹部炎症（腹痛、腹泻、腹壁柔韧感） | 腹壁柔韧感 |
| 肾结核 | 结核 + 膀胱刺激征 + 肾实质变薄并有破坏 | 膀胱刺激征 |

#### 支气管扩张
- 童年麻疹/百日咳/支气管肺炎迁延不愈史 + 咳嗽 + 脓痰 + 咯血

#### COPD
- 老年人（吸烟史）+ 咳、痰、喘 + 桶状胸 + 肺功能检查（FEV1/FVC < 70%）

#### 肺脓肿
- 脓臭痰

## 二、循环系统

### 1. 肺循环 vs 体循环

| 循环类型 | 压力 | 阻力 | 容量 |
|----------|------|------|------|
| 肺循环 | 低 | 低 | 高 |
| 体循环 | 高 | 高 | 高 |

- 肺循环：**两低一高**（低压、低阻、高容）
- 体循环：**三高**（高压、高阻、高容）

## 三、跨系统横向梳理

### 1. 夜间加重的疾病（记忆口诀）

**夜间加重——变异型心绞痛、急性左心衰（夜间阵发性呼吸困难）、支气管哮喘、痛风性关节炎、前列腺增生**

---

### 2. 病情加重时症状减轻（危险信号）

| 疾病 | 病情进展 | 表现 | 机制 |
|------|----------|------|------|
| 左心衰 | 发展为全心衰 | 憋喘减轻 | 全心衰后肺淤血减轻 |
| 重症哮喘 | 病情加重 | 哮鸣音减轻（沉默肺） | 气道严重阻塞，气流无法通过 |
| 心包炎 | 产生大量积液 | 胸痛减轻 | 心包张力降低 |
| 二尖瓣狭窄 | 后期 | 咯血减轻 | 病变进展后期 |
| 二尖瓣狭窄（重度） | 重度 | 杂音减小（安静型二狭） | 瓣口极度狭窄，血流量减少 |
| 阑尾炎/绞窄疝 | 穿孔 | 腹痛暂时减轻 | 腔内压力骤减，但随后腹膜炎加重 |
