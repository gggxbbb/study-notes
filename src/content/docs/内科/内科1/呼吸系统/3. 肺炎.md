# 3. 肺炎

## 诊断公式

| 类型 | 诊断公式 | 关键特征 |
|------|----------|----------|
| 大叶性肺炎 | 成人 + 受凉 + 高热 + 咳铁锈色痰 | 铁锈色痰 |
| 克雷白杆菌肺炎 | 咳砖红色痰 + X线片空洞 | 砖红色痰 |
| 支原体肺炎 | 儿童 + 刺激性干咳 + 关节痛 + 抗生素无效 | 刺激性干咳 |
| 支气管肺炎 | 婴幼儿 + 发热 + 呼吸困难（鼻翼煽动、三凹征） | 婴幼儿 |
| 金黄色葡萄球菌肺炎 | 高热 + 胸痛 + 脓血痰 + X线片状影 | 脓血痰 |
