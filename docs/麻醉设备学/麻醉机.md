# 麻醉机

## 工作流程与功能单元

麻醉机的结构划分有两种方式：**工作流程划分**和**功能单元划分**。

**麻醉机发展过程**：弱效麻醉气体（压缩气体）→ 强效麻醉气体 → 麻醉蒸发器 → 循环回路 → 麻醉呼吸机

## 蒸发器

- **饱和蒸气压**：与液体种类有关，与液体量无关；温度升高则饱和蒸气压增大
- **强效吸入麻醉药特点**：汽化热小、沸点低、饱和蒸气压高
- **蒸发器材料要求**：比热大、导热率大
- **蒸发器没有关闭位**（任何蒸发器在关闭位时进口与出口间气流切断 → ❌）
- 蒸发器输出浓度与**温度补偿**相关
- 蒸发器输出浓度 **大于** 患者实际吸入浓度（ACGO输出气体部分用于冲洗回路CO₂，稀释了吸入浓度）

### 泵效应（Pumping Effect）

吸气期破坏了蒸发器正常的分流比，导致输出浓度升高。
- 缩短蒸发室输入通道 → 泵效应**增加**
- 在蒸发器出口处安装防逆活瓣可减少泵效应
- 入口处加长螺旋管道也可缓解

### 压力效应（Pressure Effect）

主要由**快速充氧开关**引起——大量氧气快速输出，稀释了混合气体，使输出浓度降低。

### 蒸发器类型

- **回路内蒸发器**：流阻小
- **回路外蒸发器**：输出浓度不受患者呼吸气流影响、少受间歇逆压影响、不需清洗消毒

影响蒸发器输出浓度的因素：温度、震荡、麻醉药量、新鲜气体流量、大气压（环境气压降低→浓度升高，环境气压增高→浓度降低）。**相对湿度不影响**输出浓度。

## 流量控制系统

主要功能：
- 控制释放到麻醉蒸发器和麻醉回路的新鲜气体成分和流量
- 显示新鲜气体流量
- 根据需要为麻醉回路快速提供新鲜氧气
- 防止低氧混合气体的形成与输出

**不属于**流量控制的功能：显示麻醉药蒸气（属气体监测系统）、麻醉深度监测、二氧化碳吸收

## 共同气体出口（Common Gas Outlet, CGO/ACGO）

- **防逆流活瓣**安放位置：蒸发器与新鲜气体出口之间
- 作用：防止麻醉回路内气体倒流；有利于流量计显示稳定

## 麻醉回路

### Mapleson系统

| 回路 | 自主呼吸 | 控制呼吸 | 结构要点 |
|------|----------|----------|----------|
| **A（Magill）** | ✅ 效率高 | ❌ 效率低 | 新鲜气体入口近患者端，APL阀在面罩端 |
| **B** | 中等 | 中等 | 新鲜气体入口近患者端，呼出气自远端排出 |
| **C** | 中等 | 中等 | 类似B但加储气囊 |
| **D** | ❌ 效率低 | ✅ 效率高 | 新鲜气体入口近患者端，呼出管远端排气 |
| **E（T管/Ayre管）** | 用于小儿 | — | T型管结构，无储气囊、无活瓣 |
| **F（Jackson-Rees）** | 用于小儿 | — | E管+储气囊 |

- Mapleson系统依靠新鲜气体冲洗CO₂，重复吸入量取决于新鲜气流量
- 结构简单、阻力小、死腔小，但新鲜气流量消耗大
- 可在半开放、半紧闭、紧闭之间游走

### 循环回路

**组成**：单向活瓣、两根波纹管（吸气臂+呼气臂）、CO₂吸收罐、APL阀、储气囊

**优点**：
- 可进行辅助或控制呼吸
- 可保持吸入气体温湿度接近生理状态
- 麻醉药可再利用

**缺点**：
- 阻力大（多个阀门、波纹管）
- 小儿麻醉存在隐患
- 连接易脱落泄漏
- 有交叉感染风险

**半紧闭吸入麻醉特点**：通气量 > 麻醉气体消耗量（耗气量）

## CO₂吸收装置

- **钠石灰（Soda Lime）**：关键吸收剂，主要成分**不是 NaOH**（名称有迷惑性）
- 吸收CO₂后变化：变白、变硬、产热 → 产生CaCO₃
- **钡石灰**可与七氟醚反应，有起火事故风险
- 呼出气流向：呼气活瓣 → CO₂吸收罐 → 贮气囊

## 安全装置

### 笑气/氧气比例控制三种机制

| 机制 | 阀门特点 |
|------|----------|
| **25%联动比例限定控制** | 氧气阀与笑气阀联动，保证输出氧浓度≥25% |
| **氧气比例监测控制器** | 氧气阀和笑气阀分开独立调节，纯气动原理 |
| **定量混合气体流量计** | 只有总流量阀和氧浓度阀（参比调节阀+笑气调节阀） |

### 低氧压氧化亚氮安全切断阀

⚠️ **不能防止低氧混合气体输出！**

原理：纯气动——氧压低时笑气也随之降低，混合比例不变。仅在没有氧气压力时切断笑气。

### 阀门开关方向

| 阀门类型 | 打开方向 |
|----------|----------|
| **减压调节阀** | **顺时针→打开**（唯一特例） |
| 蒸发器补液装置 | 逆时针→打开 |
| 蒸发器可变分流控制阀 | 逆时针→打开 |
| APL阀 | 逆时针→打开 |

**注意**：阀只能调节下游压力大小，不能调节高压。

### 快速充氧开关

- **没有关闭位**（瞬间开闭，不存在微开状态）
- 一定是**手动**，不能自动
- 关闭不严 → 持续供氧稀释混合气体 → **麻醉过浅**（不是过深）
- 开关 ≠ 阀（开关只有OPEN/CLOSE，不能调节程度）

### PISS（定位销安全指示系统）

- 仅适用于麻醉机上**小型储气钢瓶**
- 防止气源供应错误，也可防止AGCO输出错误
- 四种防错机制：①钢瓶标识标准 ②PISS ③DISS ④NIST

## 储气钢瓶

| 气体 | 国际标准（ISO） | 中国标准 |
|------|---------------|----------|
| 氧气 | 白色 | 浅蓝色 |
| 二氧化碳 | — | 白色 |
| 笑气 | — | 银灰色 |

- 储气钢瓶余压要求：≥ **1 MPa**（不是0.1 MPa）
- 小型钢瓶4~5L/15MPa储气量：**600~750 L**

## 流量计

- 流量计分为电子流量计和机械流量计
- 目前临床以电子流量计为主
- 新鲜气体出口安装防逆活瓣有利于流量计显示稳定

**影响因素**：

$$流量 \propto \frac{P}{T} \quad\quad 蒸发器输出浓度 \propto \frac{T}{P}$$

| 情况 | 影响 |
|------|------|
| 大气压增高 | 指示流量高于实际 |
| 温度增高 | 指示流量高于实际 |
| 流量计倾斜 | 需区分情况 |
| 流量计漏气 | 流量显示降低（视漏气位置而定） |
