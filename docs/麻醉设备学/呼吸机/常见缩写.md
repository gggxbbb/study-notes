# 呼吸机常见缩写

## 一、通气模式

| 缩写 | 全称 | 说明 |
|------|------|------|
| CMV | Continuous Mandatory Ventilation | 持续强制通气；也常指 AC 模式 |
| AC / A/C | Assist-Control | 辅助-控制通气：患者触发则辅助，无触发则按设定频率控制送气 |
| SIMV | Synchronized Intermittent Mandatory Ventilation | 同步间歇指令通气：指令通气+自主呼吸 |
| VCV / VC | Volume-Controlled Ventilation | 容量控制通气：潮气量恒定，压力随顺应性变化 |
| PCV / PC | Pressure-Controlled Ventilation | 压力控制通气：送气压力恒定，潮气量随顺应性变化 |
| PSV / PS | Pressure Support Ventilation | 压力支持通气：患者触发，给予设定压力支持，自主呼吸模式 |
| CPAP | Continuous Positive Airway Pressure | 持续气道正压：单一恒定的正压水平，患者全程自主呼吸 |
| BiPAP / Bilevel | Bilevel Positive Airway Pressure | 双水平气道正压：两个压力水平（IPAP/EPAP），常用于无创 |
| APRV | Airway Pressure Release Ventilation | 气道压力释放通气：长时间高压+短时间低压释放 |
| PRVC | Pressure Regulated Volume Control | 压力调节容量控制：自动调节压力以达到目标潮气量 |
| NIV / NPPV | Non-Invasive Ventilation / Non-Invasive Positive Pressure Ventilation | 无创通气/无创正压通气 |
| HFNC | High-Flow Nasal Cannula | 经鼻高流量湿化氧疗：非严格通气模式，但常作为氧疗升级手段 |
| ASV | Adaptive Support Ventilation | 自适应支持通气：闭环调节 |
| P-SIMV | Pressure-SIMV | 压力目标的 SIMV |
| V-SIMV | Volume-SIMV | 容量目标的 SIMV |
| MMV | Mandatory Minute Ventilation | 指令分钟通气量：保证最低分钟通气量 |
| AMV | Adaptive Mechanical Ventilation | 适应性机械通气 |

## 二、呼吸参数

| 缩写 | 全称 | 说明 |
|------|------|------|
| PEEP | Positive End-Expiratory Pressure | 呼气末正压：呼气末维持的气道正压 |
| FiO2 | Fraction of Inspired Oxygen | 吸入氧浓度：范围 0.21（空气）~ 1.0（纯氧） |
| VT / TV | Tidal Volume | 潮气量：每次呼吸吸入或呼出的气体量 |
| RR / f | Respiratory Rate | 呼吸频率 |
| I:E | Inspiratory to Expiratory Ratio | 吸呼比：正常约 1:2 |
| PIP / Ppeak | Peak Inspiratory Pressure | 吸气峰压：吸气相气道最高压力 |
| Pplat | Plateau Pressure | 平台压：吸气末屏气时的气道压力，反映肺泡压 |
| PS | Pressure Support | 压力支持水平 |
| ΔP / DP | Driving Pressure | 驱动压 = Pplat - PEEP，ARDS 保护性通气核心指标 |
| Cstat / Cs | Static Compliance | 静态顺应性：$C_{stat} = \frac{V_T}{P_{plat} - PEEP}$ |
| Cdyn / Cd | Dynamic Compliance | 动态顺应性：$C_{dyn} = \frac{V_T}{PIP - PEEP}$ |
| Raw | Airway Resistance | 气道阻力：$R_{aw} = \frac{P_{peak} - P_{plat}}{Flow}$ |
| MV / VE | Minute Ventilation | 分钟通气量 = $V_T \times RR$ |
| TVe / VE | Expired Tidal Volume / Expiratory Minute Volume | 呼出潮气量/呼出分钟量 |
| TVi / VI | Inspired Tidal Volume / Inspiratory Minute Volume | 吸入潮气量/吸入分钟量 |
| Vd | Dead Space Volume | 生理死腔量 |
| Vd/Vt | Dead Space to Tidal Volume Ratio | 死腔/潮气量比：正常约 0.2~0.4 |
| FRC | Functional Residual Capacity | 功能残气量 |
| P0.1 | Occlusion Pressure at 0.1s | 气道闭塞压：评估呼吸驱动 |
| NIF / MIP | Negative Inspiratory Force / Maximal Inspiratory Pressure | 最大吸气负压：评估脱机能力 |
| RSBI | Rapid Shallow Breathing Index | 浅快呼吸指数 = $f / V_T$（L），脱机指标 |
| WOB | Work of Breathing | 呼吸功 |
| PTP | Pressure-Time Product | 压力-时间乘积：评估呼吸努力 |

## 五、常用评估量表与标准

| 缩写 | 全称 | 说明 |
|------|------|------|
| ARDSnet | ARDS Network | ARDS 临床研究网络：提出保护性通气策略 |
| LIP / UIP | Lower / Upper Inflection Point | 压力-容积曲线下/上拐点 |
| OI | Oxygenation Index | 氧合指数（需测平均气道压）：$OI = \frac{FiO_2 \times P_{aw} \times 100}{PaO_2}$ |
| OSI | Oxygen Saturation Index | 氧饱和度指数：$OSI = \frac{FiO_2 \times P_{aw} \times 100}{SpO_2}$ |
| ROX | Respiratory Rate-Oxygenation Index | ROX 指数 = $SpO_2/FiO_2 / RR$，HFNC 预后指标 |
| SOFA | Sequential Organ Failure Assessment | 序贯器官衰竭评分 |

## 六、口诀速记

- **通气模式**：CMV 控，AC 辅控，SIMV 间歇，PSV 支持
- **两大控制**：VC 控量压变，PC 控压量变
- **PEEP 本质**：呼气末不让肺塌，改善氧合减分流
- **Pplat 意义**：屏气测得肺泡压，肺保护它要 < 30
- **驱动压 ΔP**：Pplat 减 PEEP，ARDS 保护 < 15
- **P/F 比值**：PaO2 除 FiO2，氧合好不好，一看就知道
