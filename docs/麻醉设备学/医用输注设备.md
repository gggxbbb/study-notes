# 医用输注设备

## 第一节 概述

**医用输注泵**：由电脑控制，自动推进液体进入血管系统的机械装置。

### 分类
- **输液泵**：常规较大量的液体输注治疗
- **微量注射泵**：特殊药物注射（血管活性药、静脉麻醉药等）

### 性能要求
- **使用模式**：输注范围0~1000ml，滴速准确；按键次数少，可迅速改变输液速度；可使用不同类型大小的注射器；可单次输注，并可预设单次输注量
- **供电**：交直流两用，断电自动切换至蓄电池，不改变输注速度、不丢失储存资料
- **安全性**：主体不易损坏、防水、抗电磁干扰；具有电和机械安全性
- **操作**：各方面功能有确定键；包含"错误限制"程序防止误操作；可清晰显示各项信息
- **报警功能**：注射器大小/位置错误、接近完成报警、堵塞报警、电池不足、内部机械故障报警；报警声音大小合适，具有忽略报警功能
- **结构**：轻巧紧凑，无突出部位，可迅速安装于输液架
- **兼容性**：可输注各类液体，适用各类病人
- **记录与控制**：能记录输注资料，可远程控制

## 第二节 步进电机驱动

步进电机：角位移与输入电脉冲数成正比，速度与脉冲频率成正比。电机的转速、停止位置只取决于脉冲信号的频率和数量，不受负载变化的影响。

**驱动方法**：控制指令 → 脉冲发生控制单元 → 功率驱动单元 → 步进电机

- 脉冲发生控制单元：产生脉冲，控制电机速度和转向
- 功率驱动单元：为各绕组提供正确的通电顺序
- 保护单元：过流、过热保护

## 第三节 输液泵的工作原理

### 直线蠕动输液泵
**组成**：操作面板 + 微控制器 + 步进电机 + 蠕动装置 + 液体监控报警装置

**原理**：步进电机通过变速箱带动偏心凸轮，凸轮推动滚柱按顺序挤压输液管，驱动液体向前流动。

**液体监控报警装置**：
- **气泡探测器**（超声）：超声波发射至输液管，出现气泡时频率改变
- **光电检滴器**：液滴阻断光线 → 感光装置发出信号
- **压力检测器**：检测管路压力，堵塞或泄漏时报警
- **电刀干扰**：输液泵与电刀不应接在同一电源插座上，高频电流会造成电磁干扰

### 旋转蠕动输液泵
滚轮挤压管路，输注速度取决于滚轮速度。

### 往复活塞式输液泵
活塞往复运动，将定量液体吸入后唧出，通过单向阀门控制流向。

### 活塞启动膜式输液泵
隔膜推动 + 单向阀，可避免气体排入患者体内。

## 第四节 微量注射泵

**原理**：以恒定性压力作用于注射器活塞柱，液体通过管道流入血管。

**特点**：稳定准确，容量小，用于输注麻醉药和特殊药物。

**主要结构**：步进电机传动系统，系统中常设置有3个检测状态：
- **前置位**：设在终点前，一旦达到前置位即控制步进电机减速以避免出现过冲；同时提示药液输完，自动停机并报警
- **阻塞检测**：推注丝杠另一端轴向位置设置微动开关，当输注受阻时丝杠出现轴向运动使触点闭合，进行声光报警

## 第五节 镇痛泵

**PCA（患者自控镇痛）**：患者根据自身疼痛情况自行控制给药，可降低阿片类药物用量和副作用。

### 分类
按动力驱动方式分：
- **机械类**：一次性弹性镇痛泵（弹性膜球囊 + 流速控制器 + 患者自控表），流速由回缩力与流速控制器共同决定
- **电子类**：以电能为动能，泵体由微电脑和微电机组成，配合旋转机械装置以精确控制速度泵注液体

按自控方式分：
- **自控型（PCA）**
- **非自控型**：持续非自控输注

### PCA基本结构
1. **储药器**（贮药袋）
2. **输注设备**
3. **单次输注按钮装置**
4. **设置单次剂量装置**
5. **设置锁定时间装置**

### PCA参数设置
- 背景输注速率
- 单次注药量设定
- 单次给药锁定时间
- 预计输注总容积

## 第六节 靶浓度控制注射泵（TCI）

**TCI**：以药代动力学和药效动力学为基础，容许医生选择靶血药浓度，通过调整靶浓度控制麻醉深浅。

**目标**：使效应部位药物浓度迅速达到设定值并保持平稳。

**工作原理**：
1. 输入药物、靶浓度、患者体重等参数
2. 药代动力学模型实时模拟，计算预测浓度
3. 比较设定值与预测值
4. 计算新的输注速率
5. 改变电机转速

**计算方法（BET法）**：先注射负荷量（Bolus），再根据药物从机体排出的速率（Elimination）与药物从中央室向周边室转运的速率（Transfer），向中央室补充给药量。

**血浆靶 vs 效应室靶**：效应室浓度作为靶浓度更符合逻辑。

### TCI前景
1. 控制系统需提供可接受的系统性能（诱导时间、超射浓度和程度、稳定时间、稳定时摆动程度、靶浓度与实测浓度的最大差值）
2. 系统需能对注射器更换或人为中断反馈信号等情况作出调整
3. 效应室浓度作为靶浓度更符合逻辑
