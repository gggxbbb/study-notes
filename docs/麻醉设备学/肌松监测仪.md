# 肌松监测仪

## 概述

肌松监测仪用于监测神经肌肉传递功能，评估肌松药的效果和恢复情况。

## 监测方法

### EMG（肌电图法）
- 记录复合肌肉动作电位（CMAP）
- 通过表面电极或针电极记录
- **优点**：更直接反映神经肌肉传递功能，不易受外界干扰
- **缺点**：操作相对复杂

### AMG（加速度法）
- 基于压电加速度传感器
- 检测加速度（F = ma），反映肌肉收缩强度
- **优点**：传感器不易受干扰，不需予置前负荷，操作方便
- **缺点**：稳定性不如EMG

### PMG（压电法）
- 利用Piezo-electric传感器检测骨骼肌收缩幅度
- 检测横向共振/低频声波
- **优点**：多部位监测，准确性高
- **缺点**：受外界声波影响，电刀干扰

## 电刺激模式

| 模式 | 频率 | 用途 |
|------|------|------|
| 单次颤搐刺激（ST） | 0.1~1.0Hz | 基础评估 |
| 四成串刺激（TOF） | 2Hz×4次 | 最常用，评估肌松深度 |
| 强直后计数（PTC） | 50Hz×5s | 深度肌松监测 |
| 双重爆发刺激（DBS） | 50Hz×3次×2组 | 残余肌松评估 |

### TOF（Train of Four）解读

| T4/T1比值 | 临床意义 |
|-----------|----------|
| 0.7~0.9 | 轻度肌松 |
| 0.5~0.7 | 中度肌松 |
| 0.25~0.5 | 深度肌松 |
| <0.25 | 极深度肌松 |
| ≥0.9 | 可安全拔管 |

## 加速度传感器原理

压电陶瓷晶片两侧为电极，检测微小加速度变化。质量块在加速度作用下对晶片施加力，产生电荷信号。

## 临床应用

- 肌松药起效时间判断
- 术中肌松深度维持
- 肌松拮抗时机评估
- 残余肌松监测（TOF≥0.9方可拔管）
