# 体外循环辅助设备

## 缩写

| 缩写 | 全称 |
|------|------|
| CPB | Cardiopulmonary Bypass（人工心肺机） |
| ECMO | Extracorporeal Membrane Oxygenation（体外膜肺氧合） |
| IABP | Intra-aortic Balloon Pump（主动脉内球囊反搏） |

## 发展历史

- **苏鸿熙**（医学界"钱学森"）：1958年6月完成中国首例体外循环手术
- 2019年中国心外科手术和体外循环数据白皮书：175557例

## 第一节 人工心肺机（CPB）

**概念**：将人体静脉血液引至体外运行并人工氧合为动脉血，再依靠血泵将氧合动脉血送回人体，以维持心脏外科手术期间各器官和组织的正常灌注的动力输注设备。

**功能**：在心脏外科手术中替代心肺功能。

**原理**：上下腔静脉 → 氧合器（气体交换）→ 血泵（驱动）→ 变温器（温度调节）→ 主动脉 → 全身组织。体外循环过程中，肺循环在体外进行，体循环依靠机械动力在体内进行。

**结构组成**：血泵、氧合器、热交换系统、血液回收与过滤系统。

### 一、血泵

血泵：通过机械方法驱动血液流动以临时代替心室的搏出功能以及术中失血的回吸，以维持机体的血液循环。

**理想血泵要求**：
- 对血液破坏小，不改变血液成分
- 能对抗血管阻力，维持恒定输出量
- 转速精确可调，不易形成湍流
- 耐长时间挤压，有后备应急系统

**血泵组成**：动力系统（滚压泵/离心泵）、控制台、显示器、监测报警装置。

#### 滚压泵 vs 离心泵

| 特性 | 滚压泵 | 离心泵 |
|------|--------|--------|
| 原理 | 滚轮挤压泵管驱动血液 | 离心力驱动血液 |
| 流量调节范围 | 大，精度高 | 有波动，需监测 |
| 对血液破坏 | 轻微（长时间会破坏） | 不挤压血液，破坏更小 |
| 每搏量 | 恒定，不受阻力影响 | 转速一定时流量有波动 |
| 搏动血流 | 非搏动 | 可通过高低速交替产生 |
| 泵管破裂 | 有破裂危险 | 压力不大，不易崩脱破裂 |
| 体积 | 较大 | 体积小，可缩短管道长度，节约预充量 |

**滚压泵优点**：
- 能无级调速，流量调节范围大，精度高
- 每搏量恒定，不受血管阻力影响，保持单向流动
- 对血液破坏轻微
- 控制操作简单

**滚压泵缺点**：
- 长时间非搏动性灌注后机体会产生相应的病理生理变化
- 长时间灌注会造成红细胞破坏
- 管腔壁微栓易脱落，泵管有破裂危险

**离心泵特性**：
- 不挤压血液，对血液成分破坏少
- 转速一定时，流量会有所波动，需进行监测
- 体积小，可缩短管道长度，节约预充量
- 可通过控制马达高低速交替而产生搏动血流
- 压力不大，泵管不易崩脱破裂

#### 血泵的控制台和显示器

- **控制台**：电源开关、流量控制旋钮、流量/转数显示、功能菜单键、泵头运动方向控制键
- **显示器**：显示各种传感器数据、时间记录、系统状态信息

#### 血泵的监测系统

- **心脏停搏液灌注监测器**：气泡监测、压力监测
- **温度监测器**：心肌温度、血温、变温器进出水温、鼻咽温、肛温
- **压力监测器**：动脉管路压力、患者血压；超压自动停泵或减速

### 二、氧合器

**功能**：在体外循环中完成血液和外部氧和二氧化碳的交换，是决定体外循环质量和患者安全的关键部件。

**设计和制造要求**：
- 制造材料必须有较好的血液相容性
- 表面具备高光洁度，对血液的破坏要小
- 氧合面积要尽量大，但整体预充血量要尽量小
- 对血液的氧合效率要高，同时对二氧化碳的排除率也要高

#### 鼓泡式氧合器（BO）
工作过程：静脉血 → 氧合 → 变温 → 消泡 → 过滤 → 储血 → 动脉血

#### 膜式氧合器（MO）
气体通过膜进行交换，血液相容性更好。按血流路径分为：
- **内走血式MO**：血液在纤维内部流过，控制每根纤维的内径
- **外走血式MO**：血液在纤维外部流过，增加突起的坎或改变血流方向，使血液均匀分布、充分混合、减少层流，提高气体交换性能

**影响气体交换的因素**：
1. **膜的表面积** — 面积越大交换率越高，但异物接触面也越大
2. **膜的性能** — 甲基硅酮对CO₂透过度比O₂高6倍；聚四氟乙烯和聚乙烯对氧和二氧化碳透过率无限制；聚丙烯物理加工成薄膜后透气性好
3. **血膜厚度** — 内走血式控制纤维内径；外走血式增加突起或改变血流方向
4. **经膜驱动压** — 气体经膜弥散速率 = 透过度 × 驱动压。MO供气氧分压在100~200mmHg，二氧化碳分压在35~45mmHg较为理想

### 三、热交换系统

#### （一）变温器
直接与血液接触，执行变温功能。

**结构**：高分子材料制成中空纤维，水在纤维内通过，血在纤维外通过。可组合在氧合器上，或放置在静脉血入口端。

**影响变温效能因素**：有效热交换面积、原材料的导热性、血和水的温差、血和水的方向及流速、血流阻力。

#### （二）变温水箱
为变温器提供冷热水源的驱动装置，用于手术开始时降低病人基础体温以减少组织代谢，手术结束时恢复体温。

- **普通型**：加温、泵水，4泵速4温度档（30/38/40/42℃），有温度监测报警装置，**不能制冰**
- **全自动型**：制冷、制冰、加温，用于动脉血热交换、患者变温毯、心脏停搏液变温，具备温度显示、控制及报警

### 四、血液回收系统

**组成**：吸引头（质地光滑带侧孔）+ 连接管道（医用塑料管，尽可能短，内径合适，不能用Y型管连接两根吸引管）+ 吸引动力装置（滚压式泵）+ 回收器（塑料外壳、过滤网、去泡海绵）

**应用方法**：
- **启用条件**：患者全身肝素化后
- **使用前**：调节泵的松紧，全过程持续监测，根据台上需要调节泵速，避免管道吸瘪和血液反流
- **使用时**：出血口与氧合器静脉回流入血口同一水平，出血口用管道钳钳闭并定期开放，保证血液有充分的消泡时间，减少血液破坏

### 五、过滤系统

**作用**：在体外循环过程中，滤去由于血液与大面积塑料或金属接触而凝聚成的微粒，防止肺、脑、肾、肝等脏器微血管血栓。

**过滤方式**：

| 过滤方式 | 原理 | 特点 |
|----------|------|------|
| 渗透式 | 纤维或细孔海绵状物质吸附滤除栓子 | 硅油去泡，阻力低，滤栓能力强 |
| 滤网式 | 纤维编织成的网状物滤除栓子 | 滤网为亲水性物质，排斥气泡 |

| 过滤器类型 | 过滤方式 | 过滤物 | 使用注意 |
|-----------|----------|--------|----------|
| 动脉过滤器 | 滤网式 | 气泡、栓子 | 预充前吹入CO₂ |
| 心内血液回收储血器 | 渗透式 | 组织碎片、赘生物、小线头 | — |
| 气体过滤器 | 混合式 | 微栓和病原体 | 避免进水 |
| 白细胞过滤器 | 渗透式 | 白细胞 | 重症、长时间体外循环和心脏移植患者 |

**其他过滤器**：输血过滤器、含血心脏停搏液过滤器、晶体预充液过滤器及库血除白细胞过滤器等。

## 第二节 体外膜肺氧合（ECMO）

**概念**：通过引流患者的静脉血，在体外进行气体交换后经过机械泵将血液经静脉或动脉输回患者体内的仪器。

**功能**：
- 有效气体交换
- 长期支持性灌注为心肺功能恢复赢得时间
- 避免高氧吸入所致氧中毒
- 避免机械通气所致气道损伤
- 有效循环支持
- ECMO治疗中可用人工肾对机体内环境如电解质进行可控性调节

### ECMO vs CPB

| 项目 | ECMO | CPB |
|------|------|-----|
| 目的 | 心肺内在恢复 | 人工解决 |
| 使用场所 | ICU/急诊/手术室 | 手术室 |
| 血流方向 | V-A, V-V | V-A |
| 麻醉选择 | 局麻 | 全麻 |
| 维持时间 | 长（1~30天） | 短（≤8h） |
| 置管部位 | 体表 | 经胸 |
| 动脉过滤器 | 无 | 有 |
| 自体血回输 | 无 | 有 |

**转流模式**：
- **V-A方式**：右颈内静脉 → 膜肺 → 右颈总动脉 → 主动脉弓（体外呼吸+心脏支持）。摄氧量在一定范围内取决于血流量，二氧化碳排出量取决于进入膜肺的气流量
- **V-V方式**：颈内静脉引出 → 膜肺 → 股静脉回输；也可从一侧股静脉引出后回输入对侧股静脉

### ECMO的结构

**可抛弃部分**：PVC管道、接头、硅胶静脉储血囊、膜肺、热交换器。
**反复使用部分**：血泵、硅胶储血囊支架、血泵控制器、变温水箱、肝素微量注射泵、ACT仪、静脉血饱和度监测仪、测压装置、电池组件。

#### （一）膜肺和血泵
核心驱动与气体交换组件。

#### （二）插管及管路
- 一般选择尽可能大的插管以获得足够的静脉引流量和ECMO灌注量，具体选择因体重而定
- ECMO系统里没有特别的滤网，因为安放过滤器会增加血栓栓塞的机会和溶血

#### （三）储血囊支架与血泵控制器
- **作用**：防止血液引流量不足或泵流量超过静脉引流量引入气栓
- **方法**：在静脉引流的低水平处安置可膨缩的硅橡胶储血囊并安放进带有传感器的支架内，当储血囊萎缩时自动停泵并报警，当储血器恢复充盈状态自动恢复运转

#### （四）热交换器和变温水箱
**目的**：维持体温37℃，补偿因水分蒸发导致的热量丢失。

#### （五）压力检测仪
实时监测动脉收缩压、舒张压、平均压及膜肺进出口压。

#### （六）静脉血氧饱和度监测仪
通过探头插入静脉管路内，连续监测静脉血氧饱和度。

#### （七）ACT监测仪
抗凝水平监测，决定肝素用量。

## 第三节 主动脉内球囊反搏（IABP）

**功能**：使左心负担减轻并增加冠状动脉的血流灌注，辅助左心室克服暂时性困难，使被抑制的心肌重新恢复功能。

**原理**：通过动脉系统置入一根装有气囊的导管至降主动脉（左锁骨下动脉远端至肾动脉开口近端）。心脏舒张期气囊充气，收缩前排气。

**气囊工作过程**：
- **心室舒张时**：气囊充气扩张形成反搏，将主动脉血液逆向压至主动脉根部，使近端主动脉舒张压升高，明显增加冠状动脉血流灌注，改善缺血心肌的供应
- **心室收缩前**：迅速排气，降低主动脉腔内压力，降低舒张末期动脉压力，降低左心室射血阻力

**血流动力学效应**：
- 降低左心室后负荷，减轻心脏做功
- 提高舒张压，增加冠状动脉灌注
- 改善全身重要器官血流灌注
- 增加心排血量，降低右心室负荷

**结构**：
- **气囊导管**（聚脲氨酯高分子材料，抗血栓、生物相容性好）：气囊薄而透明柔软，两端与导管密接，导管内不同方向和部位有小孔与气囊相连
- **驱动控制与报警系统**：
  - 驱动气体：氦气（流动阻力小）或二氧化碳（价廉无气栓）
  - 触发方式：心电图触发、压力触发
  - 报警系统：漏血保护
