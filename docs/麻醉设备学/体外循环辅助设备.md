# 体外循环辅助设备

## 缩写

| 缩写 | 全称 |
|------|------|
| CPB | Cardiopulmonary Bypass（人工心肺机） |
| ECMO | Extracorporeal Membrane Oxygenation（体外膜肺氧合） |
| IABP | Intra-aortic Balloon Pump（主动脉内球囊反搏） |

## 第一节 人工心肺机（CPB）

**概念**：将人体静脉血液引至体外运行并人工氧合为动脉血，再依靠血泵将氧合动脉血送回人体，以维持心脏外科手术期间各器官和组织的正常灌注的动力输注设备。

### 一、血泵

**理想血泵要求**：
- 对血液破坏小，不改变血液成分
- 能对抗血管阻力，维持恒定输出量
- 转速精确可调，不易形成湍流
- 耐长时间挤压，有后备应急系统

#### 滚压泵 vs 离心泵

| 特性 | 滚压泵 | 离心泵 |
|------|--------|--------|
| 原理 | 滚轮挤压泵管驱动血液 | 离心力驱动血液 |
| 流量调节范围 | 大，精度高 | 有波动，需监测 |
| 对血液破坏 | 轻微（长时间会破坏） | 更小 |
| 每搏量 | 恒定，不受阻力影响 | 转速一定时流量有波动 |
| 搏动血流 | 非搏动 | 可通过高低速交替产生 |
| 泵管破裂 | 有破裂危险 | 不易崩脱 |

### 二、氧合器

**功能**：完成血液和外部氧和二氧化碳的交换。

#### 鼓泡式氧合器（BO）
工作过程：静脉血 → 氧合 → 变温 → 消泡 → 过滤 → 储血 → 动脉血

#### 膜式氧合器（MO）
气体通过膜进行交换，血液相容性更好。

**影响气体交换的因素**：
1. **膜的表面积** — 面积越大交换率越高，但异物接触面也越大
2. **膜的性能** — 甲基硅酮对CO₂透过度比O₂高6倍
3. **血膜厚度** — 控制纤维内径、增加突起、改变血流方向
4. **经膜驱动压** — 气体经膜弥散速率 = 透过度 × 驱动压

### 三、热交换系统

**变温器**：直接与血液接触，影响变温效能因素：有效热交换面积、导热性、血水温差、血流方向及流速、血流阻力。

**变温水箱**：
- 普通型：加温、泵水，4泵速4温度档（30/38/40/42℃）
- 全自动型：制冷、制冰、加温，用于动脉血热交换、患者变温毯、心脏停搏液变温

### 四、血液回收系统

**组成**：吸引头 + 连接管道 + 吸引动力装置（滚压式泵）+ 回收器（外壳、过滤网、去泡海绵）

### 五、过滤系统

**作用**：滤去血液与大面塑料或金属接触凝聚成的微粒，防止肺、脑、肾、肝微血管血栓。

| 过滤器类型 | 过滤方式 | 过滤物 |
|-----------|----------|--------|
| 动脉过滤器 | 滤网式 | 气泡、栓子 |
| 心内血液回收储血器 | 渗透式 | 组织碎片、赘生物 |
| 气体过滤器 | 混合式 | 微栓和病原体 |
| 白细胞过滤器 | 渗透式 | 白细胞 |

## 第二节 体外膜肺氧合（ECMO）

**概念**：通过引流患者的静脉血，在体外进行气体交换后经过机械泵将血液经静脉或动脉输回患者体内。

**功能**：
- 有效气体交换
- 长期支持性灌注为心肺功能恢复赢得时间
- 避免高氧吸入所致氧中毒
- 避免机械通气所致气道损伤
- 有效循环支持

### ECMO vs CPB

| 项目 | ECMO | CPB |
|------|------|-----|
| 目的 | 心肺内在恢复 | 人工解决 |
| 使用场所 | ICU/急诊/手术室 | 手术室 |
| 血流方向 | V-A, V-V | V-A |
| 麻醉选择 | 局麻 | 全麻 |
| 维持时间 | 长（1~30天） | 短（≤8h） |
| 置管部位 | 体表 | 经胸 |
| 动脉过滤器 | 无 | 有 |
| 自体血回输 | 无 | 有 |

**转流模式**：
- **V-A方式**：右颈内静脉 → 膜肺 → 右颈总动脉 → 主动脉弓（体外呼吸+心脏支持）
- **V-V方式**：颈内静脉引出 → 膜肺 → 股静脉回输

## 第三节 主动脉内球囊反搏（IABP）

**原理**：通过动脉系统置入气囊导管至降主动脉（左锁骨下动脉远端至肾动脉开口近端）。心脏舒张期气囊充气，收缩前排气。

**血流动力学效应**：
- 降低左心室后负荷，减轻心脏做功
- 提高舒张压，增加冠状动脉灌注
- 改善全身重要器官血流灌注
- 增加心排血量，降低右心室负荷

**结构**：
- 气囊导管（聚脲氨酯高分子材料，抗血栓、生物相容性好）
- 驱动控制与报警系统
  - 驱动气体：氦气（流动阻力小）或二氧化碳（价廉无气栓）
  - 触发方式：心电图触发、压力触发
