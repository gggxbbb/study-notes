# 循环功能监测仪器

## 心电图（ECG）

### 导联系统
- **标准肢体导联**（I、II、III）
- **加压单极肢体导联**（aVR、aVL、aVF）
- **胸导联**（V₁~V₆）

### 心电信号特征
- P波：心房除极（0.08~0.11s）
- QRS波群：心室除极（0.06~0.10s）
- T波：心室复极
- PR间期：0.12~0.20s

### 监护导联选择
- 通常选择II导联（P波最明显）或V₅导联（左心室缺血敏感）

## 无创血压监测（NIBP）

### 测量方法
- **手动法**：听诊法（Korotkoff音）
- **自动法**：振荡法（示波法）— 袖带放气过程中检测振荡波幅

### 正常值
- 收缩压：90~139mmHg
- 舒张压：60~89mmHg
- 平均动脉压（MAP）：70~105mmHg

## 有创血压监测（IBP）

### 适应证
- 血流动力学不稳定
- 需持续监测血压变化
- 频繁动脉血气分析
- 控制性降压

### 组成
- 动脉导管 + 压力传感器 + 冲洗装置 + 监护仪
- 常用穿刺部位：桡动脉、股动脉、足背动脉

### 注意事项
- 压力传感器需与心脏水平校零
- 管路内不能有气泡
- 定时冲洗防止血栓

## 中心静脉压（CVP）

- 正常值：4~12cmH₂O
- 反映右心室前负荷和血容量
- 评估容量状态、指导液体治疗
