# 常考知识点汇总

---

## 第一章：麻醉机

### 1. 蒸发器（出镜率最高：97次）

- **饱和蒸气压**：与液体种类有关，与液体量无关；温度升高则饱和蒸气压增大
- **强效吸入麻醉药**汽化特点
- 蒸发器输出浓度与**温度补偿**相关

### 2. 流量控制系统（不包括的功能）

| 属于流量控制 | 不属于流量控制 |
|-------------|---------------|
| 控制新鲜气体成分和流量 | 显示麻醉药蒸气浓度 |
| 显示新鲜气体流量 | 麻醉深度监测 |
| 快速供氧 | 二氧化碳吸收 |
| 防止低氧混合气输出 | |

### 3. 共同气体出口（Common Gas Outlet）

- 防逆流活瓣安放位置：蒸发器与新鲜气体出口之间
- 作用：防止麻醉回路内气体倒流

### 4. 麻醉回路

- **Mapleson系统**：A~F各回路的吸气/呼气特点
- **循环回路**：二氧化碳吸收装置、单向活瓣
- **循环紧闭式回路**特点与缺点

### 5. CO₂吸收装置

- 钠石灰（Soda Lime）是关键吸收剂
- 吸收原理：化学中和反应

### 6. 安全装置

- **笑氧/氧比例控制（ORC）**：防止低氧混合气
- PISS（输注安全指示系统）：防止气源接错
- 流量计：正确读数与误差

## 第二章：通气机/呼吸机

### 1. 基本概念和分类

| 分类方式 | 类型 |
|----------|------|
| 按发生器 | 压力发生器、恒压/非恒压、恒流/非恒流 |
| 按切换 | 时间切换、容量切换、压力切换、流量切换 |
| 按动力 | 气动气控、电动电控、电控气动 |

### 2. 通气模式（高频考点）

| 缩写 | 全称 | 要点 |
|------|------|------|
| **IPPV** | 间歇正压通气 | 最基本的机械通气模式 |
| **CMV** | 控制通气 | 不受自主呼吸影响 |
| **AMV** | 辅助通气 | 患者触发 |
| **A/CMV** | 辅助/控制 | 窒息时间7~20秒 |
| **PSV** | 压力支持通气 | 定压模式 |
| **SIMV** | 同步间歇指令通气 | 部分通气支持 |
| **CPAP** | 持续气道正压 | 自主呼吸+持续正压 |
| **PEEP** | 呼气末正压 | 防止呼气末肺泡萎陷 |
| **BIPAP** | 双水平气道正压 | 高低两个压力水平 |
| **HFV** | 高频通气 | 60~3600bpm，Vt小 |

### 3. 输气系统

| 类型 | 特点 | 特性 |
|------|------|------|
| 压力输气系统 | 高压向低压释放 | 减流/恒流特性 |
| 容量输气系统 | 容积转移 | 容量限定，正弦波 |
| 持续气流输气系统 | 持续输出 | 用于自主呼吸 |

### 4. 报警（47题涉及，非常重要）

**气道压报警**：
- 低压报警原因：气路脱落/漏气、自主呼吸停止、设置不当
- 高压报警原因：管道梗阻、积水、患者对抗

**通用处理原则（大题考点）**：
1. 立即脱离通气机，改手工呼吸
2. 临床评估患者呼吸功能
3. 检查报警提示相关部件
4. 检查电源、气源及连接
5. 自患者端开始检查所有连接部件
6. 排除人为设置不当

### 5. 吸气向呼气转换

| 转换方式 | 原理 |
|----------|------|
| 时间切换 | 预设吸气时间到→切换 |
| 容量切换 | 预设潮气量达到→切换 |
| 压力切换 | 气道压达到预设值→切换 |
| 流量切换 | 吸气流速降到峰值25%→切换 |

## 第三章：医学气体监测

### 1. 气体采集方式

| 方式 | 传感器位置 | 特点 |
|------|-----------|------|
| **主流式** | 直接放在气管导管接口与回路之间 | 速度快、实时、预热、笨重 |
| **旁流式** | 监测仪内部（通过采气管抽气） | 稳定、延迟、应用最广 |
| **截流式** | 呼气末截取 | 最接近血气分析、不能连续 |

- 最能反映患者生理状态和麻醉水平的是：**呼气末气体和肺泡气体** ✅
- 主流和旁流检测值总是**低于**动脉血气分析结果

### 2. 气体检测技术

| 技术 | 适用气体 | 原理 | 特点 |
|------|---------|------|------|
| 电化学 | O₂ | Galvanic电池/Clark电极的氧化还原反应 | 慢，测平均氧浓度 |
| 顺磁 | O₂（唯一顺磁气体） | 顺磁物质在外加磁场中排列 | 实时、稳定 |
| 红外线 | CO₂、N₂O、吸入麻药 | 分子特定红外吸收光谱 | 最常用 |

### 3. 影响因素

| 因素 | 影响 |
|------|------|
| **水蒸气** | 吸收带与CO₂和麻药部分重叠→干扰测定 |
| **高浓度O₂** | 展宽CO₂吸收带→测量值偏低 |
| **N₂O** | 红外吸收峰与CO₂重叠→CO₂测量值偏高 |
| **海拔高度** | PCO₂ = FCO₂ × Pb→不同海拔F/P换算不同 |
| **呼吸频率过快** | >40次/min→呼气时间不足→测量值不准 |

### 4. 除水措施（旁流）

采气降温→积水杯沉积→疏水过滤膜→Nafion管渗透→检测器38℃恒温

### 5. 校准

标准气体配制方法：
- CO₂：$F_X = P_X / (P_G + P_B)$
- 吸入麻药：$F_X = V_X / V_{总}$，体积由药液量和分子量计算

## 第四章：肌松监测

### 监测方法

| 方法 | 原理 | 传感器 | 特点 |
|------|------|--------|------|
| **AMG**（加速度法） | F = ma | 加速度传感器（压电陶瓷） | 不受外界干扰、不需前负荷、方便但稳定性不如EMG |
| **EMG**（肌电图法） | 复合肌肉动作电位 | 表面/针电极 | 稳定、直接，但操作复杂 |
| **PMG**（压电法） | 横向共振检测 | 压电传感器 | 多部位、准，受声波和电刀干扰 |

### 电刺激模式

| 模式 | 用法 | 临床意义 |
|------|------|----------|
| **TOF**（四成串） | 2Hz×4次，最常用 | T4/T1 ≥ 0.9可拔管 |
| **PTC**（强直后计数） | 深度肌松监测 | 用于肝移植血管吻合等需要深度肌松的场景 ✅ |
| **DBS**（双重爆发） | 50Hz×3×2组 | 检测残余肌松，敏感度高 |
| 单次颤搐 | 0.1~1.0Hz | 基础评估 |

## 第五章：输注设备

### 1. 输液泵类型

| 类型 | 原理 |
|------|------|
| **直线蠕动输液泵** | 步进电机→偏心凸轮→滚柱顺序挤压输液管 |
| 旋转蠕动输液泵 | 滚轮挤压管路 |
| 往复活塞式 | 活塞通过单向阀吸液和推液 |
| 活塞启动膜式 | 隔膜推动液体 |

### 2. 液体监控报警

| 装置 | 检测原理 | 功能 |
|------|----------|------|
| 气泡探测器 | 超声波 | 检测输液管中的气泡 |
| 光电检滴器 | 光线阻断 | 检测输液滴速 |
| 压力检测器 | 压力传感器 | 检测堵塞或泄漏 |

### 3. 微量注射泵

用于输注血管活性药、静脉麻醉药等特殊药物

### 4. PCA（患者自控镇痛）

- **参数**：背景输注速率、单次注药量、锁定时间
- **目的**：个体化镇痛，减少阿片用量
- 机械类（一次性弹性泵）vs 电子类

### 5. TCI（靶浓度控制输注）

- 以药代动力学为基础
- BET法：先负荷量，再按排出速率+转运速率补充
- 效应室靶浓度比血浆靶浓度更合理

## 第六章：体外循环

### 1. 氧合器

| 类型 | 原理 |
|------|------|
| 鼓泡式氧合器 | 氧气直接与血液鼓泡接触→消泡 |
| **膜式氧合器（MO）** | 气体通过高分子膜弥散交换，首选 |

MO影响因素：膜表面积、膜性能、血膜厚度、经膜驱动压

### 2. 血泵

| 特性 | 滚压泵 | 离心泵 |
|------|--------|--------|
| 血液破坏 | 轻微（长时间会加重） | 更小 |
| 流量 | 恒定，不受阻力影响 | 有波动，需监测 |
| 泵管破裂 | 有风险 | 不易崩脱 |

### 3. ECMO vs CPB

| 项目 | ECMO | CPB |
|------|------|-----|
| 场所 | ICU/急诊/手术室 | 手术室 |
| 时间 | 1~30天 | ≤8h |
| 血流方向 | V-A或V-V | V-A |
| 动脉过滤器 | 无 | 有 |

### 4. IABP（主动脉内球囊反搏）

- **舒张期充气** → 增加冠脉灌注
- **收缩期排气** → 降低后负荷
- 气囊位于：左锁骨下动脉远端至肾动脉近端的降主动脉

## 第九章：超声诊断仪器

### 压电效应

- 发射：电能 → 机械能（电压→晶体振动→声波）
- 接收：机械能 → 电能（声波→晶体→电压信号）
- 压电材料：石英、钛锆酸铅

### 分辨力

- **轴向分辨力**：沿声束纵轴（与频率成正比）
- **侧向分辨力**：与声束垂直（与声束宽度相关）
- **对比分辨力**：区别不同组织灰度
- **时间分辨力**：取决于帧频

### 超声模式对比

| B超（2D） | M超 | 脉冲多普勒（PW） | 连续多普勒（CW） | 彩色多普勒 |
|-----------|-----|-----------------|-----------------|-----------|
| 解剖结构 | 纵向运动 | 定点测速，有Nyquist极限 | 测高速血流，无极限 | 方向和速度显示 |

### 伪像

- 混响伪像（A线）
- 声影、回声增强、旁瓣伪像、镜面伪像

## 高频易错概念速查

| 易混概念 | 区分要点 |
|----------|----------|
| 主流 vs 旁流 vs 截流 | 传感器位置不同，截流最准但不能连续 |
| 恒压 vs 恒流发生器 | 恒压低压 → 减流；恒流高压 → 不受气道压影响 |
| 容量预置 vs 压力预置 | 定容→潮气量恒定；定压→气道压恒定 |
| 鼓泡式 vs 膜式氧合器 | 膜式对血液破坏小，气体经膜弥散 |
| AMG vs EMG vs PMG | AMG加速度、EMG肌电、PMG压电 |
| Galvanic vs Clark vs 顺磁 | 前两者电化学（慢），顺磁实时 |
| PW vs CW多普勒 | PW有Nyquist极限测低速，CW无极限测高速 |
| IPPV vs CPAP vs PEEP | IPPV最基本，CPAP自主+持续正压，PEEP呼气末正压 |
| VCV vs PCV | VCV容量控制（恒流），PCV压力控制；PLV是常规横流定压≠PCV |
| CMV vs AMV vs A/CMV | CMV控制（不受自主呼吸）、AMV辅助（患者触发）、A/CMV辅助/控制（窒息7~20秒切换） |
| CPPV vs CPAP | CPPV持续正压通气（通气机供气），CPAP持续气道正压（自主呼吸+正压），气道压曲线不同！ |

## ⚠️ 郑小峰题库特有高频易错点（原题重灾区）

### 1. 笑气/氧气比例控制三种机制

| 机制 | 原理 | 特点 |
|------|------|------|
| ① 25%联动比例限定控制系统 | 滚轮联动 | 保证输出氧浓度≥25%（氧气调节阀与笑气调节阀不独立） |
| ② 氧气比例监测控制器 | 纯气动原理 | 氧气阀和笑气阀*分开独立调节* |
| ③ 定量混合气体流量计 | 只有两个阀：参比调节阀+笑气调节阀 | 总流量阀+氧浓度阀 |

**关键掌握：**
- 联动（①）→ 两阀不独立，防低氧
- 独立调节（②）→ 有独立氧气阀和笑气阀→ 说明装了氧气比例监测控制器
- 总流量+氧浓度（③）→ 只有总流量阀和氧浓度阀

### 2. 低氧压氧化亚氮安全切断阀 ⭐

> 🚫 **不能防止低氧混合气体输出！**

原理：气动原理，"你死我也跟着死"——氧压低时笑气也低，但混合比例不变。只能做到：没有氧气压力时直接 cut down 笑气。

### 3. 阀门开关方向（特例区分）

| 阀门类型 | 打开方向 |
|----------|----------|
| **减压调节阀** | **顺时针→打开**（唯一特例） |
| 蒸发器补液装置 | 逆时针→打开 |
| 蒸发器可变分流控制阀 | 逆时针→打开 |
| APL阀 | 逆时针→打开 |
| 其他启动装置 | 逆时针→打开 |

### 4. 蒸发器关键陷阱 ⭐

- **蒸发器没有关闭位！**（"任何蒸发器在关闭位..." → 错 ❌）
- 输出浓度与患者实际吸入浓度：输出浓度 **大于** 吸入浓度（ACGO输出气体部分用于冲洗回路CO₂，会稀释）

### 5. 快速充氧开关

- 没有关闭位（开启=瞬间关闭，不存在微开状态）
- 一定是手动的（不能自动）
- 关闭不严 → 持续供氧稀释混合气体 → **麻醉过浅**（不是过深！）
- 开关 ≠ 阀（开关只 OPEN/CLOSE，不能调节程度）
- 蒸发器**压力效应**主要是由快速给氧开关引起的

### 6. 储气钢瓶颜色

| 气体 | 国际标准（ISO） | 中国标准 |
|------|---------------|----------|
| 氧气 | **白色** | **浅蓝色** |
| 二氧化碳 | — | 白色 |
| 笑气 | — | 灰色（中国） |

> 中国：银灰色=笑气，浅蓝色=氧气，白色=CO₂

### 7. PISS（定位销安全指示系统）

- 仅适用于麻醉机上**小型储气钢瓶**
- 本质上防止气源供应错误，当然也可防止AGCO输出错误
- 四种防错机制：①钢瓶标识标准 ②PISS③DISS④NIST

### 8. 流量计影响因素

$$流量 \propto \frac{P}{T} \quad\quad 蒸发器输出浓度 \propto \frac{T}{P}$$

- 流量计倾斜 → 指示流量低于实际流量 ❌（需要区分各种情况）
- 温度增高 → 指示流量高于实际流量 ❌
- 大气压增高 → 指示流量高于实际流量 ✅
- 流量计漏气 → 流量显示**降低**（不一定，需区分漏气位置）

### 9. 钠石灰真相

- **主要成分不是 NaOH！**（名字迷惑）
- 吸收CO₂后：变白、变硬、产热 → 产生CaCO₃
- 钡石灰可与七氟醚反应 → 起火事故风险

### 10. 贮气囊 → 呼吸活瓣 → CO₂吸收罐

- 病人呼出气体 → 先通过**呼吸活瓣** → 再到**CO₂吸收罐**
- 贮气囊不包括：防止重复呼吸（❌ 这是CO₂吸收罐的功能）
- 水滴附着**会影响**呼吸活瓣开启（❌"不会影响"是错的）

### 11. 循环回路 / Mapleson 要点

| 回路 | 自主呼吸 | 控制呼吸 |
|------|----------|----------|
| Mapleson A | ✅ 效率高 | ❌ 效率低 |
| Mapleson D | ❌ 效率低 | ✅ 效率高 |
| 循环回路 | 阻力大（很多阀门） | 优点突出 |

**半紧闭吸入麻醉特点**：通气量 > 麻醉气体消耗量（耗气量）

### 12. 蒸发器相关数值

| 项目 | 值 |
|------|-----|
| 麻醉机/通气机外接气源额定工作气压 | **0.3~0.5 MPa** |
| 麻醉回路吸气/呼气通道阻力标准 | 60 L/min下 < **0.6 kPa** |
| 笑氧比例控制最低氧浓度 | **25%** |
| 常频/高频通气交界点 | **60 bpm** |
| 储气钢瓶余压要求 | ≥ **1 MPa**（不是0.1！） |
| APL阀关闭态3kPa下泄漏 | ≤ **50 mL/min** |

### 13. 通气模式关键区分

- **PSV智能化版本**：VSV（容量支持通气）、**PRVCV**（压力调节容量控制通气）⭐
- **自主呼吸停止** → 只能用**FVS**（完全通气支持模式），不能选PVS
- **PEEP特点**：❌ "降低肺顺应性"是错的 → PEEP是**增加**肺顺应性！
- **BiPAP特点**：低频通气，设控制+辅助双模式可相互切换
- **叹息（Sigh）**：长时间控制通气中模拟打呵欠的深呼吸功能
- **VCV（定容）**参数：Vt、f、I:E，无气道压参数
- **PCV（定压）**参数：f、I:E、Paw，无潮气量参数
- **IPPV定义**：吸气期气道压>大气压→气体压入肺；呼气期气道压与大气压平衡→弹性回缩呼出

### 14. 报警分级 ⭐

| 等级 | 定义 | 例子 |
|------|------|------|
| **一类**（立即） | 威胁生命，需立刻处理 | ⚠️ **气道压力过高** |
| 二类（尽快） | 潜在威胁，需尽快处理 | 管路漏气、蓄电池不足、空氧混合器失灵 |
| 三类（延时） | 不会危及生命 | 吸气温度过小 |

**气道压过低报警常见原因**：气路脱落/漏气、呼气阀漏气、自主呼吸停止、报警下限设置过高 → ❌ **潮气量设置过大不是原因**（过大反而导致气道压增高）
