# 围术期保温设备

## 第一节 体温检测原理和方法

**中心温度（核心温度）**：心脏和脑部的血液温度，代表人体真实温度。

### 各部位测温特点

| 部位 | 优点 | 缺点 |
|------|------|------|
| 肺动脉导管 | 最准确 | 有创伤，操作复杂 |
| 直肠 | 不受室温影响，临床标准 | 滞后于中心温度变化 |
| 前额/腋下 | 简便、廉价、无创 | 影响因素多，准确度低 |
| 口腔 | 简便 | 受呼吸、饮食影响 |
| 耳鼓膜 | 无创、快速、准确 | 需要修正读数 |

### 体温监测适应症
- 预期体温明显改变
- 快速大量输血输液手术
- 体外循环心内直视手术
- 低温麻醉
- 小儿/高龄患者手术
- 恶性高热病史或家族史

### 测温方法

#### 热敏电阻测温法
利用电阻率随温度变化的特性。**NTC**（负温度系数）：阻值随温升下降；**PTC**：阻值随温升上升。

NTC电阻公式：
$$R_T = R_0 e^{\beta (\frac{1}{T} - \frac{1}{T_0})}$$

- $R_0$：热敏电阻在$T_0$（293K，室温）时的阻值
- $\beta$：材料常数（1500~6000K）

#### 辐射测温法（红外测温）

**原理**：任何有温度的物体以电磁波辐射能量，辐射能量与温度有关：
$$E = \sigma \varepsilon (T^4 - T_0^4)$$

**红外测温仪结构**：瞄准系统 → 聚光系统 → 红外检测器（热电堆）→ 信号处理 → 数字/画面显示

##### 红外体温计类型

| 类型 | 测量部位 | 特点 |
|------|----------|------|
| 皮肤红外温度计 | 额头 | 激光瞄准，1秒读数 |
| 红外耳鼓膜测温计 | 鼓膜 | 可近似反应基础温度 |
| 红外热成像仪 | 体表温度分布 | 二维图像显示温差，分辨率高达0.01℃ |

**影响因素**：皮肤血流、风速、辐射率ε、距离、环境辐射

**红外热成像仪**：扫描系统+聚焦系统+参考黑体+红外检测器
- 优点：大视场温度分布、高温差分辨率（0.01℃级）、直观方便
- 缺点：结构复杂、价格昂贵、准确度不如皮肤温度计

#### 化学测温法
变色测温贴片：分子结构改变使反射光颜色变化，单色/多色、可逆/不可逆。

## 第二节 围术期体温管理

### 体温改变对生理影响

| 体温降低 | 体温升高 |
|----------|----------|
| 氧耗量下降，无氧代谢增多 | 糖原分解增加，氧耗增加 |
| 呼吸减慢加深 | 代酸、高钾血症 |
| 窦房结传导减慢，心输出量下降 | 心肺负荷加重，心律失常 |
| 凝血功能紊乱 | 心肌缺血 |
| 肝代谢率及肝功能降低 | 水电解质失衡 |

### 热量散失机制

| 机制 | 说明 | 保护措施 |
|------|------|----------|
| 辐射 | 以电磁波形式发送热量 | 隔离毯、辐射加温仪 |
| 对流 | 空气循环流动带走过热 | 压缩热空气对流毯 |
| 传导 | 直接传递至手术台 | 加温毯、加热液体 |
| 蒸发 | 液体→气体带走热量 | 加温加湿、隔离毯 |

### 术中保温措施

| 措施 | 说明 |
|------|------|
| 控制环境温度 | 手术室≥23℃ |
| 被覆隔离 | 毛巾/手术单先行加热，形成停滞空气层 |
| 加热液体 | 冲洗液、血液、静脉输液加温（电子恒温水箱） |
| 加温加湿器 | 加热吸入气体 |
| 循环水变温毯 | 加热后的循环水提供热量（可升温可降温） |
| 红外辐射加温仪 | 波长2~3μm，能量被血液吸收经循环输送全身 |
| 压缩热空气对流毯 | 加热器+压缩机+鼓风机+保温毯，过滤99%微米级颗粒 |

### 红外辐射加温仪注意事项
- 选择靠近皮肤表面的动静脉吻合部位照射
- 避免灼伤，通过皮肤体温探头控制辐射能量
- 避开眼睛
