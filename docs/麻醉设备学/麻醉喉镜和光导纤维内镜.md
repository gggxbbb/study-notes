# 麻醉喉镜和光导纤维内镜

## 第一节 人工气道

**定义**：麻醉机或通气机呼吸气路与患者解剖气道之间最后一级管道连接的统称。

**结构**：
- **患者端**：与上呼吸道解剖形态相适应，密闭连接
- **气路端**：与呼吸气路连接，接口统一国际标准

### 一、面罩和鼻罩

| 类型 | 用途 |
|------|------|
| 解剖型面罩 | 现场急救通气 |
| 一次性面罩 | 日常麻醉使用 |
| 插管面罩 | 可在正压通气下进行纤维内镜操作 |
| 吸氧面罩 | 自主呼吸给氧 |
| 鼻罩 | 无创通气支持（慢性呼衰、睡眠治疗） |

### 二、通气道

| 类型 | 特点 |
|------|------|
| 口咽通气道 | 解除舌后坠，用于自主呼吸存在、咽反射不活跃的患者 |
| 鼻咽通气道 | 刺激小、易固定、可长时间保留 |
| 喉罩（LMA） | 环型气垫封闭食管和喉咽腔，经喉腔通气 |
| 喉管 | 盲探插入，双套囊封闭口咽腔和食管 |
| 食管气管联合导管 | 双腔双套囊，无论进入食管还是气管均可通气 |

#### 喉罩分类
- **普通喉罩**：硅胶，可反复使用50次以上
- **插管喉罩**：短而粗，可经导气管完成盲探气管内插管
- **双管喉罩**：有独立的胃液引流通道，更好保护气道安全性

### 三、气管内导管

**构成**：导气管 + 防漏套囊（cuff）+ 导管接头

**导气管**：要求有一定弹性和硬度，保持弧度，斜口左向（右鼻腔导管右向），侧孔（Murphy导管）

**防漏套囊**：充气密闭导管与气管间隙，防止漏气和误吸

**特殊气管导管**：
- 加强气管导管（spiral embedded tube）— 管壁含钢丝/尼龙骨架，可任意弯曲不折瘪
- 导向气管导管 — 壁内隧道穿尼龙线，拉线可使导管头内曲上翘，用于困难插管
- 低损伤气管导管
- 激光手术用气管导管 — 添加金属合成材料，防激光点燃

**气管切开导管**：经气管切开造口安置，适用于长期通气管理

### 四、支气管内导管

**用途**：肺隔离和单肺通气

**分类**：
- **单腔支气管导管**：细长，套囊短（左/右型）
- **支气管堵塞导管**：内置活动堵塞引流管，通气阻力小
- **双腔支气管导管**：
  - 卡伦斯（Carlens）双腔管 — 有隆突钩
  - 怀特（White）双腔管
  - 罗伯特肖（Robertshaw）双腔管 — 左型/右型，目前最常用

## 第二节 麻醉喉镜

**定义**：显示喉和声门，以便在明视下完成气管内插管的器械。

**组成**：喉镜片 + 镜柄

### 特殊喉镜

| 类型 | 角度 | 适应症 |
|------|------|--------|
| Alberts喉镜 | 67° | 颈部强直性过伸 |
| Polio喉镜 | 130°钝角 | 颌胸粘连致颈部过曲 |
| McCoy喉镜 | 前端活动镜片 | 困难气道 |

### 间接喉镜
- Bullard光导喉镜
- Wuscope光导喉镜
- 可视喉镜（视频喉镜）
- 光棒

## 第三节 光导纤维支气管镜

**组成**：冷光源 + 纤维内镜

**主要性能指标**：
- 视野角
- 视野方向
- 景深
- 内景光缆外径
- 有效长度
- 弯曲度
- 吸引介入管道内径
