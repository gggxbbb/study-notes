# 绪论

系统解剖学是研究正常人体形态结构的科学，按系统描述人体各器官的位置、形态、结构和功能。

## 标准姿势（解剖学姿势）

- 身体直立，两眼平视前方，上肢自然下垂于躯干两侧，手掌朝前，两足并拢，足尖向前
- 描述人体任何结构时均以此姿势为准

## 方位术语

| 术语 | 含义 |
|------|------|
| 上/下 | 近头者为上，近足者为下 |
| 前（腹侧）/后（背侧） | 近腹者为前，近背者为后 |
| 内侧/外侧 | 近正中面者为内侧 |
| 内/外 | 近内腔者为内——与"内侧/外侧"不同！ |
| 浅/深 | 近体表者为浅 |
| 近侧/远侧 | 近躯干者为近侧（用于四肢） |

**高频考点**：内侧≠内，远侧≠远端。

## 轴和面

- **垂直轴**：上下方向
- **矢状轴**：前后方向
- **冠状轴（额状轴）**：左右方向

| 面 | 切法 |
|----|------|
| 矢状面 | 沿矢状轴，分左右 |
| **正中矢状面** | 沿正中线，分左右相等的两半 |
| 冠状面（额状面） | 沿冠状轴，分前后 |
| 水平面（横切面） | 水平切，分上下 |

> 高频考点：标准姿势方向 vs 实际体位方向的转换是影像学诊断基础。
