# 肌肉系统总论

## 肌的构造和形态分类

- 每块骨骼肌由**肌腹**（红肌，收缩）+ **肌腱**（白腱，传导力）组成
- 形态分类：长肌（四肢）、短肌（躯干深层）、阔肌（躯干浅层）、轮匝肌（孔裂周围）

## 肌的起点和止点

- **起点**（定点）：肌在固定骨上的附着点，通常近躯干端
- **止点**（动点）：肌在移动骨上的附着点，通常远离躯干端
- 两端可因运动状态互换

## 肌的辅助装置

- **筋膜**：浅筋膜（皮下组织）+ 深筋膜（包绕肌群）
- **滑膜囊**：封闭结缔组织囊，含滑液，减少摩擦（如髌上囊）
- **腱鞘**：包绕肌腱的双层套管，含滑液，手指腕部多见→腱鞘炎
