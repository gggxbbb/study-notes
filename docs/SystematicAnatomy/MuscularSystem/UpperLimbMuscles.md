# 上肢肌

## 上肢带肌

6块肌围在肩关节周围（三角肌、冈上肌、冈下肌、小圆肌、大圆肌、肩胛下肌）。

- **三角肌**：起自锁骨外1/3、肩峰和肩胛冈，止于肱骨三角肌粗隆→外展肩关节。肌内注射常用部位（上臂外侧上1/3）
- **冈上肌**：外展肩关节起始阶段→冈上肌腱炎常见
- 冈下肌 + 小圆肌 → 外旋肩关节
- 肩胛下肌 → 内旋肩关节

## 臂肌

- **前群**（屈肌）：肱二头肌（屈肘+前臂旋后，临床上测腱反射）、喙肱肌、肱肌
- **后群**（伸肌）：肱三头肌（伸肘）

## 前臂肌

- **前群**（屈肌+旋前，9块）：浅层从桡侧到尺侧为肱桡肌、旋前圆肌、桡侧腕屈肌、掌长肌、尺侧腕屈肌……
- **后群**（伸肌+旋后，10块）：浅层从桡侧到尺侧为桡侧腕长伸肌、桡侧腕短伸肌、指伸肌……

## 手肌

- 外侧群（大鱼际）：拇短展肌、拇短屈肌、拇对掌肌、拇收肌→拇指精细运动
- 内侧群（小鱼际）：小指展肌、小指短屈肌、小指对掌肌→小指运动
- 中间群：蚓状肌、骨间掌侧肌/背侧肌
