# 下肢肌

## 髋肌

- **前群**：髂腰肌（腰大肌+髂肌）→屈髋关节
- **后群**：臀大肌（伸髋、外旋）、臀中肌+臀小肌（外展髋）、梨状肌（经坐骨大孔，坐骨神经多经其下缘出盆→梨状肌综合征）

## 大腿肌

- **前群**：股四头肌（伸膝、屈髋，人体最大肌）、缝匠肌（人体最长肌，"裁缝姿势"：屈髋+屈膝+外旋）
- **内侧群**：内收肌群（股薄肌、耻骨肌、长收肌、短收肌、大收肌）→髋内收
- **后群**：股二头肌、半腱肌、半膜肌（合称腘绳肌）→伸髋+屈膝
- **髌韧带**：股四头肌腱包绕髌骨后向下止于胫骨粗隆→膝跳反射

## 小腿肌

- **前群**：胫骨前肌等 → 足背屈
- **外侧群**：腓骨长肌、腓骨短肌 → 足外翻
- **后群**：
  - 浅层→**小腿三头肌**（腓肠肌+比目鱼肌）→跖屈（提足跟）
  - 其肌腱汇成**跟腱**止于跟骨，人体最粗大肌腱→跟腱反射

> 口诀：大腿前股四内收后腘绳，小腿前背屈外外翻后跖屈
