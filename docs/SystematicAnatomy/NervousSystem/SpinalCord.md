# 脊髓

## 位置和外形

- 位置：椎管内，上端经枕骨大孔连延髓，下端成人平L1下缘（新生儿平L3）
- 外形：前后略扁圆柱形。颈膨大（C4~T1，上肢神经起端）和腰骶膨大（L2~S3，下肢神经起端）
- **脊髓圆锥**：脊髓下端变细
- **马尾**：腰骶神经根在椎管内垂直下行，围绕终丝形成
- **脊髓节段**：共31节（C8 + T12 + L5 + S5 + Co1）。脊髓节段与椎骨的对应关系：

| 脊髓节段 | 对应椎骨 |
|---------|---------|
| C1~4 | = 同序数椎骨 |
| C5~T4 | 椎骨序数 - 1 |
| T5~8 | 椎骨序数 - 2 |
| T9~12 | 椎骨序数 - 3 |
| L1~5 | 平对T10~12 |
| S1~Co1 | 平对L1 |

## 内部结构

**灰质**（中央，蝶形）：
- **前角**：运动神经元（α→梭外肌收缩，γ→肌梭敏感度）
- **后角**：感觉神经元（边缘核、胶状质→痛温觉，后角固有核→触觉，胸核→本体感觉）
- **侧角**（T1~L3）：交感节前神经元细胞体
- **骶副交感核**（S2~4）：副交感节前神经元

**白质**（周围，分前索/外侧索/后索）：

| 纤维束 | 位置 | 功能 | 损伤表现 |
|--------|------|------|---------|
| **薄束、楔束** | 后索 | 同侧本体感觉+精细触觉上行 | 同侧本体感觉丧失 |
| **脊髓丘脑束** | 前索+外侧索 | 对侧痛温觉+粗略触觉上行 | 对侧痛温觉丧失 |
| **皮质脊髓束** | 外侧索+前索 | 同侧骨骼肌随意运动 | 同侧瘫痪（痉挛性） |

> 白质前连合：左右交叉处→脊髓空洞症首先累及此处→双侧痛温觉丧失（感觉分离）
