# 脊神经

共31对：颈8、胸12、腰5、骶5、尾1。每条脊神经由前根（运动）+ 后根（感觉）合成，出椎间孔后分前支和后支。

## 颈丛（C1~4）

- 皮支（枕小、耳大、颈横、锁骨上神经）→头颈部和肩部皮肤
- **膈神经**（C3~5，主要C4）：混合神经→唯一支配膈肌的运动神经。损伤→同侧膈肌瘫痪、呼吸困难

## 臂丛（C5~T1）

主要分支损伤表现（高频）：

| 神经 | 支配 | 损伤表现 |
|------|------|---------|
| **腋神经** | 三角肌 | 三角肌萎缩→"方肩"、臂不能外展 |
| **肌皮神经** | 肱二头肌等 | 屈肘减弱 |
| **桡神经** | 前臂后群肌 | **"垂腕症"**（伸腕、伸指障碍），肱骨中段骨折合并损伤 |
| **正中神经** | 前臂前群（尺侧腕屈肌和指深屈肌尺侧半以外） | **"猿手"**（拇指不能对掌、鱼际萎缩） |
| **尺神经** | 手内侧肌群 | **"爪形手"**（小鱼际萎缩、4/5指不能内收外展） |
| **胸长神经** | 前锯肌 | **"翼状肩胛"**（肩胛骨不能紧贴胸壁） |

> 口诀：桡垂腕、正猿手、尺爪形、腋方肩

## 胸神经前支（T1~12）

节段性分布（脊神经皮节）：
- T2 → 胸骨角平面
- **T4 → 乳头平面**
- T6 → 剑突平面
- T8 → 肋弓下缘平面
- **T10 → 脐平面**
- T12 → 耻骨联合与脐连线中点

## 腰丛（T12~L4）

| 神经 | 支配 | 损伤表现 |
|------|------|---------|
| **股神经** | 股四头肌、缝匠肌 | 不能伸膝、膝跳反射消失 |
| 闭孔神经 | 大腿内收肌群 | 大腿内收障碍 |

## 骶丛（L4~S5）

| 神经 | 支配 | 损伤表现 |
|------|------|---------|
| **坐骨神经** | 股后群肌 + 小腿和足全部肌肉 | 全身最粗大神经 |
| **胫神经** | 小腿后群肌 | 不能跖屈、足内翻→"仰趾足" |
| **腓总神经** | 小腿前群和外侧群肌 | **"马蹄内翻足"**（足下垂、不能背屈） |
