# 内脏神经

分布：内脏、心血管和腺体。分内脏运动神经和内脏感觉神经。

## 内脏运动神经（自主神经）

与躯体运动神经的区别：不受意识直接控制、支配平滑肌/心肌/腺体、从低级中枢至效应器要换神经元（节前纤维→自主神经节→节后纤维）

### 交感神经

| 项目 | 内容 |
|------|------|
| 低级中枢 | 脊髓T1~L3侧角（中间外侧核） |
| 神经节 | 椎旁节（交感干21~23对+奇神经节）+ 椎前节 |
| 节前纤维 | 短——至交感干换元 |
| 节后纤维 | 长——至效应器 |
| 分布 | 广泛——全身所有器官 |

### 副交感神经

| 项目 | 内容 |
|------|------|
| 低级中枢 | 脑干副交感核（动眼、面、舌咽、迷走神经副交感核）+ 脊髓S2~4骶副交感核 |
| 神经节 | 终节（壁内节、器官旁节） |
| 节前纤维 | 长——至终节换元 |
| 节后纤维 | 短——至效应器 |
| 分布 | 局限——仅分布于头颈胸腹盆腔脏器 |

### 交感和副交感的区别

| | 交感 | 副交感 |
|---|------|--------|
| 低级中枢 | T1~L3 | 脑干 + S2~4 |
| 神经节位置 | 近中枢 | 近/在效应器 |
| 节前/后纤维 | 前短后长 | 前长后短 |
| 功能 | "战斗或逃跑"（心率↑、瞳孔散大、支气管扩张…） | "休息和消化"（心率↓、瞳孔缩小、腺体分泌↑…） |

## 内脏感觉神经

特点：阈值高、定位模糊、常伴情感反应。

**牵涉性痛**：内脏病变引起特定体表区域疼痛。
- 心绞痛→左臂尺侧（T1~2）
- 胆囊炎→右肩胛区
- 阑尾炎早期→脐周（T10）
