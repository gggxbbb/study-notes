# 端脑

## 外形

三条主要沟：外侧沟、**中央沟**（额叶和顶叶分界）、**顶枕沟**。

五叶：
- 额叶（中央沟前方、外侧沟上方）
- 顶叶（中央沟后方、顶枕沟前方）
- 枕叶（顶枕沟后方）
- 颞叶（外侧沟下方）
- 岛叶（外侧沟深处）

## 基底核

| 结构 | 组成 |
|------|------|
| **纹状体** | 尾状核 + 豆状核（壳 + 苍白球） |
| 新纹状体 | 尾状核 + 壳 |
| 旧纹状体 | 苍白球 |

纹状体：锥体外系重要组成部分→肌张力调节、运动协调。帕金森病→黑质-纹状体通路多巴胺↓→肌张力增高、静止性震颤。

## 大脑髓质

- **连合纤维**：胼胝体（左右大脑半球最大连合）
- **内囊**：位于尾状核、背侧丘脑与豆状核之间，上下行纤维束集中通过的区域

| 内囊分部 | 通过纤维 | 损伤表现 |
|---------|---------|---------|
| 前肢 | 额桥束、丘脑前辐射 | — |
| **膝** | **皮质核束** | 对侧面下部及舌肌瘫痪（中枢性面瘫） |
| **后肢** | **皮质脊髓束** + 丘脑中央辐射 + 视辐射 + 听辐射 | **"三偏综合征"**：对侧偏瘫 + 对侧偏身感觉障碍 + 双眼对侧同向偏盲 |

## 大脑皮质功能定位

| 功能区 | 位置 | 损伤表现 |
|--------|------|---------|
| **第I躯体运动区** | 中央前回 + 中央旁小叶前部 | 对侧肢体瘫痪 |
| **第I躯体感觉区** | 中央后回 + 中央旁小叶后部 | 对侧躯体感觉障碍 |
| **视区** | 距状沟两侧皮质 | 视野缺损 |
| **听区** | 颞横回 | 听力减退（不完全失聪，因接受双侧投射） |
| **运动性语言区（Broca区）** | 额下回后部（优势半球） | 运动性失语（能听懂但不能说） |
| **感觉性语言区（Wernicke区）** | 颞上回后部（优势半球） | 感觉性失语（能说但听不懂自己/别人说什么） |

> 口诀：动前感后视枕听颞横，Broca额下说、Wernicke颞上听
