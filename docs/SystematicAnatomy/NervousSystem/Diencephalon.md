# 间脑

间脑：位于中脑与端脑之间，包括背侧丘脑、后丘脑、上丘脑、下丘脑、底丘脑。

## 背侧丘脑（丘脑）

- 卵圆形灰质团块，中脑水管前端脑干两侧
- 内部被"Y"形内髓板分为三大核群：

| 核群 | 主要核团 | 功能 |
|------|---------|------|
| 前核群 | — | 边缘系统相关 |
| 内侧核群 | 背内侧核 | 联络核群 |
| 外侧核群 | **腹后内侧核**（头面部感觉）、**腹后外侧核**（躯干四肢感觉） | 特异性中继核→所有感觉（除嗅觉）在此换元后投射至大脑皮质 |

> 丘脑是感觉传导的"最后一站中继"

## 后丘脑

- **内侧膝状体**：听觉中继核 → 听辐射 → 颞横回（听区）
- **外侧膝状体**：视觉中继核 → 视辐射 → 距状沟（视区）

## 下丘脑

- 位置：背侧丘脑前下方，构成第三脑室下壁和侧壁
- 主要核团：

| 核团 | 功能 |
|------|------|
| **视上核** | 合成加压素（ADH）→ 视上垂体束 → 神经垂体 |
| **室旁核** | 合成催产素 → 室旁垂体束 → 神经垂体 |
| 漏斗核 | 调节腺垂体分泌 |

- 功能：自主神经高级中枢（体温、摄食、饮水、昼夜节律、内分泌）

## 第三脑室

位于两侧间脑之间的矢状位窄隙，前借室间孔通侧脑室，后借中脑水管通第四脑室。
