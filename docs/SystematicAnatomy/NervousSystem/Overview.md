# 神经系统总论

## 分部

- **中枢神经系统**：脑 + 脊髓
- **周围神经系统**：脑神经（12对）+ 脊神经（31对）+ 内脏神经

## 基本概念

| 概念 | 位置 | 含义 |
|------|------|------|
| **灰质** | 中枢 | 神经元胞体和树突聚集处（脊髓中央、大脑皮质和小脑皮质） |
| **白质** | 中枢 | 神经纤维聚集处 |
| **皮质** | 大脑/小脑表面 | 灰质形成的层次结构 |
| **神经核** | 中枢白质内 | 形态功能相似的神经元胞体聚集 |
| **神经节** | 周围 | 形态功能相似的神经元胞体聚集 |
| **纤维束** | 中枢白质内 | 起止、经行、功能相同的神经纤维 |
| **神经** | 周围 | 神经纤维聚集成束，外裹结缔组织 |
| **网状结构** | 脑干 | 灰白质交织区 |

## 神经元的分类

- 按突起数目：假单极神经元（如脊神经节细胞）、双极神经元（如视网膜双极细胞）、多极神经元（绝大多数）
- 按功能：感觉神经元（传入）、运动神经元（传出）、联络神经元（中间神经元）

## 反射弧

感受器 → 传入神经元 → 中枢 → 传出神经元 → 效应器
