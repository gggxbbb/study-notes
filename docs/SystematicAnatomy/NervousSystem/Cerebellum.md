# 小脑

## 位置和外形

- 位置：颅后窝，脑桥和延髓后方
- 外形：中间缩窄（小脑蚓）+两侧膨大（小脑半球）
- 三对小脑脚：小脑下脚（连延髓）、小脑中脚（连脑桥）、小脑上脚（连中脑）

## 分部与功能

| 分部 | 从发生/进化 | 功能 | 损伤表现 |
|------|-----------|------|---------|
| 绒球小结叶 | 前庭小脑（古小脑） | 维持平衡 | 平衡障碍、站立不稳 |
| 小脑前叶（蚓部） | 脊髓小脑（旧小脑） | 调节肌张力 | 肌张力异常 |
| 小脑后叶（半球） | 皮质小脑（新小脑） | 协调随意运动 | **意向性震颤**、轮替运动障碍、辨距不良 |

## 小脑核

- 顶核、球状核、栓状核、**齿状核**（最大、最重要的传出核）

> 小脑损伤三联征：意向性震颤 + 肌张力减低 + 共济失调
