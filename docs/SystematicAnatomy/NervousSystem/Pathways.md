# 主要传导通路

## 上行传导通路（感觉）

### 躯干/四肢本体感觉和精细触觉

1级（脊神经节）→ 2级（薄束核/楔束核，交叉→内侧丘系交叉）→ 3级（丘脑腹后外侧核→内囊后肢）→ 中央后回上2/3 + 中央旁小叶后部

### 躯干/四肢痛温觉和粗触觉

1级（脊神经节）→ 2级（脊髓后角固有核，交叉至对侧→脊髓丘脑束）→ 3级（丘脑腹后外侧核→内囊后肢）→ 中央后回上2/3

### 头面部痛温觉和触觉

1级（三叉神经节）→ 2级（三叉神经脑桥核/脊束核，交叉）→ 3级（丘脑腹后内侧核→内囊后肢）→ 中央后回下1/3

### 视觉通路

视网膜节细胞 → 视神经 → 视交叉（鼻侧交叉、颞侧不交叉）→ 视束 → 外侧膝状体 → 视辐射（内囊后肢）→ 距状沟

**瞳孔对光反射通路**：视网膜 → 视神经 → 视交叉 → 视束 → 上丘臂 → 顶盖前区 → 双侧动眼神经副核 → 动眼神经 → 睫状神经节 → 瞳孔括约肌（缩瞳）

> 损伤定位：
> - 一侧视神经损伤 → 同侧眼全盲
> - 视交叉中部损伤（垂体瘤）→ 双眼颞侧偏盲
> - 一侧视束/视辐射/视皮质损伤 → 双眼对侧同向偏盲

## 下行传导通路（运动）

### 锥体系

- **皮质脊髓束**：中央前回上2/3 → 内囊后肢 → 锥体交叉（延髓）→ 皮质脊髓侧束（始终对侧）/前束（在脊髓交叉至对侧）→ 脊髓前角运动神经元
- **皮质核束**：中央前回下1/3 → 内囊膝 → 脑神经运动核（双侧支配大多数核团，但面神经核下部和舌下神经核只接受对侧支配）
  - 损伤：对侧下部面肌和对侧舌肌瘫痪

| | 上运动神经元损伤 | 下运动神经元损伤 |
|---|-------------|-------------|
| 瘫痪类型 | 痉挛性（硬瘫） | 弛缓性（软瘫） |
| 肌张力 | 增高 | 降低 |
| 腱反射 | 亢进 | 减弱/消失 |
| 病理反射 | 阳性（Babinski征+） | 阴性 |
| 肌萎缩 | 无/轻度（废用性） | 明显 |
