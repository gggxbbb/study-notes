# 脑和脊髓的被膜与血管

## 被膜（三层）

| 层次 | 脊髓 | 脑 |
|------|------|-----|
| 硬膜（外层） | 硬脊膜 → **硬膜外隙**（含静脉丛和脂肪，硬膜外麻醉注药部位） | 硬脑膜 + 硬脑膜窦 |
| 蛛网膜（中层） | 脊髓蛛网膜 → **蛛网膜下隙**（含脑脊液，腰穿部位） | 脑蛛网膜 + 蛛网膜下池 |
| 软膜（内层） | 软脊膜 → 齿状韧带（固定脊髓） | 软脑膜 + 脉络丛（产生脑脊液） |

**终池**：脊髓下端至S2水平，蛛网膜下隙扩大，含马尾和终丝→腰穿安全部位（L3~4或L4~5）。

### 硬脑膜窦

| 窦 | 位置 |
|----|------|
| 上矢状窦 | 大脑镰上缘 |
| 下矢状窦 | 大脑镰下缘 |
| 直窦 | 大脑镰 + 小脑幕交界 |
| **海绵窦** | 蝶鞍两侧。穿行：颈内动脉 + 展神经。外侧壁：动眼/滑车/眼/上颌神经 → 面部感染可逆行入海绵窦 |
| 横窦 → 乙状窦 | 经颈静脉孔出颅 → 颈内静脉 |

硬脑膜窦血液流向：上矢状窦 → 窦汇 → 横窦 → 乙状窦 → 颈内静脉

**小脑幕切迹**：小脑幕前缘游离，与鞍背围成→幕切迹疝（海马旁回/钩疝入→动眼神经受压+大脑脚受压）

## 脑脊液循环

侧脑室脉络丛产生 → 室间孔 → 第三脑室（+脉络丛） → 中脑水管 → 第四脑室（+脉络丛） → 正中孔/外侧孔 → 蛛网膜下隙 → 蛛网膜粒 → 上矢状窦 → 回流入血

> 梗阻性脑积水：中脑水管堵塞→侧脑室和第三脑室扩张

## 脑的动脉

两大来源：
- **颈内动脉**（前循环）：大脑前动脉、大脑中动脉、后交通动脉
- **椎-基底动脉**（后循环）：大脑后动脉

**大脑动脉环（Willis环）**：前交通动脉 + 大脑前动脉 + 颈内动脉末端 + 后交通动脉 + 大脑后动脉。前、后循环的侧支吻合 → 脑血管意外时的重要代偿途径。

大脑中动脉：经过**内囊**的穿支→高血压脑出血最常见血管（"卒中动脉"）

## 脊髓的动脉

- 椎动脉分支 → 1条脊髓前动脉 + 2条脊髓后动脉
- 节段性动脉（肋间后动脉、腰动脉等）分支补充
