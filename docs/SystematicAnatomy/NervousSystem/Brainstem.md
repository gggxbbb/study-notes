# 脑干

脑干：中脑 + 脑桥 + 延髓，上接间脑、下连脊髓、后邻小脑。

## 外形

**腹侧面**：
- 延髓：锥体（皮质脊髓束）→锥体交叉；橄榄
- 脑桥：基底部膨隆，借小脑中脚连小脑
- 中脑：大脑脚、脚间窝→动眼神经出脑

**背侧面**：
- 延髓：薄束结节（薄束核）、楔束结节（楔束核）
- 脑桥：结合臂（小脑上脚）→菱形窝上半
- 中脑：四叠体（上丘→视觉反射，下丘→听觉反射）

**第四脑室**：延髓 + 脑桥 + 小脑之间，向上→中脑水管，向下→脊髓中央管，借三个孔通蛛网膜下隙

## 脑神经核

| 功能柱 | 核团（举例） | 所在部位 |
|--------|------------|---------|
| 躯体运动柱 | 动眼神经核、滑车神经核、展神经核、舌下神经核 | 中脑-延髓 |
| 特殊内脏运动柱 | 三叉神经运动核、面神经核、疑核、副神经核 | 脑桥-延髓 |
| 一般内脏运动柱 | 动眼神经副核、上/下泌涎核、迷走神经背核 | 中脑-延髓 |
| 一般躯体感觉柱 | 三叉神经中脑核、脑桥核、脊束核 | 中脑-延髓 |
| 特殊躯体感觉柱 | 蜗神经核、前庭神经核 | 脑桥 |
| 一般内脏感觉柱 | 孤束核 | 延髓 |

## 重要传导中继核

- **薄束核 + 楔束核**：本体感觉和精细触觉第2级神经元→发出内侧丘系交叉
- **黑质**：中脑，多巴胺能神经元→帕金森病
- **上丘**：视觉反射中枢
- **下丘**：听觉反射中枢

## 重要纤维束

| 纤维束 | 起止 | 功能 |
|--------|------|------|
| **内侧丘系** | 薄束核/楔束核 → 丘脑 | 对侧本体感觉 |
| **脊髓丘系** | 脊髓→丘脑 | 对侧痛温觉 |
| **三叉丘系** | 三叉神经核→丘脑 | 对侧面部感觉 |
| **锥体束** | 大脑皮质→脊髓/脑干运动核 | 随意运动 |
