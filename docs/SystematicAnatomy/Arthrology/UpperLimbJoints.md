# 上肢骨的连结

## 上肢带骨的连结

- **胸锁关节**：锁骨胸骨端 + 胸骨柄，上肢与躯干的唯一骨性连结
- **喙肩弓**：喙突 + 喙肩韧带 + 肩峰，保护肱骨头

## 自由上肢骨的连结

- **肩关节**：肱骨头 + 肩胛骨关节盂，球窝关节
  - 关节盂浅小、关节囊松弛→活动度大（全身最灵活的关节）
  - 肱二头肌长头腱穿过关节腔
  - 前下方最薄弱→肩关节前下脱位最常见
- **肘关节**：肱尺关节 + 肱桡关节 + 桡尺近侧关节，复关节
  - 屈伸运动
  - 幼儿桡骨头环状韧带发育不全→牵拉手时易发生**桡骨头半脱位**（牵拉肘）
- **桡腕关节**（腕关节）：桡骨下端 + 腕骨近侧列（舟、月、三角骨），椭圆关节
- **拇指腕掌关节**：大多角骨 + 第1掌骨底，鞍状关节→对掌运动
