# 躯干骨的连结

## 脊柱的连结

- **椎间盘**：相邻椎体之间，纤维环 + 髓核，缓冲震荡。纤维环破裂髓核突出→椎间盘突出症
- **黄韧带**：连接相邻椎弓板，协助围成椎管后壁
- **棘间韧带、棘上韧带**：连接棘突
- **脊柱整体观**：4个生理弯曲——颈曲（前凸）、胸曲（后凸）、腰曲（前凸）、骶曲（后凸）
- **寰枢关节**：寰椎 + 枢椎，旋转运动

## 胸廓的连结

- 组成：12块胸椎 + 12对肋 + 1块胸骨
- **胸廓上口**：第1胸椎 + 第1对肋 + 胸骨柄上缘
- **胸廓下口**：第12胸椎 + 第11~12对肋 + 肋弓 + 剑突
- **胸骨下角**：两侧肋弓的夹角
