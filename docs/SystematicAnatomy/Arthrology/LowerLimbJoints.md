# 下肢骨的连结

## 下肢带骨的连结

骨盆：骶骨 + 尾骨 + 左右髋骨，借骶髂关节、耻骨联合、骶结节韧带、骶棘韧带连结。

- 以小骨盆入口*界线*分大骨盆（上）和小骨盆（下）
- **界线**：骶岬 → 弓状线 → 耻骨梳 → 耻骨结节 → 耻骨联合上缘
- **骨盆上口** = 界线
- **骨盆下口**：尾骨尖 + 骶结节韧带 + 坐骨结节 + 耻骨弓

**骨盆性别差异**：

| | 男性 | 女性 |
|---|------|------|
| 外形 | 窄长 | 宽短 |
| 上口 | 心形 | 近圆形 |
| 下口 | 窄小 | 宽大 |
| 耻骨下角 | 70°~75° | 90°~100° |

**骶结节韧带 + 骶棘韧带** → 围成坐骨大孔、坐骨小孔。

## 自由下肢骨的连结

- **髋关节**：股骨头 + 髋臼，杵臼关节
  - 髋臼深、关节囊坚韧、股骨头韧带→稳定性极高
  - 股骨颈骨折最常见部位
- **膝关节**：股骨下端 + 胫骨上端 + 髌骨，人体最大最复杂的关节
  - **半月板**：内侧半月板"C"形（大）、外侧半月板"O"形（小），缓冲震荡
  - **前交叉韧带**：防止胫骨前移
  - **后交叉韧带**：防止胫骨后移
  - **侧副韧带**：内侧副韧带与关节囊融合→损伤常见
- **距小腿关节**（踝关节）：胫腓骨下端 + 距骨滑车
  - 背屈（足背上抬）、跖屈（足尖下垂）
  - 跖屈时距骨滑车窄部进入关节窝→不稳定→踝内翻扭伤最常见
- **足弓**：跗骨+跖骨借韧带形成的弓形结构，缓冲震荡
