# 输尿管道

## 输尿管

三个生理性狭窄（结石最易嵌顿）：

| 狭窄 | 位置 |
|------|------|
| 第一狭窄 | 肾盂与输尿管移行处 |
| 第二狭窄 | 越过髂血管处（小骨盆入口） |
| 第三狭窄 | 壁内段（穿膀胱壁） |

> 口诀：肾盂输尿管移行、跨髂血管处、穿膀胱壁

## 膀胱

- 位置：盆腔前部，耻骨联合后方
- 形态（内面）：空虚时粘膜形成皱襞，底部光滑三角形区域→**膀胱三角**：
  - 围成：两侧输尿管口 + 尿道内口
  - 特点：无粘膜下层，始终光滑
  - 临床意义：肿瘤、结核和膀胱镜检查的重要区域
- 充盈时超出耻骨联合上缘，可经腹壁行耻骨上膀胱穿刺

## 女性尿道

- 特点：短（3~5cm）、宽、直，开口于阴道前庭（尿道外口位于阴道口前上方）
- 临床：比男性更易发生逆行性尿路感染
