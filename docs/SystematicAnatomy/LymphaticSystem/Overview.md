# 淋巴系统

## 淋巴管道

- 九条淋巴干（左右颈干、左右锁骨下干、左右支气管纵隔干、左右腰干、肠干）
- **胸导管**：起自乳糜池（L1前方）→向上经主动脉裂孔入胸腔→沿脊柱右前方上行→在颈根部注入**左静脉角**。收集：双下肢、腹盆部、左半胸、左上肢、左半头颈（全身3/4的淋巴）
- **右淋巴导管**：收集：右半头颈、右上肢、右半胸→注入**右静脉角**

## 淋巴器官

- **脾**：左季肋区，第9~11肋深面，长轴与第10肋一致。最大淋巴器官→滤血、免疫、储血。脾门邻胰尾
- 淋巴结：沿淋巴管分布，具有滤过淋巴和免疫功能

## 重要淋巴结群

- **锁骨上淋巴结（Virchow淋巴结）**：左锁骨上淋巴结肿大→胃癌/食管癌腹腔转移的早期征象（Virchow转移）
- **腋淋巴结群**→乳腺癌转移首要途径；**腹股沟浅淋巴结**→下肢和外生殖器感染/肿瘤
- **胃的淋巴回流**：贲门→胃左淋巴结，幽门→幽门淋巴结→腹腔淋巴结→肠干
- **肺的淋巴回流**：肺内淋巴结→肺门淋巴结→气管支气管淋巴结→支气管纵隔干
- **子宫的淋巴回流**：宫底和宫体上部→腰淋巴结，宫体下部和宫颈→髂内/外淋巴结→骶淋巴结
