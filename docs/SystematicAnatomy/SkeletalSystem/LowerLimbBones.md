# 下肢骨

## 下肢带骨——髋骨

- 由髂骨（上部）、坐骨（后下部）、耻骨（前下部）三骨汇合于髋臼
- **髂骨**：髂嵴、髂前上棘/髂后上棘——骨髓穿刺常用部位
- **坐骨**：坐骨结节——坐位承重点
- **耻骨**：耻骨联合面

## 自由下肢骨

- **股骨**：上端（**股骨头**、**股骨颈**←骨折好发部位、大/小转子）、体（粗线、臀肌粗隆）、下端（内/外侧髁）
- **胫骨**：上端（内/外侧髁、**胫骨粗隆**←髌韧带附着）、下端（**内踝**）
- **腓骨**：上端（腓骨头）、下端（**外踝**）、不参与膝关节组成

跗骨7块：距骨（与胫腓骨下端构成踝关节）、跟骨（最大）、舟骨、骰骨、楔骨×3

> 口诀：距跟舟骰楔×3
