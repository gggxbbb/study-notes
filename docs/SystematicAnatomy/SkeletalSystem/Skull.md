# 颅骨

颅的组成：脑颅（8块）+ 面颅（15块）。

## 脑颅骨

**不成对**：额骨、筛骨、蝶骨、枕骨
**成对**：顶骨、颞骨

## 面颅骨

**不成对**：下颌骨、犁骨、舌骨
**成对**：上颌骨、鼻骨、泪骨、颧骨、腭骨、下鼻甲

## 颅的整体结构

- **颅盖外面**：冠状缝、矢状缝、人字缝
- **颅底内面**：颅前窝 → 颅中窝 → 颅后窝（阶梯状加深）
  - 颅中窝有**垂体窝**和**视神经管**
  - 颅后窝有**枕骨大孔**
- **翼点**：额、顶、颞、蝶四骨交汇处，骨质薄弱，内面有**脑膜中动脉前支**经过→颅外伤易致硬膜外血肿
- **眶**：锥体形腔，借视神经管通颅中窝
- **骨性鼻腔**：位于面颅中央，借鼻后孔通咽
- **鼻旁窦**：均开口于鼻腔——额窦、筛窦、蝶窦、上颌窦

## 新生儿颅的特征

- **颅囟**：新生儿颅骨未完全骨化的膜性间隙
- **前囟**（额囟）：菱形，1~2岁闭合（最晚闭合）
- **后囟**（枕囟）：三角形，出生后2~3个月闭合

> 口诀：脑颅八块顶额筛蝶枕，面颅十五上鼻颧泪腭甲下犁舌
