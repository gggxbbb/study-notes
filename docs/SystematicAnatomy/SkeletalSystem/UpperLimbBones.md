# 上肢骨

## 上肢带骨

- **锁骨**：横架于胸廓前上方，"S"形弯曲，内侧2/3凸向前，外侧1/3凸向后。易骨折（中1/3最薄弱）
- **肩胛骨**：三角形扁骨，位于胸廓后外侧上份（第2~7肋间）
  - 三角：上角、下角（平对第7肋）、外侧角（有关节盂）
  - 三缘：上缘（喙突）、内侧缘、外侧缘
  - 两面：前面（肩胛下窝）、后面（肩胛冈 → 肩峰）

## 自由上肢骨

- **肱骨**：上端（肱骨头、解剖颈、大/小结节、**外科颈**←骨折好发）、体（三角肌粗隆、**桡神经沟**←肱骨中段骨折可损伤桡神经）、下端（肱骨小头、肱骨滑车、鹰嘴窝、内/外上髁）
- **尺骨**：上端（鹰嘴、冠突、滑车切迹、桡切迹）、下端（尺骨头、茎突）
- **桡骨**：上端（桡骨头、桡骨颈）、下端（桡骨茎突、腕关节面）

腕骨8块口诀：**舟月三角豆，大小头状钩**（近侧列：手舟骨、月骨、三角骨、豌豆骨；远侧列：大多角骨、小多角骨、头状骨、钩骨）
