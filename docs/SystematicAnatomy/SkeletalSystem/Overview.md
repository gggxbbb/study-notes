# 骨骼系统总论

## 骨的基本形态及分类

- **长骨**：长管状，分一体两端（骨干 + 骨骺），分布于四肢
- **短骨**：立方体，分布于腕、跗部
- **扁骨**：板状，构成颅腔、胸腔壁
- **不规则骨**：形态不规则，如椎骨
- 骺线：骨骺与骨干愈合后遗留的线状痕迹

## 骨的构造

- **骨质**：骨密质 + 骨松质
- **骨膜**：覆盖除关节面外的骨表面，含血管神经，对骨营养和再生重要
- **骨髓**：红骨髓（造血）→ 黄骨髓（脂肪），成人仅椎骨、髂骨、胸骨、肋骨等处保留红骨髓
- 板障：颅盖骨内、外板之间的骨松质

## 骨质的化学成分与物理性质

- 有机质（胶原纤维）：弹性、韧性
- 无机质（钙盐）：硬度
- 幼年有机质多 → 弹性大、不易骨折；老年无机质多 → 脆性大、易骨折

## 骨的发生

- 膜内成骨（如颅盖骨）
- 软骨内成骨（如四肢骨）
