# 躯干骨

躯干骨组成——椎骨（24块）、骶骨（1块）、尾骨（1块）、肋骨（12对）、胸骨（1块），共51块。

## 椎骨

椎骨基本形态：
- **椎体**：前方，负重
- **椎弓**：后方，围成椎孔（连成椎管）
- 突起：棘突（1个）、横突（1对）、上/下关节突（各1对）

各部椎骨主要特征：

| 部位 | 特征 |
|------|------|
| 颈椎 | 横突孔、棘突分叉（C2~6） |
| 寰椎(C1) | 无椎体，前弓+后弓 |
| 枢椎(C2) | 齿突 |
| 隆椎(C7) | 棘突特长，体表可触及 |
| 胸椎 | 肋凹（上/下肋凹 + 横突肋凹） |
| 腰椎 | 椎体肥大，棘突板状水平后伸 |

## 肋

- 真肋（1~7对，直接连胸骨）
- 假肋（8~10对，借肋软骨连上位肋软骨）
- 浮肋（11~12对，前端游离）
- **肋弓**：第8~10肋软骨依次连接形成

## 胸骨

- 分胸骨柄、胸骨体、剑突三部
- **胸骨角**：胸骨柄与胸骨体连接处，平对第2肋，计数肋骨的标志。平对T4下缘、主动脉弓起止、气管杈
