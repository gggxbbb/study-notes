# 消化管

## 口腔

- **咽峡**：腭垂、两侧腭舌弓和舌根围成，口腔与咽的分界
- **舌乳头**：丝状乳头（最多，无味蕾）、菌状乳头、轮廓乳头、叶状乳头。轮廓乳头和菌状乳头含味蕾
- **颏舌肌**：起自下颌骨颏棘，止于舌中线两侧。单侧收缩→舌尖偏向对侧；双侧同时收缩→伸舌
- **牙**：乳牙20颗（6月~2.5岁出齐）、恒牙32颗（6岁起替换）。牙周组织：牙龈、牙周膜、牙槽骨
- **三大唾液腺**：

| 腺体 | 导管开口 |
|------|----------|
| 腮腺 | 上颌第2磨牙颊粘膜 |
| 下颌下腺 | 舌下阜 |
| 舌下腺 | 舌下阜 + 舌下襞 |

## 咽

- 消化与呼吸的共同通道，C1~6椎体前方
- 分部：鼻咽（通鼻腔 + 咽鼓管→中耳）、口咽（通口腔）、喉咽（通喉腔）
- **咽淋巴环**：咽扁桃体 + 咽鼓管扁桃体 + 腭扁桃体 + 舌扁桃体 → 呼吸道和消化道入口的免疫防线

## 食管

三个生理性狭窄（插胃管、异物嵌顿）：

| 狭窄 | 距中切牙 | 记忆 |
|------|----------|------|
| 食管起始处（咽与食管相接） | 15cm | 一五 |
| 左主支气管交叉处 | 25cm | 二五 |
| 穿膈食管裂孔处 | 40cm | 四零 |

## 胃

- 两壁、两口（贲门→食管、幽门→十二指肠）、两缘（胃小弯→右上方，胃大弯→左下方）
- 分部：贲门部、胃底、胃体、幽门部
- **胃窦**（幽门部）：幽门窦 + 幽门管，胃溃疡和胃癌的最常见部位
- 位置：大部分位于左季肋区，小部分位于腹上区

## 小肠

- **十二指肠**：呈"C"形，分四部
  - 上部（球部）→ 溃疡最好发部位
  - 降部 → 十二指肠大乳头（**胆总管和胰管共同开口**）
  - 水平部
  - 升部 → **Treitz韧带**（十二指肠悬韧带）：确认十二指肠与空肠分界的外科标志
- **空肠 vs 回肠**：
  - 空肠：左上腹，管径粗、壁厚、血供丰富、环状襞密集
  - 回肠：右下腹，管径细、壁薄、血供差、有集合淋巴滤泡
- **Meckel憩室**：距回盲瓣约0.5~1m处回肠的卵黄管残留，约2%人群存在

## 大肠

- 特征：结肠带、结肠袋、肠脂垂（直肠、肛管、阑尾除外）
- **回盲瓣**：防止大肠内容物反流入回肠
- **阑尾**：根部体表投影——**麦氏点**（脐与右髂前上棘连线的中、外1/3交点）
- **直肠**：两个弯曲——骶曲（凸向后）、会阴曲（凸向前）
- **肛管→齿状线**：上下来源、神经支配、静脉回流均不同

| | 齿状线以上 | 齿状线以下 |
|---|-----------|-----------|
| 来源 | 内胚层 | 外胚层 |
| 神经 | 内脏神经（不痛） | 躯体神经（痛） |
| 静脉 | 肠系膜下静脉→门静脉 | 阴部内静脉→髂内静脉 |
| 病变 | 内痔、腺癌 | 外痔、鳞癌 |
