# 大消化腺

## 肝

- 位置：大部分位于右季肋区 + 腹上区，小部分至左季肋区
- 形态：楔形，上面（膈面）光滑、下面（脏面）有"H"形沟
- **肝门**：脏面横沟，肝固有动脉、门静脉、肝管、神经、淋巴管出入
- **胆囊**：位于肝下面胆囊窝内，分底、体、颈、管
  - 胆囊底体表投影：右腹直肌外侧缘与右肋弓交点（**Murphy征**检查点）
  - **Calot三角**（胆囊三角）：胆囊管 + 肝总管 + 肝下面围成，胆囊动脉多经此→胆囊切除术关键解剖标志
- **肝外胆道**：肝左管 + 肝右管 → 肝总管 + 胆囊管 → 胆总管 → 与胰管汇合开口于十二指肠大乳头
- 胆汁排出路径：肝细胞→毛细胆管→小叶间胆管→肝左/右管→肝总管→胆总管→十二指肠大乳头

## 胰

- 位置：胃后方，腹膜后，横跨L1~2前方
- 分部：胰头（被十二指肠包绕）、胰体、胰尾（邻脾门）
- 胰管：贯穿胰全长，与胆总管汇合开口于十二指肠大乳头
- 功能：外分泌（消化酶）+ 内分泌（胰岛→胰岛素/胰高血糖素）
