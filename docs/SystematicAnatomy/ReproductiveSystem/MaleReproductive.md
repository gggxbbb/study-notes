# 男性生殖系统

男性生殖器：内生殖器（睾丸、附睾、输精管、射精管、精囊腺、前列腺、尿道球腺）+ 外生殖器（阴茎、阴囊）。

## 内生殖器

- **睾丸**：位于阴囊内，产生精子和雄激素。睾丸鞘膜（脏层+壁层，中间为鞘膜腔）
- **附睾**：睾丸后上方，分头、体、尾。精子储存和成熟场所
- **输精管**：附睾尾延续→精索→腹股沟管→盆腔→与精囊腺排泄管汇合为射精管→开口于尿道前列腺部
- **精索**：从腹股沟管深环至睾丸上端，内含输精管、睾丸动脉、蔓状静脉丛等
- **精囊腺**：膀胱底后方，输精管壶腹外侧，分泌物参与精液组成
- **前列腺**：膀胱颈下方，形似栗子，尿道从中穿过
  - 分叶：前叶、中叶、后叶、两侧叶。中叶增生→压迫尿道→排尿困难
  - 经直肠指诊可触及前列腺后面

## 外生殖器

- **阴茎**：分根、体、头。两个阴茎海绵体 + 一个尿道海绵体（前端膨大为阴茎头）
- **包皮系带**：阴茎头腹侧中线上包皮与尿道外口相连的皮肤皱襞，手术时注意保护
- **阴囊**：容纳睾丸，皮肤薄、色素深，有肉膜调节温度

## 男性尿道

| 分部 | 特点 |
|------|------|
| 前列腺部 | 最宽，射精管和前列腺管开口于此 |
| 膜部 | 最短最窄，穿过尿生殖膈，外伤易损伤 |
| 海绵体部 | 最长，尿道球部为其起始膨大处 |

三个狭窄：尿道内口、膜部、尿道外口（最窄）
三个扩大：前列腺部、尿道球部、舟状窝
两个弯曲：耻骨下弯（固定）、耻骨前弯（可消失）
