# 动脉

## 肺循环动脉

- **肺动脉干**：起自右心室，在主动脉弓下方分左、右肺动脉
- **动脉韧带**：肺动脉干分叉处与主动脉弓下缘之间的纤维索（胎儿期动脉导管的遗迹）→未闭即为动脉导管未闭（PDA）

## 体循环动脉

**主动脉**：左心室 → 升主动脉 → 主动脉弓 → 降主动脉（胸主动脉 + 腹主动脉）

### 主动脉弓三大分支（从右到左）

**头臂干 → 右颈总动脉 + 右锁骨下动脉**

| 分支 | 主要供血区域 |
|------|------------|
| **左颈总动脉** | 头颈部 |
| **左锁骨下动脉** | 左上肢 |

**颈外动脉主要分支**（口诀：甲舌面咽上颞浅）：
- 甲状腺上动脉、舌动脉、面动脉（于下颌角前方可触及搏动→压迫止血）、咽升动脉、上颌动脉、颞浅动脉

**锁骨下动脉主要分支**：
- 椎动脉（经颈椎横突孔入颅）、胸廓内动脉（乳内动脉）、甲状颈干

### 腹主动脉

**不成对脏支（口诀：腹腔干、肠上肠下）**：

| 动脉 | 主要供血 |
|------|---------|
| **腹腔干** | 胃左动脉、肝总动脉、脾动脉→胃、肝、胆囊、脾、胰、十二指肠 |
| **肠系膜上动脉** | 空肠、回肠、盲肠、阑尾、升结肠、横结肠 |
| **肠系膜下动脉** | 降结肠、乙状结肠、直肠上段 |

**成对脏支**：肾上腺中动脉、肾动脉、睾丸/卵巢动脉

### 压迫止血点

| 动脉 | 止血点位置 |
|------|----------|
| 颈总动脉 | 环状软骨水平，向后压向第6颈椎横突 |
| 面动脉 | 下颌骨体下缘与咬肌前缘交点 |
| 颞浅动脉 | 耳屏前方 |
| 锁骨下动脉 | 锁骨上窝，压向第1肋 |
| 肱动脉 | 肱二头肌内侧沟 |
| 股动脉 | 腹股沟韧带中点下方 |
| 足背动脉 | 足背踇长伸肌腱外侧 |
