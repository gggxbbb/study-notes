# 心

## 位置与外形

- 位置：胸腔中纵隔，2/3位于正中线左侧。下方邻膈，上方连出入心大血管
- 外形：**一尖**（心尖，朝左前下方，体表位置→左第5肋间隙锁骨中线内侧1~2cm）、**一底**（心底，朝右后上方）、**两面**（胸肋面、膈面）、**三缘**（右缘、左缘、下缘）、**四沟**（冠状沟→房室分界，前/后室间沟→左右心室分界）

## 心腔形态

**右心房**：
- 入口：上腔静脉口、下腔静脉口、冠状窦口
- 出口：右房室口
- 卵圆窝：房间隔右心房侧，胎儿时期卵圆孔的遗迹

**右心室**：
- 入口：右房室口（**三尖瓣**→前尖、后尖、隔侧尖）
- 出口：肺动脉口（**肺动脉瓣**，防止血液逆流入右心室）
- 室上嵴：动脉圆锥与右心室其余部分的分界

**左心房**：
- 入口：左、右各两个肺静脉口（共4个）
- 出口：左房室口

**左心室**：
- 入口：左房室口（**二尖瓣**→前尖、后尖）
- 出口：主动脉口（**主动脉瓣**）
- 心壁最厚，收缩力最强 → 体循环动力泵

## 心的构造

- **心壁**：心内膜 + 心肌层（最厚）+ 心外膜
- **房间隔**：左右心房之间。卵圆窝处最薄→房间隔缺损好发部位
- **室间隔**：分肌部（下部，厚）和膜部（上部，薄）→室间隔缺损好发部位

## 心传导系

| 结构 | 位置 | 功能 |
|------|------|------|
| **窦房结** | 上腔静脉与右心房交界处心外膜下 | 正常心跳起搏点（"总司令"） |
| **房室结** | 冠状窦口前方心内膜下 | 传导延迟→房室顺序收缩 |
| 房室束（His束） | 室间隔膜部 | 传入心室 |
| 左/右束支 | 室间隔两侧心内膜下 | 至左右心室 |
| Purkinje纤维 | 心室内膜下 | 激动心室肌 |

## 心的血管

- **左冠状动脉**：起自主动脉左窦→前室间支（前降支）+ 旋支
  - 前室间支：左/右心室前壁、室间隔前2/3→心肌梗死最常见血管
- **右冠状动脉**：起自主动脉右窦→后室间支 + 左室后支
  - 供血：右心房、右心室、室间隔后1/3、窦房结（60%人群）和房室结

**冠状窦**：心大静脉+心中静脉+心小静脉→冠状窦→右心房。心静脉血回流主通道。

## 心包

- 纤维心包（外层，坚韧）
- 浆膜心包（内层）：壁层 + 脏层（心外膜）→ 心包腔（含少量浆液）
- **心包横窦**：升主动脉和肺动脉干后方，上腔静脉前方，心脏手术时阻断大血管的通道
- **心包斜窦**：左心房后方，心包壁层返折处围成
