# 前庭蜗器（耳）

前庭蜗器 = 位觉器 + 听觉器。分外耳、中耳、内耳。

## 外耳

- **耳廓**：弹性软骨 + 皮肤
- **外耳道**：外1/3软骨部 + 内2/3骨部。幼儿外耳道短而直→检查时将耳廓拉向后下方（成人拉向后上方）
- **鼓膜**：椭圆形半透明薄膜，分割外耳和中耳。上方1/4松弛部、下方3/4紧张部。正常鼓膜前下方三角形反光区→**光锥**→中耳病变时消失或变形

## 中耳

- **鼓室**：颞骨岩部内含气小腔，六壁：
  - 外侧壁（鼓膜壁）、内侧壁（迷路壁→岬、前庭窗/卵圆窗、蜗窗/圆窗）、上壁（鼓室盖→邻中颅窝，感染可入颅）、下壁（颈静脉壁）、前壁（颈动脉壁→咽鼓管开口）、后壁（乳突壁→通乳突窦）
- **听小骨链**：锤骨→砧骨→镫骨。镫骨底板封闭前庭窗→将鼓膜振动传导至内耳
- **咽鼓管**：鼓室前壁→鼻咽侧壁。儿童短、平、直→咽部感染易经此引起中耳炎

## 内耳

| 骨迷路 | 膜迷路 | 关键感受器 |
|--------|--------|----------|
| 前庭 | 椭圆囊 + 球囊 | 囊斑→直线变速运动和头部位置觉 |
| 骨半规管（3个，互相垂直） | 膜半规管 | 壶腹嵴→旋转运动觉 |
| 耳蜗 | 蜗管 | **螺旋器（Corti器）**：听觉感受器，位于基底膜上 |

声波传导：外耳道→鼓膜→听小骨链→前庭窗→外淋巴→内淋巴→螺旋器→蜗神经→听觉中枢

> 口诀：耳道鼓膜听骨链，前庭窗入内淋巴，螺旋器换能传中枢
