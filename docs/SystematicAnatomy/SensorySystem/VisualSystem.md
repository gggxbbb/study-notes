# 视器（眼）

## 眼球壁（三层）

| 层次 | 组成 | 要点 |
|------|------|------|
| **纤维膜**（外层） | 角膜（前1/6，透明无血管）+ 巩膜（后5/6，白色坚韧） | 角膜→屈光主要结构 |
| **血管膜**（中层） | 虹膜（瞳孔/瞳孔括约肌/开大肌）+ 睫状体（产生房水、调节晶状体曲度）+ 脉络膜（营养） | 虹膜中央孔为瞳孔 |
| **视网膜**（内层） | 盲部 + 视部 | **黄斑**（中央凹→视觉最敏锐）；**视神经盘**（盲点） |

## 眼球内容物

| 结构 | 特点 |
|------|------|
| **房水** | 睫状体产生→后房→瞳孔→前房→虹膜角膜角→巩膜静脉窦→回流入血。循环障碍→**青光眼** |
| **晶状体** | 双凸透镜，借睫状小带连睫状体，弹性可调→调节焦距。混浊→**白内障** |
| **玻璃体** | 透明胶状，填充眼球后4/5 |

## 屈光系统

角膜 + 房水 + 晶状体 + 玻璃体 → 将光线聚焦于视网膜黄斑。

## 眼副器

- **结膜**：睑结膜 + 球结膜 + 结膜穹窿
- **泪器**：泪腺（外上眶缘泪腺窝→分泌泪液）→ 泪点→泪小管→泪囊→鼻泪管→下鼻道
- **眼球外肌**（6条+1条提上睑肌）：

| 肌肉 | 作用 |
|------|------|
| 上直肌 | 上转 + 内转 |
| 下直肌 | 下转 + 内转 |
| 内直肌 | 内转 |
| 外直肌 | 外转（展神经支配→损伤出现内斜视/复视） |
| 上斜肌 | 下转 + 外转（滑车神经支配） |
| 下斜肌 | 上转 + 外转 |

## 眼的血管

- **视网膜中央动脉**：眼动脉分支，经视神经盘入视网膜→通过眼底镜可直接观察→动脉硬化、高血压视网膜病变等诊断窗口
