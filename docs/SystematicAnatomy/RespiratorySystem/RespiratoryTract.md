# 呼吸道（肺外部分）

**上呼吸道**：鼻 → 咽 → 喉；**下呼吸道**：气管 → 支气管树。

## 鼻

- **易出血区**（Little区）：鼻中隔前下部粘膜，血管丰富而表浅→鼻出血最常见部位
- **嗅区**：上鼻甲及其相对的鼻中隔部分，含嗅细胞
- **鼻旁窦**：四对，均开口于鼻腔

| 鼻旁窦 | 开口位置 |
|--------|----------|
| 额窦 | 中鼻道 |
| 上颌窦 | 中鼻道（开口高于窦底→引流不畅，炎症最常见） |
| 筛窦前/中群 | 中鼻道 |
| 筛窦后群 | 上鼻道 |
| 蝶窦 | 蝶筛隐窝 |

## 喉

- **喉软骨**：
  - 甲状软骨（最大，前角上端→喉结，男性明显）
  - 环状软骨（唯一完整的软骨环，喉支架的基础）
  - 会厌软骨（吞咽时覆盖喉口）
  - 杓状软骨（一对，声带活动基础）
- **弹性圆锥**：连于甲状软骨前角后面与环状软骨上缘之间，上缘游离增厚→声韧带
- **喉腔**：
  - 前庭襞（假声带）位于声襞（真声带）上方
  - 声门下腔：粘膜下组织疏松→急性喉炎时易水肿致呼吸困难（尤其幼儿）

## 气管与主支气管

- 气管：食管前方，上接环状软骨（C6水平）→ T4下缘分叉（气管杈）
- **气管隆嵴**：气管杈内面半月形隆起，气管镜检查的重要定位标志

**左右主支气管差异**（异物易入右侧）：

| | 左 | 右 |
|---|----|----|
| 走向 | 较水平 | 较垂直（陡直） |
| 管径 | 细 | 粗 |
| 长度 | 长 | 短 |

> 口诀：左细长平，右粗短直——异物向右
