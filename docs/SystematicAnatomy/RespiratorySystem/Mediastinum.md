# 纵隔

- **概念**：两侧纵隔胸膜之间所有器官和组织的总称
- **境界**：前→胸骨，后→脊柱胸段，两侧→纵隔胸膜，上→胸廓上口，下→膈
- **分部**（以胸骨角至T4下缘平面为界）：

| 分部 | 主要内容 |
|------|---------|
| **上纵隔** | 胸腺、出心大血管（主动脉弓+三大分支、上腔静脉）、气管、食管、胸导管、迷走神经、膈神经 |
| **前纵隔** | 少量结缔组织和淋巴结 |
| **中纵隔** | 心包、心、出入心大血管根部、膈神经 |
| **后纵隔** | 食管、胸主动脉、奇静脉、胸导管、迷走神经、交感干 |

> 口诀：上纵出心上纵隔心包出心，前纵疏松中纵心包心，后纵食管胸导管奇静脉迷走交感干
