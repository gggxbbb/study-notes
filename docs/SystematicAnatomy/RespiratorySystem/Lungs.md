# 肺

- 位置：胸腔内，纵隔两侧，左右各一
- 形态：一尖（**肺尖**，高出锁骨内侧1/3上方2~3cm，听诊三角区）、一底（肺底/膈面）、两面（肋面、纵隔面）、三缘（前缘、后缘、下缘）
- **左肺**：2叶（上叶+下叶），前缘有心切迹→舌叶
- **右肺**：3叶（上叶+中叶+下叶）
- **肺门**：纵隔面中部的凹陷，主支气管、肺动脉、肺静脉、神经、淋巴管出入
- **肺根**：出入肺门结构的合称，被胸膜包绕

## 肺段

- **支气管肺段**：每个肺段支气管及其所属肺组织，呈圆锥形，尖向肺门、底向肺面
- 相邻肺段间有少量结缔组织分隔→肺段切除的解剖学基础

## 肺的血管

- 功能性血管：肺动脉（含静脉血）→气体交换→肺静脉（含动脉血）
- 营养性血管：支气管动脉（发自胸主动脉，营养肺组织）
