# DNA的生物合成

## DNA复制的基本规律

- **半保留复制**：子代DNA保留一条亲代链（Meselson-Stahl实验证实）
- **半不连续复制**：前导链连续合成，后随链不连续合成（**冈崎片段**）
- **双向复制**：从复制起点向两个方向进行。原核一个起点，真核多起点
- **方向**：5' → 3'

## DNA复制的酶学

| 酶/蛋白 | 功能 |
|---------|------|
| **解旋酶**（DnaB） | 解开DNA双螺旋 |
| **单链结合蛋白**（SSB） | 稳定单链，防止复性 |
| **引物酶**（DnaG） | 合成RNA引物（~10nt，提供3'-OH） |
| **DNA聚合酶III**（原核主酶） | 5'→3'合成DNA链，3'→5'外切校对 |
| DNA聚合酶I | 切除RNA引物 + 填补间隙 + 校对 |
| **DNA连接酶** | 连接冈崎片段（需ATP/NAD⁺） |
| **拓扑异构酶** | 消除超螺旋。I型（切单链，不需ATP）、II型（切双链，需ATP→细菌DNA旋转酶） |

## 原核生物DNA复制过程

1. **起始**：DnaA识别复制起点oriC→解旋→DnaB/DnaG加入→形成复制叉
2. **延伸**：前导链连续合成（一个引物）；后随链不连续合成（多个冈崎片段）
3. **终止**：复制到达终止区→Tus蛋白阻挡→两子代环状DNA互锁→拓扑异构酶分离

## 真核生物DNA复制

特点：
- 多起点复制（复制子）
- 复制叉移动慢
- DNA聚合酶α/δ/ε
- **端粒**：真核线性染色体末端特殊结构（TTAGGG重复序列）。DNA复制→末端短缩→**端粒酶**（逆转录酶）维持端粒长度。端粒长度与衰老、肿瘤相关

## 逆转录

- 以RNA为模板合成DNA（违反中心法则→为"逆向"）
- 酶：**逆转录酶**（RNA依赖DNA聚合酶），由逆转录病毒编码（如HIV）
- 过程：RNA → RNA-DNA杂交链 → 双链DNA（cDNA）→整合入宿主基因组
- 临床：HIV治疗靶点（如齐多夫定AZT抑制逆转录酶）
- 应用：RT-PCR、cDNA文库构建
