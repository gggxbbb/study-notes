# 糖代谢

## 概述

- 糖的生理功能：氧化供能（最主要）、结构成分（糖蛋白/糖脂）、碳源
- 血糖：血液中的葡萄糖，正常空腹3.9~6.1mmol/L

## 糖的无氧氧化（糖酵解）

- 部位：**细胞质**
- 过程：葡萄糖(6C) → 2分子丙酮酸(3C)
- 两个阶段：耗能阶段（己糖磷酸化）+ 产能阶段（磷酸丙糖氧化）

### 关键酶（3个不可逆反应）

| 反应 | 酶 | 抑制剂 |
|------|-----|--------|
| 葡萄糖 → G-6-P | **己糖激酶/葡萄糖激酶**（肝） | G-6-P反馈抑制 |
| F-6-P → F-1,6-2P | **磷酸果糖激酶-1(PFK-1)**←最关键 | ATP/柠檬酸↑, AMP/ADP/F-2,6-2P↑→激活 |
| PEP → 丙酮酸 | **丙酮酸激酶** | ATP↑, 胰高血糖素（磷酸化失活） |

### 产能

无氧条件下：1分子葡萄糖 → 2ATP（底物水平磷酸化）
有氧条件下：2分子NADH+H⁺经穿梭进入线粒体可再生成3~5ATP

### 生理意义

- 迅速提供能量（剧烈运动、缺氧时尤其重要）
- 红细胞唯一能量来源（无线粒体）
- 中间产物提供合成前体

## 糖的有氧氧化

- 部位：细胞质 → **线粒体**
- 三个阶段：
  1. **酵解**：葡萄糖 → 丙酮酸（细胞质）
  2. **丙酮酸氧化脱羧**：丙酮酸 → 乙酰CoA（线粒体基质，**丙酮酸脱氢酶复合体**）
  3. **三羧酸循环**（TCA/Krebs循环）+ **氧化磷酸化**

### 丙酮酸脱氢酶复合体

- 5种辅酶：TPP、硫辛酸、FAD、NAD⁺、CoA
- 不可逆反应，关键调控点
- 产物：乙酰CoA + NADH + CO₂

### 三羧酸循环

- 部位：线粒体基质
- 过程：乙酰CoA(2C) + 草酰乙酸(4C) → 柠檬酸(6C) → ... → 草酰乙酸（再生）
- **关键酶**：柠檬酸合酶、异柠檬酸脱氢酶（最重要限速酶）、α-酮戊二酸脱氢酶复合体

**产能**：1分子乙酰CoA经TCA → 10ATP（3NADH×2.5 + 1FADH₂×1.5 + 1GTP）

**总产能**：1分子葡萄糖彻底氧化 → **30~32ATP**

> 口诀：亚宁一虎下山

## 磷酸戊糖途径

- 部位：细胞质
- **关键酶**：G-6-P脱氢酶（G6PD）
- 产物：**NADPH**（供氢体→脂肪酸/胆固醇合成、GSH还原）和 **5-磷酸核糖**（合成核苷酸）
- G6PD缺乏→红细胞NADPH↓→GSH↓→氧化损伤→溶血（蚕豆病）

## 糖原的合成与分解

- **糖原合成**：G → G-6-P → G-1-P → UDPG → 糖原。关键酶：**糖原合酶**
- **糖原分解**：糖原 → G-1-P → G-6-P → G。关键酶：**糖原磷酸化酶**（限速酶）
- 肝糖原→维持血糖；肌糖原→供自身能量（缺乏G-6-P酶，不能直接补充血糖）

## 糖异生

- 部位：肝（主要）+ 肾皮质
- 底物：乳酸、甘油、生糖氨基酸等非糖物质 → 葡萄糖
- **关键酶**：绕过糖酵解的三个不可逆反应
  - 丙酮酸羧化酶 + PEP羧激酶（代替丙酮酸激酶）
  - **果糖-1,6-二磷酸酶**（代替PFK-1）
  - **葡萄糖-6-磷酸酶**（代替己糖激酶，仅肝肾有→肌不能糖异生）
- 生理意义：饥饿时维持血糖；乳酸循环（Cori循环）

## 血糖及其调节

| | 降血糖 | 升血糖 |
|---|--------|--------|
| 激素 | **胰岛素**（唯一） | 胰高血糖素、糖皮质激素、肾上腺素、生长激素 |
| 机制 | 促进摄取利用、糖原合成↑、糖异生↓ | 糖原分解↑、糖异生↑ |

糖尿病：血糖超出肾糖阈(~10mmol/L)→尿糖阳性。三多一少症状与代谢紊乱。
