## 第十三章 DNA损伤和损伤修复

### DNA损伤

- 来源：自发（复制错误、脱嘌呤、脱氨基）或环境（UV、辐射、化学物）
- 类型：碱基错配、缺失/插入、嘧啶二聚体（UV）、链断裂

### 修复方式

| 方式 | 机制 |
|------|------|
| **直接修复** | 光修复酶→直接修复嘧啶二聚体 |
| **切除修复** | 切除损伤段→聚合酶填补→连接酶封口（NER→大片段损伤，BER→小碱基损伤） |
| **错配修复** | 识别错配碱基→切除→重合成（MutS/MutL/MutH→原核） |
| **重组修复** | 复制后再修复双链断裂→同源重组填补缺口 |
| **SOS修复** | 紧急应答，易出错（→突变），牺牲保真求生存 |

### 意义

- 维持基因组稳定性
- 修复缺陷→疾病：**着色性干皮病**（XP→NER缺陷，UV敏感→皮肤癌）；**遗传性非息肉性结直肠癌**（HNPCC→错配修复缺陷）
