# 蛋白质的生物合成（翻译）

## 蛋白质合成体系

| 组分 | 作用 |
|------|------|
| **mRNA** | 模板，含**三联体密码子**。AUG→起始+Met；UAA/UAG/UGA→终止 |
| **tRNA** | "翻译器"，反密码子识别密码子。氨基酰-tRNA合成酶→高度特异性校对 |
| **核糖体** | 场所。原核70S=30S+50S，真核80S=40S+60S。A位（氨基酰位）、P位（肽位）、E位（出口位） |

### 遗传密码特点

- **方向性**：5' → 3'
- **连续性**：无间隔（无逗号），阅读框不重叠
- **简并性**：多个密码子编码同一氨基酸（第三个碱基→"摆动"）
- **通用性**：几乎全部生物共用一套密码子
- **摆动配对**：反密码子第1位可与密码子第3位非标准配对

## 肽链合成过程

### 原核生物

1. **起始**：IF-3 + 30S亚基→mRNA+IF-1/IF-2+fMet-tRNA⁡ (识别AUG)→50S亚基加入→**70S起始复合体**
2. **延伸**（循环）：
   - **进位**：EF-Tu将氨基酰-tRNA送入A位
   - **成肽**：核糖体大亚基23S rRNA（核酶）催化P位肽酰基与A位氨基酸脱水缩合
   - **移位**：EF-G催化，核糖体沿mRNA移动一个密码子。A位空出→进入下一轮
3. **终止**：释放因子(RF)识别终止密码子→肽链释放→核糖体解离

### 真核生物

- 起始更复杂：小亚基+Met-tRNA⁡先结合mRNA 5'端帽子→**扫描机制**找到第一个AUG→起始
- 延伸和终止类似

## 蛋白质合成后的加工和靶向输送

- **折叠**：分子伴侣(Hsp70、Hsp60/伴侣蛋白)辅助
- **化学修饰**：磷酸化、乙酰化、糖基化、二硫键形成、切除信号肽…
- **靶向输送**：内质网（信号肽）、线粒体/核定位（信号序列）

## 蛋白质生物合成的干扰和抑制

| 药物/毒素 | 作用 |
|----------|------|
| **四环素** | 阻止氨基酰-tRNA与A位结合（原核） |
| **氯霉素** | 抑制肽键形成（原核+线粒体核糖体50S） |
| **链霉素** | 结合30S亚基→错读密码子 |
| **红霉素** | 结合50S→阻止肽链延伸 |
| **嘌呤霉素** | 提前释放未完成肽链（原核+真核） |
| **白喉毒素** | 修饰eEF-2→阻止真核移位 |
| **蓖麻毒素** | 切除真核28S rRNA→核糖体失活 |
