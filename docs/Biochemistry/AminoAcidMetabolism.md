# 氨基酸代谢

## 蛋白质的生理功能和营养作用

- 必需氨基酸（8种）：**缬、异亮、亮、苯丙、蛋（甲硫）、色、苏、赖**
  - 口诀：借一两本淡色书来（缬异亮苯蛋色苏赖）
  - 组氨酸、精氨酸为半必需（儿童必需）
- **氮平衡**：摄入氮 = 排出氮（成人）；正氮平衡（生长期/恢复期）；负氮平衡（饥饿/消耗性疾病）
- 蛋白质互补作用：不同食物蛋白质搭配食用，氨基酸互相补充

## 体内氨基酸的来源

- 食物蛋白质消化吸收
- 组织蛋白质分解
- 体内合成（非必需氨基酸）

氨基酸代谢库：体内游离氨基酸的总和，来源与去路动态平衡。

## 氨基酸的分解代谢

### 脱氨基（主要代谢途径）

1. **转氨基**：α-氨基酸 + α-酮酸 ⇌ α-酮酸 + α-氨基酸。关键酶：**转氨酶**（辅酶=磷酸吡哆醛→VitB₆）
   - **ALT**（GPT，肝含量高）→ 肝病时血清ALT↑
   - **AST**（GOT，心肌含量高）
2. **氧化脱氨基**：**谷氨酸脱氢酶**（肝、肾线粒体，辅酶NAD⁺/NADP⁺）→ α-酮戊二酸 + NH₃
3. **联合脱氨基**（主要方式）：转氨基 + 氧化脱氨基
4. **嘌呤核苷酸循环**（肌肉主要方式）

### α-酮酸的代谢

- 重新合成氨基酸
- 转变为糖或脂（生糖氨基酸、生酮氨基酸）
- 氧化供能

### 氨的代谢

- **血氨来源**：氨基酸脱氨基（主要）、肠道吸收（细菌+尿素酶）、肾产氨
- **血氨去路**：
  - **尿素循环**（鸟氨酸循环，肝线粒体+细胞质）：2NH₃ + CO₂ + 3ATP → 尿素，关键酶：**CPS-I**（氨甲酰磷酸合成酶I，线粒体）、**精氨酸代琥珀酸合成酶**（限速酶）
  - 合成谷氨酰胺（脑氨解毒主要方式）

**高血氨症**：肝功能严重障碍→尿素合成↓→血氨↑→脑（消耗α-酮戊二酸→TCA障碍→脑供能不足）→**肝性脑病**

## 个别氨基酸的代谢

- **一碳单位**：含一个碳原子的基团（甲基、亚甲基、甲酰基等），载体：**四氢叶酸(FH₄)**。参与核苷酸合成→缺叶酸影响DNA合成→巨幼细胞性贫血
- **含硫氨基酸**：蛋氨酸→SAM（甲基供体），半胱氨酸→牛磺酸/GSH，胱氨酸
- **芳香族氨基酸**：苯丙氨酸 → 酪氨酸 → 多巴/多巴胺/去甲肾上腺素/肾上腺素 + 黑色素 + 甲状腺素
  - **苯丙酮酸尿症**（PKU）：苯丙氨酸羟化酶缺陷→代谢旁路→苯丙酮酸蓄积→智力障碍
  - **白化病**：酪氨酸酶缺陷→黑色素合成障碍
