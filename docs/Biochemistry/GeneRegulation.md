# 基因表达调控

## 基本概念

- **基因**：编码功能性RNA或蛋白质多肽链的DNA片段
- **基因表达**：基因转录为RNA→翻译为蛋白质的过程
- **基因表达调控**：基因表达的开启/关闭及表达强度的控制。可在转录水平、转录后加工、翻译水平、翻译后修饰等多层次调控

## 顺式作用元件

DNA分子上调控基因转录的特定序列，位于同一DNA链上：

| 元件 | 位置 | 功能 |
|------|------|------|
| **启动子** | 转录起始位点5'上游 | RNA聚合酶结合位点。含TATA盒、CAAT盒、GC盒 |
| **增强子** | 可远距离(上/下游) | 增强基因转录，位置和方向不依赖 |
| **沉默子** | 同增强子 | 抑制基因转录 |

## 反式作用因子

- 能识别顺式作用元件并与之结合的**蛋白质因子**（转录因子→TF）
- **转录因子的结构域**：
  - DNA结合域：螺旋-转角-螺旋、锌指(zinc finger)、亮氨酸拉链(leucine zipper)、螺旋-环-螺旋(HLH)
  - 转录激活域

## DNA与蛋白质相互作用

- 大多数转录因子以**二聚体**形式结合DNA（同源二聚体/异源二聚体）
- 识别机制：碱基序列的直接读取 + 间接读取（DNA构象识别）

## 原核基因的转录调控——操纵子

### 乳糖操纵子（lac operon）

- 结构基因：lacZ(β-半乳糖苷酶)、lacY(通透酶)、lacA
- 调控序列：启动子(P) + 操纵序列(O) + CAP结合位点
- **负调控**：阻遏蛋白(I)结合O→阻止转录；乳糖→诱导物→结合阻遏蛋白使其脱离O→转录开放
- **正调控**：葡萄糖↓→cAMP↑→CAP-cAMP复合物→结合CAP位点→促进转录

### 色氨酸操纵子（trp operon）

- 阻遏：Trp充足→Trp-阻遏蛋白复合物激活→结合O→转录关闭
- **衰减作用**：前导肽的翻译速度调控转录提前终止

## 真核基因的转录调控

- 比原核复杂：多启动子、多增强子、多转录因子
- **转录起始复合体**：TFIID(TBP+TAFs)结合TATA盒→TFIIA/B/F加入→RNA聚合酶II + TFIIE/H→转录起始
- **染色质重塑**：组蛋白乙酰化（开放染色质）、甲基化（可激活或抑制）、DNA甲基化（CpG岛，抑制表达）
