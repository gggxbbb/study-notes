# RNA的生物合成（转录）

转录：以DNA为模板，在RNA聚合酶作用下合成RNA的过程。

## 概述

- 模板：DNA的模板链（反义链），编码链（有义链）与RNA序列一致（T→U）
- RNA聚合酶（原核）：单一酶（核心酶+σ因子→全酶）
- RNA聚合酶（真核）：三种——**RNA pol I**（45S rRNA）、**RNA pol II**（mRNA+snRNA）、**RNA pol III**（tRNA+5S rRNA）

## 原核生物转录

1. **起始**：σ因子识别启动子→闭合转录复合体→开放转录复合体→合成第一个磷酸二酯键→σ因子解离
2. **延伸**：核心酶沿模板3'→5'移动，RNA链5'→3'延伸
3. **终止**：
   - **不依赖ρ因子**（内部终止）：RNA转录产物形成发夹结构+polyU→终止
   - **依赖ρ因子**：ρ因子识别终止序列，解旋RNA-DNA杂交体

## 真核生物mRNA的转录、加工和降解

### 加工修饰

真核mRNA初级转录产物（hnRNA）→ 成熟mRNA：

| 加工方式 | 特点 |
|---------|------|
| **5'端加帽** | m⁷GpppN（7-甲基鸟苷三磷酸），保护5'端、识别核糖体 |
| **3'端加尾** | PolyA尾（~200nt），保护3'端、增加稳定性 |
| **剪接** | 切除内含子→连接外显子。剪接体（snRNP）催化 |

**可变剪接**：同一pre-mRNA不同剪接方式→多种成熟mRNA→蛋白质多样性

### 断裂基因

- **外显子**（exon）：编码区，保留在成熟mRNA中
- **内含子**（intron）：非编码区，转录后切除

## 真核生物非编码RNA的生物合成

- **rRNA**：RNA pol I转录45S→加工为18S、5.8S、28S rRNA。RNA pol III转录5S rRNA
- **tRNA**：RNA pol III转录→加工（切除5'端、3'端加CCA、某些碱基修饰）
- **miRNA**：RNA pol II转录pri-miRNA→Drosha/Dicer剪切→成熟miRNA→RISC→靶mRNA降解/翻译抑制
