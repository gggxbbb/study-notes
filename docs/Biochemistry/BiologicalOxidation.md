# 生物氧化

营养物质在体内氧化分解为CO₂和H₂O，同时释放能量生成ATP的过程。主要在线粒体进行。

## 线粒体氧化体系

呼吸链（电子传递链）：位于**线粒体内膜**，将NADH/FADH₂的氢传递给氧生成H₂O，同时驱动ATP合成。

### 呼吸链组成

| 组分 | 辅基 | 功能 |
|------|------|------|
| 复合体I (NADH脱氢酶) | FMN, Fe-S | NADH→CoQ |
| 复合体II (琥珀酸脱氢酶) | FAD, Fe-S | 琥珀酸→CoQ（TCA循环酶） |
| **CoQ**（泛醌） | — | 氢传递体，连接复合体I/II与III |
| 复合体III (细胞色素bc₁) | 血红素, Fe-S | CoQ→Cyt c |
| **Cyt c** | 血红素 | 外周膜蛋白，电子传递 |
| 复合体IV (细胞色素氧化酶) | Cyt a, a₃, Cu | Cyt c→O₂（**CN⁻、CO中毒→抑制此酶**） |

### 两条呼吸链

- **NADH氧化呼吸链**：NADH → 复合体I → CoQ → 复合体III → Cyt c → 复合体IV → ½O₂ → H₂O（P/O≈2.5）
- **FADH₂氧化呼吸链**（琥珀酸氧化）：琥珀酸 → 复合体II → CoQ → 复合体III → Cyt c → 复合体IV → ½O₂ → H₂O（P/O≈1.5）

## 氧化磷酸化与ATP合成

- **氧化磷酸化**：呼吸链电子传递过程中释放的能量驱动ADP + Pi → ATP
- **底物水平磷酸化**：代谢物脱磷酸基团直接转移给ADP→ATP（如糖酵解中）
- **偶联部位**：复合体I、III、IV

### ATP合酶（复合体V）

- 位于线粒体内膜，分F₀（H⁺通道）和F₁（催化亚基，合成ATP）
- **化学渗透假说**（Mitchell）：呼吸链传递电子时将H⁺泵出内膜→电化学梯度→H⁺经ATP合酶回流→驱动ATP合成

### 氧化磷酸化影响因素

| 因素 | 机制 |
|------|------|
| **ADP浓度**（最主要） | ADP↑→氧化磷酸化加速（呼吸控制） |
| **甲状腺激素** | 诱导Na⁺/K⁺-ATP酶↑→ADP↑→耗氧产热↑ |
| 抑制剂 | 呼吸链抑制剂（CN⁻、CO、鱼藤酮）、解偶联剂（DNP→质子漏回） |
| 线粒体DNA突变 | 影响呼吸链组分→线粒体病（母系遗传） |

## 其他氧化和抗氧化体系

- **微粒体氧化体系**：加单氧酶（细胞色素P450），内源性/外源性物质羟化→失活/解毒
- **活性氧（ROS）**：O₂⁻·、H₂O₂、·OH，正常少量产生，过多→氧化应激
- **抗氧化防御**：
  - 酶：SOD（O₂⁻· → H₂O₂ + O₂）、CAT（H₂O₂ → H₂O + O₂）、GSH-Px
  - 非酶：GSH、VitC、VitE、β-胡萝卜素
