# 肝的生物化学

肝是"物质代谢中枢"。

## 肝在物质代谢中的作用

- **糖代谢**：糖原合成/分解（维持血糖）、糖异生（饥饿时主要场所）
- **脂代谢**：脂肪酸β-氧化（能源）、酮体生成（唯一场所）、胆固醇合成/转化为胆汁酸、脂蛋白合成(VLDL/HDL)
- **蛋白质代谢**：血浆蛋白合成（白蛋白、凝血因子除外γ-球蛋白）、氨基酸分解（尿素循环**唯一场所**）
- **维生素和激素**：VitA/D/B₁₂储存、激素灭活
- **生物转化**（见下）

## 肝的生物转化

肝将非营养性异源物(药物/毒物)和内源物(激素/胆红素)转化为易排出物质。

- **两相反应**：
  - **第一相**：氧化/还原/水解（细胞色素P450→关键酶），引入极性基团
  - **第二相**：结合反应（葡糖醛酸结合最常见，UDPGA供体）→水溶性↑→排出

**特点**：连续性、多样性、双重性（解毒 + 有时增毒）

> 临床：新生儿黄疸→葡糖醛酸转移酶未成熟→结合反应不足→胆红素蓄积

## 胆汁酸代谢

- **初级胆汁酸**：肝细胞从胆固醇合成。胆酸 + 鹅脱氧胆酸→与甘氨酸或牛磺酸结合→甘氨/牛磺胆酸（促进脂类消化吸收）
- **次级胆汁酸**：初级胆汁酸经肠道细菌作用（脱羟）→脱氧胆酸 + 石胆酸
- **肠肝循环**：95%胆汁酸在回肠重吸收→门静脉→肝再利用→高效利用

> 胆汁酸合成的关键酶：**7α-羟化酶**（限速酶，胆汁酸负反馈调节）

## 胆色素代谢与黄疸

- 血红素 → 胆绿素 → **游离胆红素**（脂溶性，未结合，ICterus间接胆红素）
- 在肝与葡糖醛酸结合→**结合胆红素**（水溶性，直接胆红素）→经胆汁排出

| 黄疸类型 | 血中升高胆红素 | 病因 |
|---------|-------------|------|
| 溶血性 | 间接（游离） | 溶血过多 |
| 肝细胞性 | 间接 + 直接 | 肝功能损伤 |
| 梗阻性 | 直接（结合） | 胆道阻塞 |
