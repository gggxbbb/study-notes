# 血液的生物化学

## 血液的组成

- 血浆（~55%）：水 + 血浆蛋白 + 代谢物 + 电解质
- 血细胞（~45%）：红细胞、白细胞、血小板

## 血浆蛋白

| 蛋白 | 合成部位 | 功能 |
|------|---------|------|
| **白蛋白** | 肝 | 维持胶体渗透压、运输脂肪酸/药物/胆红素 |
| α₁/α₂球蛋白 | 肝 | 运输激素/金属 |
| β球蛋白 | 肝 | 运输脂质 |
| γ球蛋白（免疫球蛋白） | 浆细胞 | 抗体 |
| **纤维蛋白原** | 肝 | 凝血→转化为纤维蛋白 |

> 肝合成除γ球蛋白外的全部血浆蛋白 → 肝功衰竭时血清白蛋白↓、凝血障碍

## 红细胞代谢

- **无氧酵解**（主要）：成熟红细胞无线粒体，90%能量来自糖酵解
- **磷酸戊糖途径**：提供NADPH→维持GSH还原→保护红细胞免受氧化损伤
- **2,3-BPG旁路**：糖酵解中间产物→调节Hb对O₂的亲和力（2,3-BPG↑→Hb-O₂亲和力↓→促进组织供氧）

## 血红素的生物合成

- 部位：**线粒体 + 细胞质**（有核红细胞和肝细胞）
- 原料：甘氨酸 + 琥珀酰CoA → δ-氨基-γ-酮戊酸(ALA) → ... → 血红素
- **关键酶**：**ALA合酶**（限速酶，血红素反馈抑制）

## 卟啉症

血红素合成途径酶缺陷→中间产物（卟啉/卟啉原）蓄积：
- **急性间歇性卟啉症**：PBG脱氨酶缺陷→腹痛+神经精神症状。诱因：药物、激素
- 卟啉具光敏性→光照后皮肤损害

## 铁的代谢

- 吸收部位：十二指肠
- 转运：转铁蛋白(transferrin)
- 储存：铁蛋白(ferritin) + 含铁血黄素(hemosiderin)
- 缺铁→缺铁性贫血（小细胞低色素性贫血）
