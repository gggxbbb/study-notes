# 绪论

## 生物化学与分子生物学概念

- **生物化学**：研究生物体化学组成和化学变化规律的科学，从分子水平探讨生命现象
- **分子生物学**：研究核酸、蛋白质等生物大分子结构功能及基因表达调控的学科

## 发展简史

| 阶段 | 时间 | 代表成果 |
|------|------|---------|
| 叙述生化 | 18~19世纪 | 发现酶、蛋白质、核酸等 |
| 动态生化 | 20世纪前半 | 阐明代谢途径（Krebs循环、糖酵解等） |
| 分子生物学 | 1953年至今 | DNA双螺旋→中心法则→基因工程→人类基因组计划 |

## 主要研究内容

- **生物大分子**：结构、功能及构效关系（蛋白质、核酸、酶）
- **物质代谢**：糖/脂/氨基酸/核苷酸的代谢途径、调控及能量转换
- **遗传信息传递**：DNA复制→RNA转录→蛋白质翻译（中心法则）
- **基因表达调控**：原核/真核生物的基因调控机制
- **细胞信号转导**：细胞间通讯的分子机制
- **专题篇**：血液生化、肝生化、基因工程等

> 中心法则：DNA → RNA → 蛋白质
