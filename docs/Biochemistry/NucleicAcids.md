# 核酸的结构与功能

## 核酸的化学组成

- 元素：C、H、O、N、P（**含磷量9~10%**→定量测定基础）
- 基本单位：**核苷酸** = 碱基 + 核糖 + 磷酸

| 核酸 | 戊糖 | 碱基 |
|------|------|------|
| **DNA** | 脱氧核糖 | A、G、C、T |
| **RNA** | 核糖 | A、G、C、U |

- **核苷** = 碱基 + 戊糖（糖苷键）；**核苷酸** = 核苷 + 磷酸（酯键）
- 一级结构：核苷酸通过**3',5'-磷酸二酯键**连接，方向5'→3'

## DNA空间结构与功能

**双螺旋结构模型（Watson & Crick, 1953）**要点：
- 两条反向平行的多核苷酸链围绕同一轴盘旋
- 碱基对位于内侧（**A=T, G≡C**），遵循Chargaff规则
- 螺旋直径2nm，螺距3.4nm（10bp），相邻碱基对间距0.34nm
- 大沟和小沟：蛋白质识别结合位点

**DNA二级结构多形性**：B-DNA（标准右手）、A-DNA、**Z-DNA**（左手螺旋，可能与基因表达调控有关）

**DNA功能**：遗传信息的载体。基因→DNA分子上编码功能性产物的特定区段。

## RNA的空间结构与功能

| RNA类型 | 特点 | 功能 |
|---------|------|------|
| **mRNA** | 5'端帽子(m⁷GpppN)、3'端polyA尾，编码区含三联体密码子 | 蛋白质合成的模板 |
| **tRNA** | 二级结构为三叶草形（氨基酰臂、反密码环、TψC环、DHU环），含稀有碱基 | 转运氨基酸、"翻译器" |
| **rRNA** | 与蛋白质结合成核糖体（原核70S=30S+50S，真核80S=40S+60S） | 蛋白质合成场所 |
| **非编码RNA** | miRNA、siRNA、lncRNA等 | 基因表达调控（miRNA→mRNA降解/翻译抑制） |

> 口诀：三叶草tRNA运氨基酸，帽子尾巴mRNA做模板

## 核酸的理化性质

- **紫外吸收**：260nm（碱基共轭双键），定量测定依据
- **DNA变性**：加热/碱→双链解链为单链，**增色效应**（A₂₆₀升高）
- **Tm值**（解链温度）：50%DNA解链时的温度。影响因素：**G+C含量越高→Tm越高**、离子强度
- **复性**：缓慢降温→互补单链重新形成双螺旋（**退火**）
- **核酸分子杂交**：不同来源的DNA/DNA或DNA/RNA通过碱基互补配对形成杂化双链→Southern/Northern blot等
