# 核苷酸代谢

## 嘌呤核苷酸的代谢

### 合成

**从头合成**（肝为主）：
- 原料：天冬氨酸、甘氨酸、谷氨酰胺、CO₂、一碳单位、5-磷酸核糖
- **关键酶**：PRPP合成酶 + **PRPP酰胺转移酶**（限速酶）
- 先合成**IMP**（次黄嘌呤核苷酸）→ 再转变为AMP和GMP

**补救合成**：利用现成嘌呤/嘌呤核苷→节约能量
- HGPRT：次黄嘌呤+PRPP→IMP；鸟嘌呤+PRPP→GMP
- **Lesch-Nyhan综合征**：HGPRT缺陷→补救合成障碍→尿酸↑→自残行为

### 分解

嘌呤核苷酸 → 次黄嘌呤 → 黄嘌呤 → **尿酸**（终产物，经肾排出）
关键酶：**黄嘌呤氧化酶**（别嘌呤醇抑制→治疗痛风）

> 痛风：尿酸过高→尿酸盐结晶沉积关节→炎症

## 嘧啶核苷酸的代谢

### 合成

从头合成（与嘌呤不同：先合成嘧啶环，再连PRPP）：
- 原料：天冬氨酸、谷氨酰胺、CO₂
- 关键酶：**CPS-II**（细胞质，与尿素循环CPS-I区分）、**天冬氨酸转氨基甲酰基酶**（原核）

先合成UMP → UTP → CTP（谷氨酰胺供氨基）
**dTMP**：dUMP → dTMP，**胸苷酸合酶**催化，需N⁵,N¹⁰-亚甲基四氢叶酸供甲基→抗癌药5-FU抑制

## 脱氧核糖核苷酸的合成

- **核糖核苷酸还原酶**：NDP → dNDP（二磷酸水平还原）
