# 基因工程

## 基本概念

- **基因工程**（DNA重组技术）：在体外将目的基因与载体DNA连接→导入受体细胞→表达/扩增
- **工具酶**：
  - **限制性核酸内切酶**：识别特定DNA序列并切割（Type II最常用）→产生粘性末端/平端。如EcoRI识别GAATTC
  - DNA连接酶、DNA聚合酶、逆转录酶、Taq DNA聚合酶等

## 基因工程载体

| 载体 | 容量 | 用途 |
|------|------|------|
| **质粒** | <10kb | 常规克隆、原核表达 |
| **λ噬菌体** | ~20kb | 基因组文库 |
| 黏粒 | ~45kb | — |
| BAC/PAC/YAC | 100kb~1Mb | 大片段操作 |

载体必备要素：**复制起点**(ori)、**多克隆位点**(MCS)、**选择标记**(抗性基因如Amp^R、lacZ蓝白斑筛选)

## 基因工程的一般过程

1. **目的基因获取**：PCR扩增、RT-PCR(从mRNA)、化学合成、基因文库筛选
2. **基因重组**：目的基因 + 载体 → 连接酶→重组DNA分子
3. **转化/转染**：重组DNA导入宿主细胞（原核：感受态转化；真核：脂质体/电穿孔/病毒）
4. **筛选与鉴定**：抗生素筛选 + lacZ蓝白斑初筛 + 酶切验证 + 测序
5. **表达与纯化**：诱导表达→蛋白纯化（His标签亲和层析等）

## 克隆基因的表达

- **原核表达**：大肠杆菌最常用→操作简便、产量高，但不能进行翻译后修饰（糖基化等）
- **真核表达**：酵母、昆虫细胞、哺乳动物细胞→有修饰功能

## 基因工程技术在医学方面的应用

- **重组蛋白药物**：胰岛素（首个）、生长激素、EPO、干扰素
- **基因诊断**：PCR/RT-PCR、基因芯片、FISH
- **基因治疗**：正常基因导入缺陷患者细胞（如CAR-T）
- **转基因动物/植物**
- **法医学**：DNA指纹、亲子鉴定
