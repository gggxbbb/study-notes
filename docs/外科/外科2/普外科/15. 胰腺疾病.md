# 15. 胰腺疾病

## 慢性胰腺炎

- 非手术治疗可给予大量外源性胰酶制剂

## 胰头癌

- 最重要体征：无痛肿大可推动的胆囊
- 检查：首选增强 CT
- 治疗：胰头十二指肠切除术

## 黄疸鉴别

- 胰头癌：进行性加重的黄疸
- 壶腹周围癌：进行性波动性黄疸

## 急性胰腺炎（外科视角）

- 血钙持续低于 2.25mmol/L 提示预后不良，不属于手术指征
- 早期休克提示重症胰腺炎

## 手术指征

1. 急性腹膜炎不能排除其他急腹症时
2. 伴胆总管下端梗阻或胆道感染者
3. 合并肠穿孔、大出血或胰腺假性囊肿
4. 胰腺和胰周坏死组织继发感染
