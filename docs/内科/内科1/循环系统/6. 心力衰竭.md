# 6. 心力衰竭

## 射血分数(EF)

- LVEF 正常范围：50%~70%

## 按 EF 分类

### 1. EF 降低性 HF（收缩性 HF）

- LVEF < 40%
- 如：高血压病晚期、扩张型心肌病

### 2. EF 保留性 HF（舒张性 HF）

- LVEF ≥ 50%
- 常见原因：
  - 高血压病早期（最常见）
  - 冠心病早期
  - 肥厚型心肌病、限制性心肌病、缩窄性心包炎
- 发病机制：缺氧 → ATP↓ → 肌浆钙泵↓ → 舒张障碍

### 3. EF 中间范围 HF

- LVEF 40%~49%
- 轻度收缩障碍 + 舒张功能不全

## 无症状性 HF

- 特点：只是无症状，其他都有
- 左室功能↓
- 射血分数(LVEF) < 50%
- 神经-体液激活：交感 N、RAS 系统激活

## Killip 分级（急性心肌梗死）

- I 级：无心力衰竭（无啰音）
- II 级：轻度心力衰竭（50% 以下肺野）
- III 级：严重心力衰竭（大于 50% 肺野）
- IV 级：心源性休克（血压小于 90/60）
- 有湿罗音就是心力衰竭：
  - 小于 50% 肺野 → 心力衰竭 2 级
  - 大于 50% 肺野 → 心力衰竭 3 级

## 血流动力学分类

- 心脏指数（CI）：
  - 冷 CI：
    - 干（无淤血）：低血容量 → 补充血容量
    - 湿（有淤血）：肺水肿 → 利尿、扩管
  - 暖 CI：
    - 干（无淤血）：非心源性休克 → 心源性给予扩管药物
    - 湿（有淤血）：
      - 高血压→利尿扩管
      - 低血压→强心扩管（多巴酚丁胺、硝普钠）

## 分期

- 心衰代偿期：无肺循环淤血、体循环淤血
  - 代偿期：左室功能下降，心脏的射血分数可以正常（LVEF > 50%），可以是射血分数保留心衰（舒张性心衰），不能耐受劳力
- 心衰失代偿期：存在肺淤血、体循环淤血
  - 是射血分数下降的心衰（收缩性心衰）

## 心脏参数

- 心脏前负荷：前负荷取决于血容量多少，前负荷取决于容量负荷
- 后负荷：是压力负荷，看收缩压和舒张压，引起血压升高的疾病会引起后负荷增加
- 高血压导致心脏负荷增加：后负荷
- 二尖瓣关闭不全导致心脏负荷增加：前负荷（容量负荷）

## 病因与分类

一、按来源分类
  1. 原发性：心肌本身有问题
  2. 继发性：别的地方有问题导致心脏出问题
  3. 诱因：放在正常人身上不会发病
  4. 病因：放在正常人身上也会发病

二、高血压与心衰
  1. 早期：舒张型心力衰竭（向心性肥厚）
  2. 晚期：收缩型心力衰竭（离心性肥厚）

三、心律失常与心衰
  1. 快房颤：诱发心衰的最重要因素
  2. 室颤：直接致死

## 循环淤血

一、左心衰 = 肺循环淤血 = 肺静脉充血
  1. 表现：白色泡沫痰
  2. 肺水肿：粉红色泡沫痰
  3. 呼吸困难分级：
    1. 劳力性呼吸困难——最早
    2. 夜间阵发性呼吸困难——最典型
    3. 端坐呼吸
    4. 急性肺水肿——最严重（可有哮鸣音，心源性哮喘）
  4. 夜间阵发性呼吸困难机理：
    1. 平卧→静脉回流量增加→加重肺淤血
    2. 夜间迷走神经兴奋→小支气管收缩
    3. 横膈抬高→肺活量减少

二、右心衰 = 体循环淤血 = 上下腔静脉充血
  1. 表现：下肢水肿、肝颈静脉回流征阳性、胃肠道淤血（恶心呕吐食欲差）
  2. 马蹄腹肝

三、全心衰竭：肺部湿罗音 + 下肢水肿
  1. 注意：由左心衰继发右心衰发展而来的全心衰

四、充血性质
  1. 动脉充血 = 主动
  2. 静脉充血 = 被动
  3. 淤血 = 静脉充血 = 被动

## 心脏负荷

一、后负荷（压力负荷）
  1. 狭窄：主狭（左心）、肺狭（右心）
  2. 高血压（左心）、肺动脉高压（右心）

二、前负荷（容量负荷）
  1. 关闭不全
  2. 缺损
  3. 血容量增加：评价动脚人（严重贫血、甲亢、动静脉瘘、脚气病、妊娠）
  4. 口诀："不缺钱"——不（关闭不全）、缺（缺损）、钱（血容量增加）

## 心衰诱因与病因

一、诱因
  1. 最常见：感染（肺部感染）
  2. 最重要：房颤
  3. 最易诱发心衰的心律失常：房颤

二、病因
  1. 心衰最常见病因：冠心病
  2. 按年龄：
    1. >75岁：高血压
    2. <75岁：冠心病
    3. <35岁：心肌炎、心肌病

三、治疗重点
  1. 急性心衰：降负荷
  2. 慢性心衰：抗重构

## 神经体液机制

一、交感神经与迷走神经
  1. 交感神经兴奋：运动时（心跳加速）
  2. 迷走神经兴奋：睡眠时（心跳放缓）
  3. 慢性心功能不全时：心脏代偿，心率增高以提高输出量

二、体液因子
  1. 内皮素：强效血管收缩
  2. 内皮依赖性释放因子（NO）：扩血管
  3. 只有心脏自己释放的因子才有益于自己，其他都是"好心办坏事"
  4. 心房肽（ANP）：对抗RAAS系统，把醛固酮的作用反过来
    1. ANP主要来源于心房
    2. BNP主要来源于心室
  5. BNP增高→提示心衰
  6. 血清肌钙蛋白增高→提示心肌损伤

## 心肌肥厚与重构

一、心肌肥厚机制
  1. 好处：收缩力增加，心排量增加
  2. 代价：顺应性降低，室壁应力增加
  3. 早期：好处占主要，维持心功能正常
  4. 晚期：心衰
  5. 收缩和舒张速度都降低（与顺应性降低有关）
  6. 通俗理解：变胖了→行动反应变慢，但力气变大

二、心室重构
  1. 心肌纤维化，心肌肥厚
  2. 线粒体增多落后于心肌纤维化→ATP产生减少→心肌功能不足
  3. Ca2+内流不消耗ATP，但流出需要Ca泵消耗ATP→心肌舒张延缓
  4. 收缩速度取决于横桥周期长短，消耗ATP

三、舒张性vs收缩性心衰
  1. 舒张性心衰可单独发生
  2. 收缩性心衰一定伴有舒张性心衰
  3. 肌肉收缩：Ca2+顺浓度差易化扩散释放→粗细肌丝滑行（容易）
  4. 肌肉舒张：Ca泵主动转运回吸收（困难）
  5. 所以舒张障碍总是发生在早期
  6. 顺序：早期代偿→向心性肥厚（舒张障碍）→晚期失代偿→离心肥厚（收缩障碍）
  7. 心脏疾病最终都会是收缩障碍

## 特殊体征与检查

一、脉搏
  1. 交替脉：一下强一下弱——左心衰
  2. 水冲脉：如潮水——高排量心衰
  3. 奇脉（吸停脉）：心室舒张受限

二、6分钟步行试验
  1. <150m：重度心衰
  2. 150~450m：中度心衰
  3. >450m：轻度心衰

三、心衰分级
  1. NYHA分级（非急性）：一无二轻三明显四级不动也困难，端坐夜呼三
  2. Killip分级（急性心梗）：一无二啰半三肿四休克，上肺三级上中四

四、水肿鉴别
  1. 心衰患者：身体低垂部位
  2. 眼睑颜面：肾性水肿
  3. 双手：静脉或淋巴回流障碍
  4. 腹部：腹腔积液
  5. 水肿机制：
    1. 血浆胶体渗透压降低：低蛋白性水肿
    2. 粘多糖沉积：甲低粘液性水肿
    3. 小动脉壁通透性增加：炎性水肿
    4. 淋巴回流受阻：乳腺癌、丝虫病
    5. 右心衰：毛细血管静水压升高

五、消化道出血检查
  1. 首选胃镜
  2. 无胃镜选B超，临床直接增强CT
  3. 钡餐禁用：梗阻和出血
  4. 上中下消化道分界：Treitz韧带/回盲部
  5. 出血量与表现：
    1. 5ml：潜血试验阳性
    2. 50ml：黑便
    3. 250ml：呕血
    4. 400ml：全身症状
    5. 1000ml：休克
  6. 尿素氮：出血后数小时上升，24-48h高峰，3-4日正常
  7. 早期看循环状态，血象无明显变化
  8. 用药：
    1. 溃疡性出血：奥美拉唑（抑酸）
    2. 门脉性出血：生长抑素
    3. 大咯血：垂体后叶素

## 心包疾病速记

一、纤维蛋白性心包炎
  1. 积液少，纤维蛋白成分多
  2. 特征：心包摩擦音
  3. 口诀：摩擦蛋

二、渗出性心包炎
  1. 积液多，心脏舒张受限
  2. 回心血量减少，淤积在肺中
  3. 特征：呼吸困难
  4. 口诀：深呼吸（渗=渗出性，呼吸=呼吸困难）
  5. 贝克三联征（Beck）：血压下降、颈静脉怒张、心音遥远
  6. 心脏压塞：水在心脏→心脏跳不动→心音低钝→动脉血压低→静脉血多（颈静脉怒张）

三、规律
  1. 急非特和风湿性心包炎：积液量少，心包摩擦音明显，胸痛明显
  2. 特殊：心梗后综合征胸痛明显，但心包摩擦音不明显（自己的问题导致的胸痛，不是纤维素渗出摩擦）

## 胰腺炎要点

一、病因差异
  1. 我国：胆道疾病
  2. 欧美：酒精

二、诊断指标
  1. 血清淀粉酶>500：急性胰腺炎
  2. 腹痛高热黄疸+血清淀粉酶≤440：急性胆管炎
  3. 腹痛为首发症状，但老年人等体弱者可腹痛不明显

三、并发症时间
  1. 脓肿：2-3周
  2. 假性囊肿：3-4周
  3. 腹部肿块+高热+2周：胰腺脓肿
  4. 4周+不发热+肿块：假性囊肿

四、淀粉酶特点
  1. 水肿型：血淀粉酶升高
  2. 坏死型：血淀粉酶可升可降（不能判断严重程度）
  3. 重症胰腺炎：血清淀粉酶可升高、正常或降低；腹水淀粉酶一定升高
  4. 淀粉酶值越高，诊断正确率越高

五、酶的作用
  1. 磷脂酶A：溶解细胞膜（磷脂双分子层）→胰腺细胞坏死
  2. 脂肪酶：分解长链脂肪→脂肪小滴→脂肪坏死液化
  3. 激肽释放酶：释放缓激肽→舒张血管→血压降低→休克

六、治疗
  1. 初期通常不需要手术，除非并发症（胰腺坏死、感染等）
  2. 呕吐后不缓解：①急性胰腺炎 ②碱性反流性胃炎 ③急性完全性输入袢梗阻
  3. 急性胰腺炎易高钾低钙

## 消化道出血用药要点

  1. 溃疡性出血：奥美拉唑（抑酸）
  2. 门脉性出血：生长抑素
  3. 大咯血：垂体后叶素

## 肝硬化要点

  1. 最常见并发症：上消化道出血
  2. 上消化道出血最常见原因：消化道溃疡

## 理解万岁

  1. 心衰所有因子都是增加的（考试一下就能选出来）
  2. 内皮素≠内皮依赖性释放因子，别混淆
  3. 只有心脏自己释放的才有益于自己
  4. 舒张障碍总是发生在早期，因为钙重吸收更困难
  5. 心脏疾病最终都会是收缩障碍
  6. 心肌肥厚=变胖了，行动反应变慢，但力气变大
  7. 记住机制比死记答案重要，但考试别想太多

## 洋地黄药物

- 常考适应证：心脏扩大（最好有舒张期奔马律）→ 对洋地黄药物最敏感
- 洋地黄中毒：
  - 一般中毒用钾
  - 快速中毒用苯妥英钠
  - 缓慢中毒用阿托品
- 心脏性毒性机制：
  - 心肌细胞膜内 K+ 减少
  - 洋地黄与 Na+-K+-ATP 酶结合，竞争抑制钠泵
  - 导致细胞内 Na+ 增加，Ca+ 增加，加强心肌收缩力
  - 但中毒时抑制钠泵，导致细胞内 K+ 减少，心肌细胞自律性增加，出现心律失常
- 最特征性心电图表现：室早二联律、ST 段鱼钩样改变
- 血钾浓度：低钾时易发生中毒，所以补钾
- 快速性心律失常：利多卡因、苯妥英钠（有特异性）
  - 室早二联律：苯妥英钠
  - 室颤：利多卡因
- 缓慢性心律失常：阿托品（0.5-1mg 静脉注射）
- 血钾正常：补钾无效，用利多卡因或苯妥英钠
- 血钾低：补钾有效
