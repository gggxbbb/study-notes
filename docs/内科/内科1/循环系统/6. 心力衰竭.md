# 6. 心力衰竭

## 射血分数(EF)

- LVEF 正常范围：50%~70%

## 按 EF 分类

### 1. EF 降低性 HF（收缩性 HF）

- LVEF < 40%
- 如：高血压病晚期、扩张型心肌病

### 2. EF 保留性 HF（舒张性 HF）

- LVEF ≥ 50%
- 常见原因：
  - 高血压病早期（最常见）
  - 冠心病早期
  - 肥厚型心肌病、限制性心肌病、缩窄性心包炎
- 发病机制：缺氧 → ATP↓ → 肌浆钙泵↓ → 舒张障碍

### 3. EF 中间范围 HF

- LVEF 40%~49%
- 轻度收缩障碍 + 舒张功能不全

## 无症状性 HF

- 特点：只是无症状，其他都有
- 左室功能↓
- 射血分数(LVEF) < 50%
- 神经-体液激活：交感 N、RAS 系统激活

## Killip 分级（急性心肌梗死）

- I 级：无心力衰竭（无啰音）
- II 级：轻度心力衰竭（50% 以下肺野）
- III 级：严重心力衰竭（大于 50% 肺野）
- IV 级：心源性休克（血压小于 90/60）
- 有湿罗音就是心力衰竭：
  - 小于 50% 肺野 → 心力衰竭 2 级
  - 大于 50% 肺野 → 心力衰竭 3 级

## 血流动力学分类

- 心脏指数（CI）：
  - 冷 CI：
    - 干（无淤血）：低血容量 → 补充血容量
    - 湿（有淤血）：肺水肿 → 利尿、扩管
  - 暖 CI：
    - 干（无淤血）：非心源性休克 → 心源性给予扩管药物
    - 湿（有淤血）：
      - 高血压→利尿扩管
      - 低血压→强心扩管（多巴酚丁胺、硝普钠）

## 分期

- 心衰代偿期：无肺循环淤血、体循环淤血
  - 代偿期：左室功能下降，心脏的射血分数可以正常（LVEF > 50%），可以是射血分数保留心衰（舒张性心衰），不能耐受劳力
- 心衰失代偿期：存在肺淤血、体循环淤血
  - 是射血分数下降的心衰（收缩性心衰）

## 心脏参数

- 心脏前负荷：前负荷取决于血容量多少，前负荷取决于容量负荷
- 后负荷：是压力负荷，看收缩压和舒张压，引起血压升高的疾病会引起后负荷增加
- 高血压导致心脏负荷增加：后负荷
- 二尖瓣关闭不全导致心脏负荷增加：前负荷（容量负荷）

## 检查

- 对心力衰竭最有价值的检查：心导管检查

## 洋地黄药物

- 常考适应证：心脏扩大（最好有舒张期奔马律）→ 对洋地黄药物最敏感
- 洋地黄中毒：
  - 一般中毒用钾
  - 快速中毒用苯妥英钠
  - 缓慢中毒用阿托品
- 心脏性毒性机制：
  - 心肌细胞膜内 K+ 减少
  - 洋地黄与 Na+-K+-ATP 酶结合，竞争抑制钠泵
  - 导致细胞内 Na+ 增加，Ca+ 增加，加强心肌收缩力
  - 但中毒时抑制钠泵，导致细胞内 K+ 减少，心肌细胞自律性增加，出现心律失常
- 最特征性心电图表现：室早二联律、ST 段鱼钩样改变
- 血钾浓度：低钾时易发生中毒，所以补钾
- 快速性心律失常：利多卡因、苯妥英钠（有特异性）
  - 室早二联律：苯妥英钠
  - 室颤：利多卡因
- 缓慢性心律失常：阿托品（0.5-1mg 静脉注射）
- 血钾正常：补钾无效，用利多卡因或苯妥英钠
- 血钾低：补钾有效
