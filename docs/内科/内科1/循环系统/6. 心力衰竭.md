# 6. 心力衰竭

## 射血分数(EF)

- LVEF 正常范围：50%~70%

## 按 EF 分类

### 1. EF 降低性 HF（收缩性 HF）

- LVEF < 40%
- 如：高血压病晚期、扩张型心肌病

### 2. EF 保留性 HF（舒张性 HF）

- LVEF ≥ 50%
- 常见原因：
  - 高血压病早期（最常见）
  - 冠心病早期
  - 肥厚型心肌病、限制性心肌病、缩窄性心包炎
- 发病机制：缺氧 → ATP↓ → 肌浆钙泵↓ → 舒张障碍

### 3. EF 中间范围 HF

- LVEF 40%~49%
- 轻度收缩障碍 + 舒张功能不全

## 无症状性 HF

- 特点：只是无症状，其他都有
- 左室功能↓
- 射血分数(LVEF) < 50%
- 神经-体液激活：交感 N、RAS 系统激活
