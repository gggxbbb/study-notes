# 1. 急性胃炎与慢性胃炎

## 应激性溃疡（急性胃黏膜病变）

一、Curling溃疡
  1. 定义：严重烧伤后引发的胃、十二指肠黏膜急性病变
  2. 机制：烧伤导致机体强烈应激，胃酸分泌增加、黏膜屏障受损，进而形成溃疡
  3. 属于应激性溃疡/急性糜烂性溃疡

二、Cushing溃疡
  1. 定义：严重脑外伤、大手术后引发的应激性溃疡
  2. 机制：中枢性应激反应导致胃酸分泌增加、黏膜屏障受损

三、应激性溃疡常见病因
  1. 休克
  2. 复合性创伤
  3. 严重感染
  4. 严重烧伤（Curling溃疡）
  5. 严重脑外伤（Cushing溃疡）
  6. 大手术

四、发病率
  1. 应激性溃疡或急性糜烂性溃疡约占20%

## 记忆口诀

一、Curling溃疡 vs Cushing溃疡
  1. Curling溃疡：源于严重烧伤，记"头发被烧卷（Curling）"
  2. Cushing溃疡：源于严重脑外伤，记"库欣病在大脑"，源于中枢神经损伤
