# 2. 消化性溃疡

## 病因和发病机制

一、Hp感染
  1. 最常见病因
  2. DU（十二指肠溃疡）感染率90%以上
  3. GU（胃溃疡）感染率60~90%

二、胃酸和胃蛋白酶
  1. 胃酸、胃蛋白酶对胃黏膜的侵袭作用与黏膜屏障的防御能力间失去平衡

三、药物
  1. 长期服用NSAIDs（阿司匹林）、糖皮质激素、氯吡格雷等
  2. NSAIDs引起溃疡机制：抑制胃壁组织COX-1生成PG（胃壁PG对胃黏膜细胞有保护作用）

四、其他
  1. 黏膜防御与修复异常
  2. 遗传易感性

五、常见诱因
  1. 大量饮酒、长期吸烟、应激

## 临床表现

一、症状
  1. 上腹痛，呈慢性、周期性、季节性（秋冬、冬春之交）
  2. 节律性：
    1. GU（胃溃疡）：进食痛，进食后1h疼痛，2h逐渐缓解
    2. DU（十二指肠溃疡）：饥饿痛、夜间痛，进餐缓解，进食后2-4h再痛
    3. 口诀：GU=餐后痛，DU=空腹/夜间痛，进食缓解

二、体征
  1. 发作时剑突下、上腹部有局限性压痛
  2. 缓解后无体征

三、特殊溃疡

| 类型 | 特点 |
|------|------|
| 复合溃疡 | 胃和十二指肠均有活动性溃疡 |
| 幽门管溃疡 | ①"一快二无三易"：一快（餐后很快疼痛）二无（无节律、抗酸无效）三易（易出血、穿孔、幽门梗阻）②进食引起幽门蠕动加剧，刺激溃疡→餐后很快/立即疼痛 ③食物在胃或排空进入十二指肠，幽门都要蠕动→腹痛无节律，无明显缓解 ④幽门管血液循环差，抗酸剂药物浓度低→抗酸剂无效 ⑤幽门管血液循环差→易出现并发症且不易愈合 |
| 球后溃疡 | 发病快、穿孔快、无节律、抗酸药无效、易出血 |
| 巨大溃疡 | 直径>2cm |
| 老年人及儿童溃疡 | 老年人临床表现不典型，可无症状，疼痛多无规律；儿童期溃疡疼痛可在脐周 |
| 难治性溃疡 | 经正规抗溃疡治疗而溃疡未愈合 |

## 并发症（高频考点）

| 并发症 | 核心表现 |
|--------|----------|
| 出血 | 最常见并发症；上消化道出血最常见的病因是消化性溃疡。表现为大便隐血试验阳性、黑便，重者可有呕血、暗红色血便。十二指肠溃疡上血首选胃镜检查。 |
| 穿孔 | 突发剧烈腹痛，上腹→全腹，腹壁板样僵直，压痛、反跳痛阳性，肝浊音界消失。十二指肠溃疡后壁穿孔易出血。穿孔首选腹部立位平片（注意不是钡餐）。 |
| 幽门梗阻 | ①症状：上腹胀痛、餐后加重、吐后缓解，呕吐物为宿食；②体征：胃蠕动波、振水音阳性；③其他：失水、低氯低钾碱中毒、体重下降、营养不良 |
| 癌变 | DU一般不发生癌变；胃镜+活检有助于鉴别诊断 |

## 诊断公式

  1. 胃溃疡 = 上腹痛 + 规律餐后痛 + 上腹部压痛
  2. 十二指肠溃疡 = 上腹痛 + 夜间痛、饥饿痛 + 进食缓解
  3. 溃疡合并出血 = 溃疡表现 + 消化道出血表现（黑便、血压下降、心率增快、Hb下降、便潜血阳性等）
  4. 溃疡合并穿孔 = 溃疡表现 + 突发剧烈腹痛 + 腹膜炎体征 + 肝浊音界消失 + 立位腹平片可见游离气体
  5. 溃疡合并幽门梗阻 = 溃疡表现 + 餐后上腹胀痛 + 呕吐隔夜宿食、吐后腹痛缓解 + 振水音阳性 + 低钾低氯碱中毒
  6. 胃溃疡合并癌变 = 腹痛节律发生改变 + 体重下降

## 实验室和其他检查

一、上消化道内镜+活检
  1. 诊断首选、"金标准"
  2. 可确定有无病变及部位
  3. 鉴别良恶性
  4. 对合并出血者给予治疗

二、X线钡剂造影
  1. 了解胃的运动情况
  2. 胃镜禁忌症时选用

三、Hp检查

| 类型 | 方法 | 说明 |
|------|------|------|
| 非侵入性 | ¹³C或¹⁴C尿素呼气试验 | 首选 |
| 侵入性 | 快速尿素酶试验 | 首选 |
| 侵入性 | 胃黏膜组织切片染色镜检 | — |
| 侵入性 | 细菌培养 | 多用于科研 |

  1. 血清学Hp抗体检查不易出现假阴性

## 鉴别诊断

一、胃泌素瘤/佐林格-埃里森综合征/Zollinger-Ellison
  1. 一种内分泌肿瘤，可促进胃酸分泌、细胞增殖、胃肠运动
  2. 特征：多发溃疡、部位不典型、易出现并发症、正规抗溃疡药物疗效差
  3. 应尽可能手术切除肿瘤，否则溃疡反复发生

## 治疗

一、药物治疗

1. 抑制胃酸分泌
  1. H₂受体拮抗剂：西咪替丁、雷尼替丁、法莫替丁等
  2. 质子泵抑制剂（PPI）：首选药物，作用机制为不可逆抑制H⁺-K⁺-ATP酶
    1. PPI用于溃疡出血的"一箭多雕"：
      1. 抑酸→抑制胃蛋白酶活性→减少血凝块被消化（维持止血）
      2. 抑酸→pH↑→胃内pH提升后利于血小板聚集（促进止血）

2. 根除Hp
  1. Hp阳性患者均应进行治疗根除
  2. 根除Hp可显著降低溃疡的复发率
  3. 推荐四联：PPI + 胶体铋 + 两种抗生素（阿莫西林和甲硝唑等）

3. 保护胃黏膜
  1. 铋剂：保护胃黏膜、抑制Hp，但长期服用铋会蓄积，可引起舌苔、粪便变黑
  2. 弱碱性抗酸药：硫糖铝、氢氧化铝凝胶等
  3. 前列腺素类药物

二、内镜治疗
  1. 活动性出血首选内镜+PPI治疗
  2. 必要时手术

三、手术治疗适应症
  1. 并发消化道大出血经药物、内镜及血管介入治疗无效时
  2. 急性穿孔
  3. 慢性穿透溃疡
  4. 瘢痕性幽门梗阻，内镜治疗无效
  5. GU疑有癌变

## 消化道出血处理

- 消化性溃疡出血：奥美拉唑（抑酸）
- 溃疡性出血就用抑酸
- 上消化道出血最常见的原因：消化性溃疡
