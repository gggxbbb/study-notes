# 2. 肺结核

## 诊断公式
- 青壮年 + 咯血 + 午后低热 + 夜间盗汗 + 抗生素无效
- 关键特征：午后低热、盗汗

## 结核病分型诊断

| 类型 | 诊断公式 | 关键特征 |
|------|----------|----------|
| 结核性胸膜炎 | 结核 + 胸膜积液（胸痛 + 语颤消失 + 叩诊实音/呼吸音消失） | 胸膜积液体征 |
| 结核性心包炎 | 结核 + 心包积液（心前区痛 + 呼吸困难 + 上腹部闷胀 + 下肢浮肿） | 心包积液 |
| 肠结核 | 结核 + 腹部症状（腹痛、腹泻、右下腹肿块） | 右下腹肿块 |
| 结核性腹膜炎 | 结核 + 腹部炎症（腹痛、腹泻、腹壁柔韧感） | 腹壁柔韧感 |
| 肾结核 | 结核 + 膀胱刺激征 + 肾实质变薄并有破坏 | 膀胱刺激征 |
