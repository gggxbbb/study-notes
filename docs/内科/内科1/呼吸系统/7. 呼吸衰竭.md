# 7. 呼吸衰竭

## 酸碱失衡（哮喘视角补充）

| 阶段 | 酸碱状态 | 机制 |
|------|----------|------|
| 早期 | 呼吸性碱中毒（呼碱） | 呼吸频率代偿性增快，CO₂呼出过多 |
| 重症 | 呼吸性酸中毒（呼酸） | 通气严重受阻，CO₂潴留 |
| 晚期 | 代谢性酸中毒（代酸） | 缺氧时间长，酸性代谢产物堆积 |

## 氧疗原则
- 慢性支气管炎、COPD → 持续低流量吸氧
- ARDS → 呼气末正压通气（PEEP）
- 气性坏疽、CO 中毒 → 高压氧疗
- 肺炎 → 无控制给氧
