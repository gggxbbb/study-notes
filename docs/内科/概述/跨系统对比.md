### 咯血机制对比（呼吸系统 vs 循环系统）

| 疾病 | 咯血类型 | 机制 |
|------|----------|------|
| **肺结核** | 痰中带血 | 毛细血管扩张、通透性增高 |
| **肺结核** | 咯血量多 | 支气管动脉损伤或空洞血管瘤破裂 |
| **支气管扩张** | 大咯血 | 支气管动脉损伤 |
| **二尖瓣狭窄** | 痰中带血 | 毛细血管扩张、通透性增高、破裂，肺静脉压升高 |
| **二尖瓣狭窄** | 粉红色泡沫痰 | 毛细血管破裂 |
| **二尖瓣狭窄** | 大咯血 | 左房压力突然升高 → 支气管静脉破裂 |
| **二尖瓣狭窄** | 胶冻状暗红色痰 | 晚期并发右心衰 → 附壁血栓脱落 → 肺栓塞 → 肺梗死 |
