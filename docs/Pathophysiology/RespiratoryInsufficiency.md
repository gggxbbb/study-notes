# 第十六章 肺功能不全

## 教学要求
- **掌握**：呼吸衰竭概念、判断标准、分类和发病机制

## 关键内容

### 概念与诊断标准
- **呼吸衰竭**：PaO₂ < 60 mmHg（伴或不伴PaCO₂ > 50 mmHg）
- **Ⅰ型**：PaO₂↓，PaCO₂正常/↓（换气障碍）
- **Ⅱ型**：PaO₂↓，PaCO₂↑（通气障碍+换气障碍）

### 发病机制
| 机制 | 类型 | 说明 |
|------|------|------|
| **通气障碍** | 限制性 | 胸廓/呼吸肌病变→肺扩张受限→肺泡通气量↓ |
| | 阻塞性 | 气道阻力↑（COPD、哮喘）→通气不足 |
| **换气障碍** | 弥散障碍 | 膜增厚/面积↓（肺纤维化、肺水肿） |
| | V/Q比例失调 | 静-动脉样分流效应（V/Q↓）或死腔样通气（V/Q↑） |
| | 解剖分流增加 | 肺不张、肺实变→右向左分流 |

### 主要功能代谢变化
| 系统 | 变化 |
|------|------|
| **酸碱电解质** | Ⅱ型→呼酸+代酸；Ⅰ型→呼碱+代酸 |
| **呼吸** | 低O₂→呼吸兴奋；CO₂潴留→CO₂麻醉 |
| **循环** | 肺动脉高压→右心衰（肺心病） |
| **CNS** | 缺氧→精神障碍；CO₂潴留→肺性脑病 |
| **肾** | 肾功能障碍 |
| **胃肠** | 应激性溃疡、出血 |
