# 肝功能不全

## 基本概念

- **肝功能不全**：各种致肝损伤因素→肝细胞严重损伤→代谢、分泌、合成、解毒和免疫功能障碍
- **肝功能衰竭**：肝功能不全的晚期阶段→肝性脑病、肝肾综合征等

## 肝性脑病

### 概念

严重肝病→代谢紊乱→中枢神经系统功能障碍→以意识障碍、行为异常和昏迷为主要表现的综合征

### 发病机制（氨中毒学说——最主要）

- 肝功能严重障碍→**尿素合成↓**→**血氨↑**
- 门-体侧支循环→肠道吸收的氨绕过肝脏直接入体循环

**氨对脑的毒性**：
1. 干扰脑能量代谢：NH₃ + α-酮戊二酸→谷氨酸→消耗α-酮戊二酸→**TCA循环↓→脑ATP↓**
2. NH₃ + 谷氨酸→谷氨酰胺（ATP供能）→进一步消耗ATP
3. 抑制性神经递质（GABA）↑、兴奋性神经递质（谷氨酸）↓
4. 干扰神经细胞膜Na⁺/K⁺-ATP酶

### 诱发因素

| 诱因 | 机制 |
|------|------|
| 消化道出血 | 血液→肠道细菌分解→**氨产生↑**（最主要诱因） |
| 高蛋白饮食 | 蛋白质分解→氨↑ |
| 碱中毒 | NH₄⁺→NH₃→更易透过血脑屏障 |
| 感染/缺氧 | 加重肝损伤、脑能量代谢障碍 |
| 镇静剂/麻醉剂 | 肝功能↓→药物蓄积、GABA受体敏感性↑ |

### 假性神经递质学说

- 肝衰→芳香族氨基酸↑→**苯乙醇胺、羟苯乙醇胺**（假性神经递质）取代多巴胺、去甲肾上腺素→**脑干网状结构功能障碍**→意识障碍、扑翼样震颤

## 肝肾综合征

- 严重肝病→肾血管收缩→**功能性肾衰竭**
- 特点：肾脏本身无器质性病变（肾移植给肝功能正常者可恢复功能）

## 黄疸（参考肝的生物化学章节）

- 肝细胞处理胆红素障碍→高胆红素血症→皮肤巩膜黄染
