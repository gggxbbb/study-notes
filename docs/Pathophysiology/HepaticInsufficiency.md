# 第十七章 肝功能不全

## 教学要求
掌握肝功能不全的概念、肝性脑病的主要学说及黄疸鉴别、肝肾综合征机制。

## 关键内容

### 肝功能不全概念
肝细胞严重受损→代谢、解毒、合成、分泌等功能障碍→全身性病理状态。

### 肝性脑病
| 学说 | 要点 |
|------|------|
| **氨中毒学说** | 肝解毒↓→血氨↑→NH₃入脑→干扰能量代谢、神经递质→意识障碍 |
| **假性神经递质学说** | 芳香族氨基酸↑→生成苯乙醇胺、羟苯乙醇胺→竞争取代正常递质（NE、多巴胺）→中枢抑制 |
| **GABA学说** | 肠源性GABA入血↑→脑内GABA受体↑→中枢神经抑制 |

### 黄疸
| 类型 | 病因 | 血胆红素 | 尿胆红素 |
|------|------|---------|---------|
| **肝前性（溶血性）** | 溶血↑ | 非结合型↑ | - |
| **肝细胞性** | 肝细胞损伤 | 结合+非结合均↑ | + |
| **肝后性（梗阻性）** | 胆道梗阻 | 结合型↑ | + |

### 肝肾综合征
- 严重肝病→肾血管收缩→肾小球滤过率↓→功能性肾衰竭
- 肾脏本身无器质性病变
