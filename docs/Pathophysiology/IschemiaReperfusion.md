# 第十二章 缺血-再灌注损伤

## 教学要求
- **掌握**：概念；发生机制（自由基、钙超载、白细胞的作用）
- **熟悉**：原因和影响因素

## 关键内容

### 概念
缺血组织恢复血流灌注后，损伤反而加重的现象。

### 原因和影响因素
| 因素 | 影响 |
|------|------|
| 缺血时间 | 时间过长或过短损伤轻，特定时间窗口最重 |
| 侧支循环 | 侧支丰富→损伤轻 |
| 需氧程度 | 需氧高的组织（心、脑）更敏感 |
| 再灌注条件 | 低压低温再灌注可减轻损伤 |

### 发生机制

**① 氧自由基生成增多**
- 黄嘌呤氧化酶途径（缺血期→XO↑→再灌注时产O₂⁻）
- 中性粒细胞呼吸爆发
- 线粒体电子漏

**② 钙超载**
- Na⁺/Ca²⁺交换增加（Na⁺-H⁺交换↑→胞内Na⁺↑→Na⁺/Ca²⁺交换↑）
- 生物膜损伤→Ca²⁺内流

**③ 白细胞作用**
- 趋化、黏附、浸润→释放炎症介质、蛋白酶→微血管阻塞、组织损伤

### 防治原则
- 缩短缺血时间、控制再灌注条件
- 自由基清除剂、钙拮抗剂、抗炎治疗
