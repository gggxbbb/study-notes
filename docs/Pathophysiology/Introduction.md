# 绪论

## 病理生理学的任务

- 研究疾病发生的原因和条件
- 研究疾病全过程中机体机能、代谢的动态变化及发生机制
- 揭示疾病发生、发展和转归的规律，为疾病防治提供理论基础

## 主要内容

| 板块 | 内容 |
|------|------|
| **疾病概论** | 疾病发生发展的原因、基本机制和规律 |
| **基本病理过程** | 多种疾病中共同的、成套的功能和代谢变化（水电解质紊乱、酸碱失衡、缺氧、发热、休克、DIC等） |
| **各系统病理生理学** | 心/肺/肝/肾等功能不全 |

> **基本病理过程**：不同疾病中可出现的共同病理变化，如水电解质紊乱、缺氧、发热、休克等。区别于各论中特定器官的病理生理。

## 学科地位

- 病理生理学 = 基础医学 ↔ 临床医学的**桥梁学科**
- 与生理学、生物化学、病理学密切相关，为临床各科打基础

## 研究方法

- 动物实验（最主要）、临床观察、流行病学调查
