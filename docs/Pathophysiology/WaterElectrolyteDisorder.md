# 水、电解质代谢紊乱

## 水钠代谢基础知识

- **体液容量**：成人体液占体重60%（细胞内液40% + 细胞外液20%）
- 细胞外液中：组织间液15%，血浆5%
- 体液渗透压：280~310mOsm/L，Na⁺是细胞外液渗透压的主要决定因素

## 脱水

| 类型 | 失水vs失钠 | 血清Na⁺ | 渗透压 | 主要病因 |
|------|-----------|---------|--------|---------|
| **高渗性脱水** | 失水>失钠 | >150mmol/L | ↑ | 饮水不足、大量出汗、尿崩症 |
| **低渗性脱水** | 失钠>失水 | <130mmol/L | ↓ | 大量呕吐/腹泻后只补水、利尿剂 |
| **等渗性脱水** | 水钠等比例丢失 | 130~150 | 正常 | 呕吐/腹泻/大面积烧伤（最常见） |

### 高渗性脱水特征

- 口渴明显、尿少、尿比重高
- 细胞内液向细胞外转移→**细胞脱水**→脑细胞脱水→中枢神经功能障碍（烦躁、昏迷）

### 低渗性脱水特征

- 无口渴（早期）、尿量可正常
- 细胞外液向细胞内转移→**细胞水肿**→血容量↓→休克倾向→**脱水征**（皮肤弹性↓、眼窝凹陷）

## 水肿

- 定义：过多液体在组织间隙或体腔内积聚
- 机制：

| 机制 | 说明 |
|------|------|
| 毛细血管流体静压↑ | 静脉回流受阻（心衰、静脉血栓） |
| 血浆胶体渗透压↓ | 血浆蛋白↓（肾病综合征、肝硬化、营养不良） |
| 毛细血管通透性↑ | 炎症、过敏 |
| 淋巴回流受阻 | 丝虫病（象皮肿）、肿瘤压迫 |

## 低钾血症和高钾血症

| | 低钾血症 | 高钾血症 |
|---|---------|---------|
| 标准 | [K⁺] < 3.5mmol/L | [K⁺] > 5.5mmol/L |
| 主要病因 | 摄入不足、丢失过多（呕吐/腹泻/利尿剂）、向细胞内转移（碱中毒） | 肾排钾↓（肾衰最常见）、细胞内K⁺外移（酸中毒、组织破坏） |
| **骨骼肌** | 肌无力、软瘫、呼吸肌麻痹 | 轻度兴奋→重度去极化阻滞→肌无力 |
| **心脏** | 心律失常、T波低平倒置、U波、ST压低 | **最危险！** T波高尖、P波消失、QRS增宽→室颤、心脏骤停 |
| 酸碱 | 可致碱中毒（反常性酸性尿） | 可致酸中毒（反常性碱性尿） |
