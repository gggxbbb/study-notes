# 健康与疾病

## 健康
躯体、精神和社会适应上的完好状态。

## 亚健康
介于健康与疾病之间的非完美状态，呈动态变化。

## 疾病
在病因作用下，机体内稳态调节紊乱而导致的异常生命活动。

### 疾病的特点
- 病因是必不可少的，没有无病因的疾病
- 基础是内稳态/自稳调节机制紊乱
- 表现为功能、代谢和/或形态结构变化（症状+体征）

## 病因学
- **病因**：引起疾病、决定疾病特异性的因素
- **条件**：影响疾病发生发展的因素
- **诱因**：作用于病因或机体促进疾病发生的因素

## 发病学
### 疾病发展的一般规律
- **因果转化**：原始病因→结果作为新原因→推动疾病发展
- **损伤与抗损伤**：推动疾病发展的核心矛盾
- **局部与整体**：互相影响

### 疾病发生的基本机制
- 神经机制、体液机制、细胞和分子机制

## 疾病的经过与转归
1. 潜伏期 → 前驱期 → 症状明显期 → 转归期
2. **转归**：康复（完全/不完全）、死亡
3. **脑死亡**：全脑功能不可逆丧失；判断标准（自主呼吸停止、瞳孔散大固定、脑干反射消失、脑电波消失）

# 发病学基础（拓展）
- **内稳态失衡**：疾病发生的基础
- **因果交替**：推动疾病发展
- **局部与全身**：相互关联
