# 疾病概论

## 基本概念

| 概念 | 定义 |
|------|------|
| **健康** | 身体上、精神上和社会适应上的完好状态 |
| **疾病** | 机体在一定条件下，由病因与机体相互作用产生的损伤与抗损伤斗争过程 |
| **病因** | 能引起疾病并赋予该疾病特征性的因素 |
| **条件** | 能影响疾病发生发展的因素（如年龄、性别、营养） |
| **诱因** | 能加强病因作用、促进疾病发生的因素 |

> 病因决定疾病性质，条件影响疾病发生发展

## 疾病发生发展的一般规律

- **因果交替**：原始病因→机体变化→新病因→新变化…→恶性循环或良性循环
- **损伤与抗损伤**：贯穿疾病全过程，决定疾病的发展和转归
- **局部与整体**：局部病变可影响全身，全身状态可影响局部

## 疾病的转归

- **完全康复**：形态结构修复 + 功能代谢恢复正常
- **不完全康复**：主要症状消失，但遗留病理状态（如烧伤后瘢痕）
- **死亡**：
  - 传统概念：心跳和呼吸停止
  - **脑死亡**：全脑功能不可逆丧失。判定标准——不可逆性昏迷、自主呼吸停止（需呼吸机维持）、脑干反射消失（瞳孔对光/角膜/眼前庭反射等）、脑电波平坦。**脑死亡 = 死亡**
  - 脑死亡的意义：科学判定死亡终点、节约医疗资源、器官移植的合法时机
