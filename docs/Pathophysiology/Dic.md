# 第十四章 凝血与抗凝血平衡紊乱/DIC

## 教学要求
- **掌握**：DIC诱因、发病机制、临床表现
- **熟悉**：凝血/抗凝/纤溶系统

## 关键内容

### DIC概念
弥散性血管内凝血——凝血系统广泛激活→微血栓形成→凝血因子消耗+纤溶亢进→出血倾向。

### 发病机制
| 机制 | 说明 |
|------|------|
| **外源凝血激活** | 组织因子（TF）释放入血 |
| **内源凝血激活** | 胶原暴露→FXII激活 |
| **血细胞破坏** | RBC溶血→释放ADP、磷脂；WBC释放促凝物质 |
| **促凝物质入血** | 蛇毒、肿瘤细胞、胰蛋白酶等 |

### 影响因素
- 单核吞噬系统功能↓
- 肝功能↓（抗凝物质合成↓）
- 血液高凝状态（妊娠、肿瘤）
- 微循环障碍（休克、酸中毒）

### 分期分型
- **高凝期** → **消耗性低凝期** → **继发性纤溶亢进期**

### 临床表现
| 表现 | 机制 |
|------|------|
| **出血** | 凝血因子消耗、纤溶亢进 |
| **器官功能障碍** | 微血栓→缺血 |
| **休克** | 微循环障碍、出血 |
| **微血管病性溶血性贫血** | 纤维蛋白网切割RBC→裂红细胞 |
