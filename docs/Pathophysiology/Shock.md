# 第十三章 休克

## 教学要求
- **掌握**：概念、分期、早期代偿机制
- **熟悉**：原因分类；各期机制

## 关键内容

### 概念
有效循环血量锐减→组织微循环灌流不足→细胞代谢障碍→多器官功能障碍。

### 原因分类
| 分类 | 常见原因 |
|------|---------|
| 低血容量性 | 失血、失液、烧伤 |
| 心源性 | 心梗、心衰 |
| 分布性 | 感染性、过敏性、神经源性 |
| 梗阻性 | 肺栓塞、心包填塞 |

### 分期
| 分期 | 微循环特点 | 代偿/机制 |
|------|-----------|----------|
| **缺血性缺氧期（代偿期）** | 微血管收缩，少灌少流 | 交感-肾上腺髓质↑→维持血压；自身输血、自身输液 |
| **淤血性缺氧期（失代偿期）** | 微血管扩张，灌>流 | 酸中毒→血管对儿茶酚胺反应↓→淤血 |
| **微循环衰竭期（不可逆期）** | DIC，微血栓 | 凝血系统激活→器官衰竭 |

### 细胞代谢障碍
- 有氧氧化↓→无氧酵解↑→乳酸↑→代酸
- ATP↓→Na⁺-K⁺泵失灵→细胞水肿

### 器官功能障碍
- **心**、**肾**、**肺**（ARDS）、**脑**、**胃肠**、**肝**
