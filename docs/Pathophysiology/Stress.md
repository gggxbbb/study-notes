# 应激

## 基本概念

| 概念 | 定义 |
|------|------|
| **应激** | 机体在受到各种强烈刺激时出现的**非特异性**全身反应 |
| **应激原** | 能引起应激反应的因素（创伤、感染、精神紧张、缺氧等） |
| **热休克蛋白(HSP)** | 应激时合成增多的蛋白质，保护细胞免受损伤（"分子伴侣"） |
| **急性期蛋白(APP)** | 应激时血浆中浓度迅速变化的蛋白质（C反应蛋白、补体等） |

## 应激反应的神经内分泌基础

### 蓝斑-交感-肾上腺髓质系统

- 中枢：脑桥蓝斑
- 效应：交感兴奋→去甲肾上腺素和**肾上腺素↑**
- 表现：心率↑、血压↑、血糖↑、呼吸↑、精神兴奋
- 意义：快速动员、"战斗或逃跑"反应

### 下丘脑-垂体-肾上腺皮质轴（HPA轴）

- 通路：下丘脑(CRH) → 腺垂体(ACTH) → 肾上腺皮质(**皮质醇↑**)
- 皮质醇作用：
  - ↑糖异生→血糖↑
  - ↑脂肪分解→游离脂肪酸↑
  - ↑蛋白质分解→提供氨基酸
  - 抗炎/免疫抑制作用（大剂量）
- 意义：增强机体耐受力，但过度→免疫抑制→感染风险↑

## 应激的细胞反应

- **热休克蛋白**：分子伴侣→帮助新生蛋白折叠、修复变性蛋白
- **急性期反应**：C反应蛋白(CRP)→激活补体、调理吞噬

## 应激性溃疡

- 严重应激（烧伤、颅脑损伤、大手术）→胃十二指肠粘膜急性糜烂、溃疡→出血
- 机制：胃粘膜缺血（交感→血管收缩）+ 糖皮质激素→粘膜防御↓ → H⁺逆向扩散→损伤

---
**执业医师关联**：Cushing溃疡（颅脑损伤相关）和Curling溃疡（烧伤相关）是典型应激性溃疡
