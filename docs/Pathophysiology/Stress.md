# 应激概述

**应激**：机体在内外环境因素强烈刺激下，稳态发生改变与重塑，导致生理和心理行为的适应性反应。

**应激原**：引起应激反应的各种因素
- 物理性、化学性、生物性、心理社会性
- 躯体型 vs 心理社会型
- 急性 vs 慢性
- 生理性（良性） vs 病理性（劣性）

# 应激的神经内分泌反应

## 蓝斑-交感-肾上腺髓质系统（LSAM）
### 中枢效应
兴奋、警觉、专注；过度→焦虑、恐惧、愤怒

### 外周效应
- 心率加快、心肌收缩力加强 → 心排血量↑
- 血流重分布：皮肤/内脏血管收缩，冠脉/骨骼肌扩张
- 支气管扩张、通气增强
- 糖原分解↑、血糖↑、脂肪动员↑

### 不利影响
- 内脏缺血 → 胃肠黏膜糜烂/溃疡
- 血小板聚集↑ → 血栓形成风险
- 心肌缺血、心律失常

## 下丘脑-垂体-肾上腺皮质系统（HPAC）
CRH分泌→ACTH→GC分泌↑

### 中枢效应
- 适量CRH→兴奋/愉悦
- 过度CRH→焦虑、抑郁、记忆下降

### 外周效应（GC介导）
- 允许作用维持血压
- 促进糖异生、升高血糖
- 促进脂肪动员
- 稳定细胞膜、抗损伤
- 抑制炎症反应

### 不利影响
- 免疫力下降→易感染
- 抑制甲状腺/性腺→内分泌紊乱
- 胰岛素抵抗→血糖血脂升高

# 应激的免疫反应
- 急性应激→先天性免疫↑（中性粒细胞、单核细胞↑）
- 慢性应激→免疫抑制（GC和儿茶酚胺过度释放）

# 急性期反应（APR）
感染、创伤等诱发的快速防御反应

### 急性期蛋白（APP）
- CRP ↑1000倍以上，血清淀粉样蛋白A ↑1000倍
- α1-抗胰蛋白酶、纤维蛋白原↑2-3倍
- 铜蓝蛋白、补体C3↑50%
- 白蛋白、转铁蛋白反而↓

### APP功能
- **抗感染**：CRP激活补体、调理吞噬
- **抗损伤**：蛋白酶抑制物、清除氧自由基
- **调节凝血**：纤维蛋白原促进凝血
- **结合运输**：载体功能

# 细胞应激反应
有害因素导致生物大分子损伤时，细胞通过调节蛋白表达重建稳态。

### 热休克蛋白（HSP）
- 分子伴侣：帮助蛋白质折叠、复性、降解
- 应激时表达↑→细胞保护
- 按分子量分：HSP110、HSP90、HSP70、HSP60等

### 未折叠蛋白反应（UPR）
内质网中未折叠蛋白积聚→减少蛋白合成、促进降解

### 氧化应激
氧化剂产生过多/清除减少→细胞氧化损伤与抗损伤反应

# 应激与疾病

## 应激性溃疡
- 强烈应激导致的胃、十二指肠急性黏膜病变
- **机制**：胃肠黏膜缺血 + 黏膜屏障功能降低

## 心血管疾病
- 心源性猝死（最严重）
- 冠心病、高血压

## 免疫相关疾病
- 免疫功能失调
- 自身免疫性疾病诱发

## 精神障碍
- 急性应激障碍（ASD）→应激原消除后恢复
- 创伤后应激障碍（PTSD）→延迟出现、长期持续
- 抑郁症→重要诱发因素
