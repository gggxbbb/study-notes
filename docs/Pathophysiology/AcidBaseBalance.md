# 第四章 酸碱平衡和酸碱平衡紊乱

## 教学要求
- **掌握**：常用指标及其意义；四种单纯型酸碱平衡紊乱的概念、代偿调节、指标变化规律、对机体影响
- **熟悉**：酸碱中毒概念、原因、分类
- **了解**：酸碱物质来源；混合型类型

## 关键内容

### 酸碱平衡调节
| 调节方式 | 主要作用 |
|---------|---------|
| 血液缓冲 | 最快，即刻反应 |
| 肺调节 | 通过改变呼吸排出CO₂ |
| 肾调节 | 排H⁺保HCO₃⁻，作用持久 |
| 组织细胞调节 | 离子跨膜交换 |

### 常用指标
- **pH**：正常 7.35–7.45，<7.35 酸中毒，>7.45 碱中毒
- **PaCO₂**：正常 33–46 mmHg，反映呼吸性因素
- **SB/AB**：标准碳酸氢盐 / 实际碳酸氢盐
- **BB**：缓冲碱
- **BE**：碱剩余（正值=碱多，负值=酸多）
- **AG**：阴离子间隙，鉴别代酸类型

### 单纯型酸碱平衡紊乱
| 类型 | 原发性变化 | 代偿反应 |
|------|-----------|---------|
| 代酸 | HCO₃⁻↓ | PaCO₂↓（肺代偿） |
| 呼酸 | PaCO₂↑ | HCO₃⁻↑（肾代偿） |
| 代碱 | HCO₃⁻↑ | PaCO₂↑（肺代偿） |
| 呼碱 | PaCO₂↓ | HCO₃⁻↓（肾代偿） |

### 混合型
- 双重/三重紊乱，如呼酸+代酸、呼碱+代碱等

### 酸碱失衡判断步骤

1. **第一步：看 pH** — pH 7.35~7.45 为代偿性，不在这个区间为失代偿性
2. **第二步：看呼吸性因素**
   - PaCO₂ > 45 mmHg → 呼吸性酸中毒
   - PaCO₂ < 35 mmHg → 呼吸性碱中毒
   - 或：AB > SB → 呼吸性酸中毒；AB < SB → 呼吸性碱中毒
3. **第三步：看代谢性因素**
   - BE > +3 → 代谢性碱中毒
   - BE < -3 → 代谢性酸中毒
   - 或：AB = SB 时，AB > 27 → 代谢性碱中毒；AB < 22 → 代谢性酸中毒
4. **看血钾**
   - 血钾 < 3.5 mmol/L → 代谢性碱中毒
   - 血钾 > 3.5 mmol/L → 代谢性酸中毒

### 口诀

- 低钾低氯性碱中毒，反常性酸性尿
- 高钾高氯性酸中毒，反常性碱性尿
