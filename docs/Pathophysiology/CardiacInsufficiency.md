# 第十五章 心功能不全

## 教学要求
- **掌握**：概念、病因与诱因；代偿机制；心衰机制；呼吸困难形式及机制

## 关键内容

### 概念
心脏泵血功能↓→心输出量不能满足机体代谢需要。

### 病因与诱因
| 类别 | 内容 |
|------|------|
| **病因** | 原发性心肌舒缩障碍（心肌炎、缺血）、心脏负荷过度（高血压、瓣膜病） |
| **诱因** | 感染、水电解质紊乱、心律失常、妊娠、劳累 |

### 分类
- 左心衰 / 右心衰 / 全心衰
- 急性 / 慢性
- 收缩性 / 舒张性

### 心衰机制
- **收缩性减弱**：心肌细胞丢失、收缩蛋白破坏、能量代谢障碍、Ca²⁵转运异常
- **舒张功能障碍**：心室顺应性↓、心肌松弛性↓

### 代偿
| 代偿方式 | 机制 | 意义 |
|---------|------|------|
| 心率加快 | 交感↑ | 短期增加心输出量 |
| 心脏扩张 | Frank-Starling机制 | 增加每搏量 |
| 心肌肥大 | 向心性/离心性 | 长期代偿，但最终转失代偿 |
| 血容量↑、血流重分布 | 肾素-血管紧张素-醛固酮↑ | 维持灌注压 |

### 临床表现
- **肺循环充血/呼吸困难**（左心衰）：劳力性呼吸困难→端坐呼吸→夜间阵发性呼吸困难→肺水肿
- **体循环淤血**（右心衰）：颈静脉怒张、肝大、下肢水肿
- **心输出量不足**：疲乏、少尿、低血压
