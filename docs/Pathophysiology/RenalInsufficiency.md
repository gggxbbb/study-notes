# 第十八章 肾功能不全

## 教学要求
掌握急慢性肾衰竭的概念、分类、分期及肾性高血压/贫血/骨病的发生机制。

## 关键内容

### 急性肾衰竭（ARF）
**概念**：肾小球滤过率（GFR）急剧↓→氮质血症+水电解质酸碱平衡紊乱。

| 分类 | 原因 |
|------|------|
| **肾前性** | 有效循环血量↓（休克、脱水） |
| **肾性** | 肾实质损伤（急性肾小管坏死、肾小球肾炎） |
| **肾后性** | 尿路梗阻（结石、肿瘤） |

**少尿期**：
- 尿量 < 400 mL/d
- 高钾血症（最危险）、水中毒、代酸、氮质血症

**多尿期**：
- 肾小管修复但浓缩功能未恢复→尿量↑（>400 mL/d）
- 仍可发生脱水、电解质紊乱

### 慢性肾衰竭（CRF）
**分期**：

| 分期 | GFR (mL/min) | 表现 |
|------|-------------|------|
| 代偿期 | 50–80 | 无临床症状 |
| 失代偿期（氮质血症期） | 25–50 | 夜尿多、贫血、乏力 |
| 衰竭期 | 10–25 | 明显尿毒症症状 |
| 终末期（尿毒症期） | <10 | 多系统严重受累，需透析 |

### 继发性病变
- **肾性高血压**：水钠潴留、RAAS↑、肾内降压物质↓
- **肾性贫血**：EPO生成↓、毒素抑制骨髓、RBC寿命↓
- **肾性骨病**：1,25-(OH)₂D₃↓→低钙→PTH↑→骨质吸收
