# 肾功能不全

## 基本概念

- **肾功能不全**：各种病因→肾小球滤过和/或肾小管重吸收功能障碍→代谢废物潴留、水电解质和酸碱平衡紊乱、内分泌功能障碍的病理过程
- **急性肾衰竭(ARF)** vs **慢性肾衰竭(CRF)** vs **尿毒症**

## 急性肾衰竭(ARF)

### 概念和分类

- 各种原因→短时间内（数小时~数天）肾功能急剧↓→GFR↓→氮质血症、水电解质和酸碱紊乱

| 分类 | 病因 | 举例 |
|------|------|------|
| **肾前性** | 肾灌注不足 | 休克、脱水、心衰→有效循环血量↓ |
| **肾性** | 肾实质损伤（最常见） | 急性肾小管坏死(ATN)、急性间质性肾炎、肾小球肾炎 |
| **肾后性** | 尿路梗阻 | 结石、前列腺增生、肿瘤压迫 |

### 少尿型ARF分期

| 分期 | 特点 |
|------|------|
| **少尿期** | 尿量<400ml/d（最危险）。高钾血症（最致命）、代谢性酸中毒、水中毒、氮质血症 |
| **多尿期** | 尿量>400ml/d→可达数升。肾功能开始恢复，但大量失水→脱水、低钾、低钠 |
| **恢复期** | 尿量恢复、肾功逐渐恢复（可数月~1年） |

### 少尿期的高钾血症

- **最危险的并发症**——可致心脏骤停
- 机制：肾排钾↓ + 组织破坏→K⁺释放↑ + 酸中毒→K⁺从细胞内移出

## 慢性肾衰竭(CRF)

- 各种慢性肾病→肾单位进行性破坏→残余肾单位不能充分排出代谢废物→**氮质血症**

### 分期（按肾功能损害程度）

| 分期 | GFR | 特点 |
|------|-----|------|
| 代偿期 | >50%正常 | 无症状，血肌酐正常 |
| 失代偿期（氮质血症期） | 25~50% | 夜尿↑、轻度贫血 |
| 肾衰竭期 | 10~25% | 氮质血症、贫血、酸中毒 |
| **尿毒症期** | <10% | 尿毒症症状 |

### 功能代谢变化

- **水电解质**：多尿→低渗尿→夜尿；少尿→高钾、高磷、低钙（↑PTH→肾性骨营养不良）
- **酸中毒**：排酸↓→代谢性酸中毒
- **肾性贫血**：**EPO↓**（最主要原因）+ 毒素→红细胞寿命缩短
- **肾性高血压**：钠水潴留 + RAAS激活

## 尿毒症

- 急/慢性肾衰竭最严重阶段→代谢终产物和内源性毒物蓄积→**自身中毒**
- 毒素：尿素、肌酐、胍类化合物（甲基胍→神经毒性）、中分子物质
- 治疗：**透析**（血液透析/腹膜透析）、肾移植
