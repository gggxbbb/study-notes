# 卫生法知识点

## 医疗废物

- 医疗废物存放不超过 2 天（"废物两天"）

## 处方管理

- 一张处方不超过 5 种药（"处方5种"）
- 处方有效期不超过 3 天（"事不过三"）
- 长期用麻醉药品每 3 个月复诊一次

## 假药 vs 劣药

**假药：** 禁用的、变质的、被污染、超范围、未经批准（生产、进口、批准文号）

**劣药：** 有效期问题（无有效期、更改有效期、超有效期）、生产批号问题（不注明或更改）、药品问题（接触药品包装容器未经批准、擅自添加其他辅料、其他不符合药品标准规定的）

## 行政处罚

- 考核不合格→3~6个月
- 违反规章制度→6~12个月
- 严重后果→暂停执业医师
- 情节严重→吊销执业医师
- 违反规章制度→责任事故
- 专业技术水平和经验不足→技术事故

## 各级医疗机构职责

- 省级：产前遗传诊断、院内配制剂、放射治疗、核医学、疫源地大兴土木
- 市级：介入、事故鉴定、麻药精一
- 县级：传染病爆发、结扎流产、X线
