# 心律失常

## 定义
心脏起搏和传导功能紊乱而引起的心脏节律、频率或激动顺序的异常。

## 抗心律失常药物分类（Vaughan Williams）

| 分类 | 作用机制 | 代表药物 | 对APD/QT影响 |
|------|---------|---------|-------------|
| **Ⅰ类（钠通道阻滞剂）** | | | |
| Ⅰa | 阻滞钠通道+延长复极 | 奎尼丁、普鲁卡因胺 | 延长APD和QT间期 |
| Ⅰb | 阻滞钠通道+缩短复极 | **利多卡因**、苯妥英钠、美西律 | 缩短APD，不延长QT |
| Ⅰc | 阻滞钠通道（强） | **普罗帕酮（心律平）**、氟卡尼 | 延长APD，QT不明显延长 |
| **Ⅱ类** | β-受体阻滞剂 | **美托洛尔**、普萘洛尔、阿替洛尔 | — |
| **Ⅲ类** | 钾通道阻滞剂 | **胺碘酮（可达龙）**、索他洛尔 | 明显延长APD和QT |
| **Ⅳ类** | 钙通道阻滞剂 | **维拉帕米（异搏定）**、地尔硫䓬 | — |
| 其他 | — | 腺苷、洋地黄类 | — |

## 常见心律失常心电图特点

### 房颤（Atrial Fibrillation）
| 项目 | 特点 |
|------|------|
| ECG | 窦性P波消失，代之以不规整f波（V₁导联清楚），频率350~600次/min；心室率极不规整 |
| 听诊 | **心律绝对不齐 + S₁强弱不等 + 脉搏短绌** |
| 分类 | 阵发性（≤7d）、持续性（>7d）、长期持续性（≥1年）、永久性 |
| 治疗 | 控制心室率（Ⅱ/Ⅳ类）；转复（**胺碘酮首选**）；抗凝（CHA₂DS₂-VASc评分→华法林/NOAC） |

### 阵发性室上性心动过速（PSVT）
| 项目 | 特点 |
|------|------|
| ECG | 心率150~250次/min，RR规则，QRS多正常；逆行P波 |
| 特点 | **突发突止** |
| 治疗 | 刺激迷走神经→药物（**腺苷/维拉帕米首选**）→电复律(血流动力学不稳定)→**射频消融**（根治） |

### 室性心动过速（VT）
| 项目 | 特点 |
|------|------|
| ECG | 宽大畸形QRS（>0.12s），频率100~250次/min；**房室分离、心室夺获、室性融合波**（特征性） |
| 治疗 | 无血流动力学障碍→**胺碘酮/利多卡因**；有障碍→同步电复律；TDP→静注硫酸镁；ICD预防猝死 |

## 房室传导阻滞（口诀记忆法）

| 类型 | ECG特点 | 口诀 |
|------|---------|------|
| **一度AVB** | PR间期>0.20s | 每天回家比平时晚 |
| **二度Ⅰ型（文氏）** | PR间期逐渐延长→直至QRS脱落 | 越来越晚→突然不回家 |
| **二度Ⅱ型** | PR间期固定，间歇性QRS脱落 | 正常→毫无征兆不回家 |
| **三度AVB** | P波与QRS波完全无关 | 你过你的，我过我的 |

## 治疗原则
- **病因/诱因治疗**为主
- **控制心率和恢复节律**：药物（Ⅰ~Ⅳ类）、电复律（同步/非同步）
- **预防复发**：射频消融（根治性）、起搏器、ICD
