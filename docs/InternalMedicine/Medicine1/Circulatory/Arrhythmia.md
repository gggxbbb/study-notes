# 心律失常

## 定义

心脏起搏和传导功能紊乱而引起的心脏节律、频率或激动顺序的异常。

## 心律失常分类

| 分类 | 机制 | 举例 |
|------|------|------|
| **冲动起源异常** | 自律性↑/异常 | 窦速、房早、室早、房速、房颤 |
| **冲动传导异常** | 折返/阻滞 | 预激、AVB、束支阻滞、室上速（折返） |
| 混合型 | — | 并行心律 |

## 抗心律失常药物分类（Vaughan Williams）

| 分类 | 作用机制 | 代表药物 | 对 APD/QT 影响 |
|------|---------|---------|-------------|
| **Ⅰa** | 阻滞钠通道 + 延长复极 | 奎尼丁、普鲁卡因胺 | 延长 APD 和 QT |
| **Ⅰb** | 阻滞钠通道 + 缩短复极 | **利多卡因**、苯妥英钠、美西律 | 缩短 APD，不延长 QT |
| **Ⅰc** | 阻滞钠通道（强） | **普罗帕酮（心律平）**、氟卡尼 | 延长 APD，QT 不明显延长 |
| **Ⅱ类** | β 受体阻滞剂 | **美托洛尔**、普萘洛尔 | — |
| **Ⅲ类** | 钾通道阻滞剂 | **胺碘酮（可达龙）**、索他洛尔 | 明显延长 APD 和 QT |
| **Ⅳ类** | 钙通道阻滞剂 | **维拉帕米（异搏定）**、地尔硫䓬 | — |
| 其他 | — | 腺苷、洋地黄类 | — |

> **抗心律失常药物均有促心律失常作用**（致心律失常）。

---

## 窦性心律失常

| 类型 | 心率 | 特点 |
|------|------|------|
| 窦速 | > 100 次/min | 针对病因治疗，**不专门用药** |
| 窦缓 | < 60 次/min | 无症状不需治疗 |
| 窦性心律不齐 | — | 呼吸性，正常变异 |

---

## 期前收缩（早搏）

### 房性期前收缩

| 项目 | 特点 |
|------|------|
| ECG | 提前出现的 P' 波（形态与窦性不同），P'-R > 0.12s，QRS 正常 |
| 发生率 | 正常人群 **60%** 可出现 |
| 治疗 | 多不需治疗，消除诱因即可 |

### 室性期前收缩

| 项目 | 特点 |
|------|------|
| ECG | 提前出现的宽大畸形 QRS（> 0.12s），前无 P 波，**完全性代偿间歇** |
| 频发定义 | > 5 次/min 或 > 30 次/h |
| 二联律 | 1 个正常 + 1 个早搏交替 |
| 三联律 | 2 个正常 + 1 个早搏交替 |
| 常见病因 | **冠心病**最常见 |
| 治疗 | 无器质性心脏病 → 不需治疗；有器质性 → 病因治疗，必要时 β 阻滞剂 |

> **R-on-T 室早**：早搏落在前一个 T 波上 → 易诱发室速/室颤。

---

## 房颤（Atrial Fibrillation）

### 心电图与听诊

| 项目 | 特点 |
|------|------|
| ECG | 窦性 P 波消失，代之以不规整 **f 波**（V₁ 导联最清楚），频率 **350~600 次/min**；心室率极不规整 |
| 听诊 | **心律绝对不齐 + S₁ 强弱不等 + 脉搏短绌** |
| 确诊 | **ECG**（最可靠依据） |

### 分类

| 类型 | 持续时间 |
|------|----------|
| 阵发性 | ≤ 7 天，自行终止 |
| 持续性 | > 7 天 |
| 长期持续性 | ≥ 1 年 |
| 永久性 | 患者和医生接受不再转复 |

### 治疗

| 目标 | 方法 |
|------|------|
| **控制心室率** | β 阻滞剂、CCB（地尔硫䓬/维拉帕米）、洋地黄 |
| **转复窦律** | 药物：**胺碘酮**（首选）；电复律 |
| **抗凝** | CHA₂DS₂-VASc 评分 → 华法林/NOAC |
| **根治** | 射频消融 |

> **慢性房颤治疗三原则**：控制心室率 + 抗凝 + 病因治疗。
> **不宜转复**：左房内径 > 60mm、房颤 > 1 年、有附壁血栓。

---

## 心房扑动（Atrial Flutter）

| 项目 | 特点 |
|------|------|
| ECG | P 波消失，代之以锯齿样 **F 波**，频率 **250~350 次/min** |
| 心室率 | 多为 150 次/min（2:1 传导） |
| 治疗 | 电复律最有效；射频消融可根治 |

---

## 阵发性室上性心动过速（PSVT）

| 项目 | 特点 |
|------|------|
| 最常见机制 | **房室结折返**（AVNRT） |
| ECG | 心率 **150~250 次/min**，RR 规则，QRS 多正常；逆行 P 波（Ⅱ、Ⅲ、aVF 倒置） |
| 特点 | **突发突止** |
| 治疗 | ① 刺激迷走神经（Valsalva、颈动脉窦按摩）→ ② 药物：**腺苷/维拉帕米**首选 → ③ 血流动力学不稳定 → 同步电复律 → ④ **射频消融**（根治） |

> **预激合并房颤**：禁用洋地黄、利多卡因、维拉帕米（抑制正常通路 → 促进旁路传导 → 可能诱发室颤）。

---

## 预激综合征（WPW）

| 项目 | 特点 |
|------|------|
| ECG | **PR 间期缩短**（< 0.12s）+ QRS 增宽 + **δ 波**（QRS 起始部粗钝） |
| 最常见并发心律失常 | **房室折返性心动过速（AVRT）** |
| 合并房颤 | 禁用洋地黄、维拉帕米、利多卡因；可用**普罗帕酮、胺碘酮** |
| 根治 | **射频消融** |

---

## 室性心动过速（VT）

| 项目 | 特点 |
|------|------|
| ECG | 宽大畸形 QRS（> 0.12s），频率 100~250 次/min |
| 特征性表现 | **房室分离 + 心室夺获 + 室性融合波** |
| 最常见病因 | **冠心病（急性心梗）** |
| 与室上速鉴别 | 刺激迷走神经 → 室速**无反应**；室上速可终止或减慢 |
| 治疗 | 无血流障碍 → **胺碘酮/利多卡因**；有障碍 → **同步电复律** |
| TdP（尖端扭转型） | 静注**硫酸镁** |

---

## 心室颤动（VF）

| 项目 | 特点 |
|------|------|
| 最常见病因 | **冠心病（急性心梗）** |
| ECG | 无规律、形态振幅不等的颤动波 |
| 治疗 | **立即非同步电除颤** + CPR |

---

## 房室传导阻滞（AVB）

| 类型 | ECG 特点 | 口诀 |
|------|---------|------|
| **一度** | PR 间期 > 0.20s，每个 P 后均有 QRS | 每天回家比平时晚 |
| **二度Ⅰ型（文氏）** | PR 间期逐渐延长 → 直至 QRS 脱落 | 越来越晚 → 突然不回家 |
| **二度Ⅱ型** | PR 间期固定，间歇性 QRS 脱落 | 正常 → 毫无征兆不回家 |
| **三度** | P 波与 QRS 波完全无关，**PP < RR**（心房率 > 心室率） | 你过你的，我过我的 |

### 治疗

| 类型 | 处理 |
|------|------|
| 一度/二度Ⅰ型 | 一般不需治疗 |
| 二度Ⅱ型/三度 | **阿托品**（提高心率）→ 无效 → **起搏器** |
| 洋地黄中毒致 AVB | 停洋地黄，阿托品，**不宜电复律** |

---

## 束支传导阻滞

| 类型 | ECG 特点 |
|------|----------|
| **完全性右束支（RBBB）** | V₁ 呈 rSR' 型（"兔耳征"），QRS ≥ 0.12s |
| **完全性左束支（LBBB）** | V₅~V₆ R 波宽大、有切迹，QRS ≥ 0.12s |

---

## 抗心律失常药物选择

| 心律失常 | 首选药物 |
|----------|----------|
| 房颤（转复） | **胺碘酮** |
| 房颤（控制心室率） | β 阻滞剂 / CCB |
| 室上速 | **腺苷 / 维拉帕米** |
| 室速 | **胺碘酮 / 利多卡因** |
| 洋地黄中毒致心律失常 | **利多卡因 / 苯妥英钠** |
| 预激合并房颤 | 普罗帕酮 / 胺碘酮 |

---

## 电复律

| 类型 | 方式 |
|------|------|
| 房颤、房扑、室上速、室速 | **同步**电复律 |
| 室颤、无脉室速 | **非同步**电除颤 |

> **洋地黄中毒**：禁用电复律（易诱发室颤）。

---

## 起搏器

| 适应证 | 说明 |
|--------|------|
| 二度Ⅱ型/三度 AVB（有症状） | 永久起搏器 |
| 病态窦房结综合征 | 永久起搏器 |
| 急性心梗 + 三度 AVB | 临时起搏器 |

---

## 题库高频考点补充

### 心电图鉴别

| 特征 | 结论 |
|------|------|
| P 波消失 + f 波 + RR 绝对不齐 | **房颤** |
| P 波消失 + 锯齿 F 波（250~350/min） | **房扑** |
| 宽大畸形 QRS + 房室分离 | **室速** |
| PR 缩短 + δ 波 | **预激** |
| PR 逐渐延长 → 脱落 | **二度Ⅰ型 AVB** |
| PR 固定 + 间歇脱落 | **二度Ⅱ型 AVB** |
| P 与 QRS 无关 + PP < RR | **三度 AVB** |
| 提前出现宽大畸形 QRS + 完全代偿间歇 | **室早** |
| QRS 正常 + 逆行 P + 突发突止 | **室上速** |

### 晕厥/阿-斯综合征

- 易引起晕厥：**三度 AVB、室速、室颤、病窦**
- 不易引起晕厥：房早、一度 AVB、完全右束支

### 听诊线索

| 听诊 | 提示 |
|------|------|
| 心率正常而整齐 | 一度 AVB、完全右束支、窦缓 |
| 心律绝对不齐 + S₁ 强弱不等 | **房颤** |
| 大炮音 | **三度 AVB** |

### 预激综合征

- 旁路定位：PR 缩短 + QRS 正常 → **左侧旁路**（James 束）
- 合并房颤时：禁用洋地黄、维拉帕米、利多卡因

### 阿托品试验

- 注射阿托品后心率 < 90 次/min → **阳性**（提示窦房结功能不全）

### Q-T 间期延长

- 易诱发：**尖端扭转型室速（TdP）** → 治疗：硫酸镁
