# 心肌炎与心肌病

## 心肌病分类

| 类型 | 心肌病分类 |
|------|-----------|
| 我国(2007) | 扩张型、肥厚型、限制型、致心律失常性右室心肌病、未定型 |
| 美国(2006) | 遗传性（如肥厚型）、获得性（如感染性）、混合性（如扩张型、限制型） |

## 三种常见心肌病对比

| 特征 | 扩张型心肌病（DCM） | 肥厚型心肌病（HCM） | 限制型心肌病（RCM） |
|------|-------------------|-------------------|-------------------|
| **特点** | 心室腔扩大（尤其左室）、收缩功能障碍 | 心室肥厚（非对称性）、舒张期顺应性↓、心室腔变小 | 心室充盈受限、舒张期容量↓；收缩功能/室壁厚度正常 |
| **最常见病因** | 病毒感染（肠道病毒）→免疫损伤 | 遗传因素（AD最常见） | 淀粉样变性（继发性最常见） |
| **临床表现** | 活动性呼吸困难、心界向两侧扩大、晚期顽固性心衰/栓塞/猝死；无心绞痛 | 劳力性呼吸困难（最常见）、胸痛、晕厥、猝死（**运动员/青年猝死主因**） | 发热、倦怠→心悸、呼吸困难、右心衰较重、奇脉 |
| **心超特征** | 室壁弥漫性搏动↓、LVEF显著↓、**"钻石样"二尖瓣改变** | **非对称性室间隔肥厚（≥15mm）**、SAM现象（梗阻性） | 双房增大，舒张期快速充盈→突然终止 |
| **杂音特点** | — | 胸骨左缘3/4肋间粗糙递增递减型收缩期喷射性杂音 | — |
| **ECG** | 无特异性 | 可有病理性Q波 | 低电压、各类心律失常 |
| **治疗** | β-受体阻滞剂（首选）、ACEI/ARB、利尿剂；CRT；ICD | **β-受体阻滞剂首选**；慎用洋地黄/硝酸酯类（加重梗阻）；禁用多巴胺 | 对症治疗；心脏移植 |

### 肥厚型梗阻性心肌病杂音增强/减弱
- **增强**：强心剂、硝酸甘油、Valsalva动作、站立位（心肌收缩力↑或后负荷↓）
- **减弱**：β-受体阻滞剂、下蹲、握拳（心肌收缩力↓或后负荷↑）

## 病毒性心肌炎

### 病因
- 感染性：**病毒最常见（柯萨奇B组病毒）**
- 非感染性：过敏、药物、理化因素

### 临床表现
- 发病前1~3周有前驱症状（发热、乏力、肌肉酸痛）
- 随后：心悸、胸痛、呼吸困难，甚至晕厥、猝死
- 体征：各类心律失常，**与发热程度不平行的心动过速**（特征性）

### 临床分型
| 分型 | 特点 |
|------|------|
| 亚临床型 | 无自觉症状，ECG异常 |
| **轻症自限型（最常见）** | — |
| 隐匿进展型 | 一过性心肌炎，数年后→DCM |
| 急性重症型 | 胸痛、心悸、气短、奔马律、心衰 |
| 猝死型 | 活动时猝死，死前无心脏病表现 |

### 诊断与治疗
- **心内膜心肌活检(EMB)**：金标准
- **CMR**：钆延迟增强(LGE)示心肌片状强化
- 治疗：一般治疗、抗病毒（干扰素、黄芪）、心肌保护（维生素C、辅酶Q10、曲美他嗪）
