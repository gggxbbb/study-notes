# 心瓣膜病

## 二尖瓣狭窄（MS） vs 二尖瓣关闭不全（MR）

| 特征 | 二尖瓣狭窄（MS） | 二尖瓣关闭不全（MR） |
|------|----------------|-------------------|
| **最常见病因** | 风湿热（急性后至少2年） | 瓣叶损害（风湿性）、瓣环扩大、腱索断裂（越来越多见） |
| **病理生理** | MVA 4~6cm²：轻1.5~2、中1~1.5、重<1cm² | 急性：肺淤血/肺水肿；慢性：肺淤血少见 |
| **症状** | 呼吸困难（最早最常见）、咳嗽、咯血（大咯血→支气管静脉破裂） | 急性→左心衰/休克；慢性→CO↓，呼吸困难出现较晚 |
| **听诊** | **心尖区低调舒张中晚期隆隆样杂音**（左侧卧位清楚），S₁亢进+开瓣音 | **心尖部全收缩期吹风样杂音**（向左腋/肩胛下区传导），可伴震颤 |
| **杂音传导** | 局限不传导 | 前叶病变→左腋/肩胛下区；后叶→心底部 |
| **面容** | **二尖瓣面容（双颧绀红色）** | — |
| **胸片** | **梨形心**、肺淤血（Kerley B线） | 左房增大 |
| **ECG** | 二尖瓣型P波、电轴右偏 | 左房增大/肥厚 |
| **心超确诊** | M型：二尖瓣前叶**"城墙样改变"** | 彩色多普勒可确诊 |
| **并发症** | **房颤（早期常见）**、急性肺水肿、右心衰（晚期）、血栓栓塞 | 房颤、感染性心内膜炎（较MS更常见）、心衰 |
| **治疗** | PBMV（单纯二狭、有开瓣音）、直视分离术、人工瓣膜置换术 | 二尖瓣修复术、人工瓣膜置换术、MitralClip |

### 咯血机制（二尖瓣狭窄）
| 咯血类型 | 机制 |
|----------|------|
| 痰中带血 | 毛细血管扩张/通透性↑/破裂，肺静脉压升高 |
| 粉红色泡沫痰 | 毛细血管破裂（急性肺水肿） |
| 大咯血 | 左房压力突然升高→支气管静脉破裂 |
| 胶冻状暗红色痰 | 晚期并发右心衰→附壁血栓脱落→肺栓塞→肺梗死 |

## 主动脉瓣狭窄（AS） vs 主动脉瓣关闭不全（AR）

| 特征 | 主动脉瓣狭窄（AS） | 主动脉瓣关闭不全（AR） |
|------|------------------|---------------------|
| **最常见病因** | 退行性（成人最常见）、风心病（多合并二尖瓣病变） | 慢性：风心病、感染性心内膜炎、先天畸形、Marfan综合征 |
| **临床表现** | 典型**三联征：呼吸困难（劳力性首发）、心绞痛（最常见首发）、晕厥** | 慢性→可长时间无症状；急性→重者可心绞痛、左心衰 |
| **听诊** | 主动脉瓣区**递增递减型粗糙收缩期吹风样杂音**（向颈动脉传导），可伴震颤；A₂减弱/S₂逆分裂 | **主动脉瓣第二听诊区高调叹气样递减型舒张早期杂音**（向心尖传导）；S₁减弱；**Austin-Flint杂音** |
| **周围血管征** | 无 | 有：**点头征、水冲脉、血管枪击音、杜氏双重杂音、毛细血管搏动征** |
| **脉搏** | **细迟脉** | 水冲脉 |
| **病理生理** | 轻>1.5cm²；中1~1.5；重<1.0cm²；平均压力阶差>40mmHg | — |
| **治疗** | **人工瓣膜置换术**（成人主要方式）；TAVI（不能手术者）；禁用β-受体阻断剂 | 慢性→血管扩张剂（CCB、ACEI/ARB）；外科→人工瓣膜置换术 |

## Austin-Flint杂音
主动脉瓣关闭不全引起的**相对性二尖瓣狭窄**→心尖区舒张期隆隆样杂音（与器质性二狭鉴别：无S₁亢进/开瓣音）
