# 高血压

## 定义与分类
原发性高血压（高血压病）→ 占95%以上；继发性高血压（症状性高血压）。

## 血压水平分类

| 类别 | 收缩压(mmHg) | | 舒张压(mmHg) |
|------|-------------|--|-------------|
| 正常血压 | <120 | 和 | <80 |
| 正常高值 | 120~139 | 和（或） | 80~89 |
| **1级高血压（轻度）** | 140~159 | 和（或） | 90~99 |
| **2级高血压（中度）** | 160~179 | 和（或） | 100~109 |
| **3级高血压（重度）** | ≥180 | 和（或） | ≥110 |
| 单纯收缩期高血压 | ≥140 | 和 | <90 |

## 临床表现
- **起病缓慢**，一般无特殊症状。常见：头晕、头痛、颈项板紧、疲劳、心悸
- **恶性高血压**：中青年多见，舒张压持续≥130mmHg，伴头痛、视力模糊、眼底出血/渗出/乳头水肿，肾脏损害突出

## 并发症
| 系统 | 表现 |
|------|------|
| **脑（最常见死因）** | 脑血管病（脑出血、脑血栓、TIA）、高血压脑病 |
| **心脏** | 左心室肥厚（心尖抬举性搏动）、心绞痛/心梗、心力衰竭 |
| **肾脏** | 慢性肾衰、肾小动脉玻璃样变性 |
| **血管** | 主动脉夹层 |
| **眼底** | 视网膜痉挛→渗出、出血 |

## 诊断标准
诊室非同日3次测量：**收缩压≥140mmHg和（或）舒张压≥90mmHg**

## 24h动态血压（ABPM）参考值
| 时段 | 正常值 |
|------|--------|
| 24h平均 | <130/80mmHg |
| 白天均值 | <135/85mmHg |
| 夜间均值 | <120/70mmHg |
| 节律 | 双峰（6~10时&16~20时）一谷（夜间） |

## 治疗药物

| 药物类别 | 代表药物 | 适用/特点 |
|---------|---------|----------|
| **利尿剂** | 氢氯噻嗪、呋塞米、螺内酯 | 老年、盐敏感性高血压 |
| **β-受体阻滞剂** | 美托洛尔、比索洛尔 | 心率快、冠心病、心衰 |
| **ACEI** | 卡托普利、依那普利、培哚普利 | 糖尿病肾病、心衰、蛋白尿 |
| **ARB** | 氯沙坦、缬沙坦、厄贝沙坦 | 同ACEI，副作用少（无干咳） |
| **CCB（钙通道阻滞剂）** | 硝苯地平、氨氯地平（二氢吡啶类）；维拉帕米（非二氢吡啶类） | 老年、单纯收缩期高血压 |
| **α-受体阻滞剂** | 特拉唑嗪、多沙唑嗪 | 前列腺增生、难治性高血压 |

**五大类核心药物口诀：利尿、β阻、ACEI、ARB、CCB**

## 高血压危象
- **高血压急症**：血压严重升高(>180/120mmHg)伴靶器官损害（需静脉降压，1h内降25%）
- **高血压亚急症**：血压严重升高但不伴靶器官损害
