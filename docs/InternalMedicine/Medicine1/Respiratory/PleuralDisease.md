# 胸膜疾病

## 胸腔积液

### 发病机制
| 机制 | 临床病因 |
|------|---------|
| 毛细血管静水压↑ | 心衰、缩窄性心包炎→漏出液 |
| 毛细血管胶体渗透压↓ | 低蛋白血症、肝硬化、肾病综合征→漏出液 |
| 胸膜通透性↑ | 炎症（结核、肺炎）、肿瘤、肺梗死→渗出液 |
| 壁胸膜淋巴回流受阻 | 癌性淋巴管阻塞 |
| 损伤 | 主动脉瘤破裂、食管破裂→血胸/脓胸/乳糜胸 |

### 漏出液 vs 渗出液（Light标准）
| 指标 | 渗出液 | 漏出液 |
|------|--------|--------|
| 胸液/血清蛋白比值 | >0.5 | ≤0.5 |
| 胸液LDH | >正常上限2/3 | ≤正常上限2/3 |
| 胸液/血清LDH比值 | >0.6 | ≤0.6 |

符合任意1项为渗出液。

### 常见病因鉴别
| 病因 | 特点 | 关键检查 |
|------|------|---------|
| **结核性胸膜炎** | 青年多见，ADA>45IU/L，淋巴细胞为主 | PPD(+)、胸膜活检肉芽肿 |
| **恶性胸腔积液** | 中老年，血性，CEA↑，LDH>500IU/L，增长快 | 脱落细胞、胸腔镜 |
| **肺炎旁胸腔积液** | 发热咳嗽，中性粒细胞为主，pH<7.2 | 胸水培养 |
| **心衰所致** | 双侧（右>左），NT-proBNP>1500pg/ml | 心超 |

### 治疗
- **结核性**：抗结核+抽液（首次<800ml，每次<1000ml）；糖皮质激素可加用
- **恶性**：治疗原发肿瘤+胸腔置管引流/胸膜固定术
- **脓胸**：抗生素+引流

## 气胸

### 分类
| 类型 | 特点 |
|------|------|
| **闭合性（单纯性）** | 破口小已闭合，抽气后压力不升 |
| **开放性（交通性）** | 破口持续开放，胸膜腔压=大气压 |
| **张力性（高压性）** | 单向活瓣，气体只进不出，最危险 |

### 临床表现
- **突发胸痛** + **呼吸困难**
- **体征**：患侧叩诊鼓音、呼吸音减弱/消失、气管向健侧移位
- **张力性气胸**：严重呼吸循环障碍，需紧急排气

### 诊断
- **X线**：气胸线（确诊依据）
- **大量**：侧胸壁至肺边缘距离≥2cm
- **CT**：更敏感，鉴别肺大疱

### 治疗
| 情况 | 处理 |
|------|------|
| 稳定型小量（<20%） | 保守（卧床+吸氧） |
| 肺压缩20~50%闭合性 | 胸腔穿刺抽气（锁骨中线第2肋间） |
| 不稳定型/交通性/张力性 | 胸腔闭式引流 |
| 反复发作 | 胸膜固定术/手术切除肺大疱 |
