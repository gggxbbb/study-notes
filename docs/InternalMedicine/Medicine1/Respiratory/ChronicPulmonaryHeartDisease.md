# 慢性肺源性心脏病

## 定义
由慢性支气管肺疾病、胸廓疾病或肺血管疾病引起肺循环阻力增加、肺动脉高压，进而引起右心室肥厚/扩大，甚至发生右心衰竭。排除先心病/左心病变引起的心脏病。

## 病因
| 类型 | 常见疾病 |
|------|---------|
| 慢性支气管/肺疾病 | **COPD（最常见）**、支气管哮喘、支气管扩张、重症肺结核、ILD |
| 胸廓运动障碍性疾病 | 严重胸廓畸形、神经肌肉疾病 |
| 肺血管疾病 | 特发性肺动脉高压、肺小动脉炎 |
| 其他 | 睡眠呼吸暂停低通气综合征 |

## 发病机制（先决条件：肺动脉高压）

### 肺动脉高压的形成
| 因素 | 机制 |
|------|------|
| **功能性因素（最重要）** | 缺氧→肺血管收缩；高碳酸血症→肺血管痉挛 |
| **解剖性因素** | 血管炎→管壁增厚；肺气肿压迫；**肺血管重塑（最重要）**；血栓形成 |
| 血液黏稠度↑+血容量↑ | 慢性缺氧→RBC↑→血液黏滞→阻力↑ |

### 心脏病变
肺动脉高压→右心后负荷↑→右室肥厚/扩大→右心衰竭

## 临床表现（代偿期 → 失代偿期）

### 代偿期
- 胸肺基础疾病体征 + 肺动脉高压/右室扩大体征
- **P₂ > A₂**、三尖瓣收缩期杂音、剑突下搏动、颈静脉充盈

### 失代偿期
| 类型 | 症状 | 体征 |
|------|------|------|
| **呼吸衰竭** | 呼吸困难加重、头痛、失眠、神志不清（肺性脑病） | 明显发绀、球结膜水肿、视乳头水肿 |
| **右心衰竭** | 心悸、食欲降低、腹胀、恶心 | 发绀更明显、**颈静脉怒张**、肝大压痛、**肝颈静脉回流征（+）**、下肢水肿 |

## 辅助检查
| 检查 | 表现 |
|------|------|
| **胸片**（诊断主要依据） | 右下肺动脉干增宽、肺动脉段凸出、残根征、心尖圆隆/上翘、右室增大 |
| **超声心动图** | 敏感性和特异性高；可判断肺动脉管径和压力 |
| **ECG**（诊断参考） | 电轴右偏、顺钟向转位、**肺型P波**、RV₁+SV₅≥1.05mV |
| 血气分析 | 指导急性发作期治疗 |

## 治疗

### 急性加重期（失代偿期）
| 措施 | 说明 |
|------|------|
| **控制感染（首选和关键）** | 积极抗感染 |
| **氧疗** | 降低肺动脉高压的关键治疗 |
| 右心衰竭治疗 | 一般不常规使用利尿剂和强心剂；必要时小剂量低效利尿剂 |
| 强心剂使用指征 | ①感染控制+呼吸功能改善+利尿无效；②合并室上速；③右心衰无急性感染；④合并急性左心衰 |
| 强心剂原则 | 选用作用快、排泄快的强心剂（西地兰），小剂量（常用量1/2~2/3） |

### 并发症
- **肺性脑病（首要死亡原因）**
- 酸碱失衡和电解质紊乱
- 心律失常（**紊乱性房性心动过速**最具特征性）
- 休克、消化道出血、DIC

### 缓解期（代偿期）
中西医结合：增强免疫力、延缓心肺基础病变发展、去除诱发因素。
