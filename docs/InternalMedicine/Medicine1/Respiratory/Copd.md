# COPD

## 定义
一组具有气流受限不完全可逆、呈进行性发展的肺部疾病。慢支和阻塞性肺气肿是导致COPD的最常见疾病。

## 诊断公式
老年人（吸烟史）+ 咳、痰、喘 + 桶状胸 + 肺功能检查（FEV1/FVC < 70%）

## 发病机制与肺功能
- 主要效应细胞：中性粒细胞
- 属于阻塞性通气障碍，肺功能检查：带 F 的指标均↓
- **蛋白酶-抗蛋白酶失衡**：蛋白酶↑、α1-AT缺乏（欧美多见）

## 肺功能检查（诊断金标准）
| 指标 | 意义 |
|------|------|
| **FEV1/FVC < 70%**（吸入支气管扩张药后） | 确诊**不完全可逆气流受限**（必备条件） |
| **FEV1%预计值** | 评价严重程度 |
| RV/TLC > 40% | 诊断参考（肺气肿） |

## GOLD 分级（肺功能严重程度）

| 分级 | 标准（FEV1%预计值） | 推荐治疗 |
|------|---------------------|---------|
| 1级（轻度） | ≥80% | 避免危险因素、按需使用短效支气管舒张药 |
| 2级（中度） | 50%~79% | 加用长效支气管舒张药 |
| 3级（重度） | 30%~49% | 反复发作时使用激素 |
| 4级（极重度） | <30% | 长期氧疗、外科治疗 |

## ABCD 分组评估

| 分组 | 特征 | 上年急性加重次数 | GOLD | mMRC | 首选药 |
|------|------|-----------------|------|------|--------|
| A | 低风险、症状少 | ≤1次 | 1~2 | 0~1级 | SAMA/SABA |
| B | 低风险、症状多 | ≤1次 | 1~2 | ≥2级 | LAMA/LABA |
| C | 高风险、症状少 | ≥2次 | 3~4 | 0~1级 | ICS+LABA/LAMA |
| D | 高风险、症状多 | ≥2次 | 3~4 | ≥2级 | ICS+LABA/LAMA |

## 临床表现
- **标志性症状**：气促、进行性呼吸困难（劳累→休息时发作）
- 体征：桶状胸、语颤减弱、过清音、呼吸音减弱伴呼气延长、心音遥远
- 晚期：体重下降、食欲减退、营养不良

## 并发症
自发性气胸、呼吸衰竭、慢性肺源性心脏病

## 治疗

### 稳定期
| 措施 | 说明 |
|------|------|
| 教育与管理 | **最重要的戒烟**、脱离粉尘环境 |
| 支气管舒张药 | 核心药物 |
| ICS | 高风险组（C/D组）、合并哮喘患者 |
| **长期家庭氧疗（LTOT）** | 指征：PaO₂≤55mmHg或SaO₂≤88%；持续低流量1~2L/min，>15h/d |
| 康复治疗 | 缩唇呼吸 |

### 急性加重期（AECOPD）
- 确定原因：多数是感染
- 控制性氧疗：低流量给氧28%~30%
- 药物治疗：抗生素（有脓痰时使用）、支气管舒张药、糖皮质激素
- 机械通气：适用于并发较严重呼吸衰竭者
