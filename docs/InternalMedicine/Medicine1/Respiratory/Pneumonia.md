# 肺炎

## 定义
终末气道、肺泡和肺间质的感染性炎症。是否发生肺炎取决于病原体和宿主因素两个因素。

## 分类

### 按解剖/影像学

| 类型 | 特点 | 致病菌 | 胸片表现 |
|------|------|--------|----------|
| **大叶性（肺泡性）** | 肺泡先炎症→Cohn孔扩散；不累及细支气管 | 肺炎链球菌 | 肺叶/段实变影+支气管充气征 |
| **小叶性（支气管性）** | 继发于其他疾病，起始于细支气管→累及肺泡 | 多种 | 沿肺纹理分布不均斑片影，无实变 |
| **间质性** | 以肺间质为主 | 病毒/支原体 | 一侧/双侧下肺条索状/网状阴影 |

### 按病原体

| 类型 | 常见病原体 |
|------|-----------|
| 细菌性肺炎 | 肺炎链球菌、金葡菌、克雷伯杆菌、流感嗜血杆菌、铜绿假单胞菌 |
| 非典型病原体 | 军团菌、支原体、衣原体 |
| 病毒性肺炎 | 呼吸道病毒、SARS、疱疹病毒（儿童多见） |
| 真菌性肺炎 | 条件致病菌（多继发性） |

### 按发病场所（最重要分类）

| 类型 | 特点 |
|------|------|
| **社区获得性肺炎（CAP）** | 最常见；20%需住院，1~2%重症 |
| **医院获得性肺炎（HAP）** | 高发病率、高死亡率、高消费（"三高"） |
| **健康护理相关肺炎（HCAP）** | 如护理院获得性肺炎 |
| **免疫低下宿主肺炎（ICHP）** | 起病隐匿，进展急剧 |

## 诊断公式

| 类型 | 诊断公式 | 关键特征 |
|------|----------|----------|
| 大叶性肺炎 | 成人+受凉+高热+咳铁锈色痰 | 铁锈色痰 |
| 克雷伯杆菌肺炎 | 咳砖红色痰+X线空洞 | 砖红色胶冻样痰 |
| 支原体肺炎 | 儿童+刺激性干咳+关节痛+抗生素无效 | 刺激性干咳 |
| 支气管肺炎 | 婴幼儿+发热+呼吸困难（鼻翼煽动、三凹征） | 婴幼儿 |
| 金葡菌肺炎 | 高热+胸痛+脓血痰+X线片状影+易变 | 脓血痰、肺气囊肿 |

## CAP 诊断标准（四条均需满足）
1. 社区发病
2. 新出现或进展性肺部浸润性病变
3. 肺炎相关临床表现（4选1）：①发热≥38℃；②新咳/咳痰加重伴脓痰；③肺实变体征/湿啰音；④血WBC>10×10⁹/L或<4×10⁹/L
4. 除外非感染性疾病

## 病情评估：CURB-65 计分法
| 项目 | 评分标准 |
|------|---------|
| C=意识障碍 | 新出现的意识障碍 |
| U=尿毒症 | BUN>7.0 mmol/L |
| R=呼吸 | 呼吸频率>30次/min |
| B=血压 | BP<90/60 mmHg |
| 65=年龄 | ≥65岁 |

- **0~1分** → 门诊治疗
- **2分** → 住院治疗
- **≥3分** → 入住ICU

## 治疗原则
1. **经验性治疗+靶向治疗结合**：尽早（4~8h内）经验性治疗，48~72h后评估
2. 青壮年无基础病：青霉素类、多西环素、一二代头孢、大环内酯类、氟喹诺酮类
3. 老年人/有基础病：二代头孢±大环内酯类、β-内酰胺类/β-内酰胺酶抑制剂±大环内酯类
4. 重症（ICU）：A组（无铜绿危险）→头孢曲松/头孢噻肟+大环内酯类；B组（有铜绿危险）→抗铜绿β-内酰胺类+大环内酯类/氟喹诺酮类
5. 疗程：热退2~3d且呼吸道症状明显好转时停药
