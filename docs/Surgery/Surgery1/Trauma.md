# 创伤

## 伤口分类

| 类型 | 定义 | 处理 |
|------|------|------|
| 清洁伤口 | 无细菌存在及感染 | 直接缝合 |
| 污染伤口 | 有细菌但未造成感染 | 先清创，再直接缝合或延期缝合 |
| 感染伤口 | 有细菌造成感染 | 先引流，再做其他处理 |

伤口红肿有渗液 → 感染伤口

## 清创缝合时限

| 部位 | 时限 |
|------|------|
| 一般伤口 | ≤6-8h |
| 头皮 | ≤24h |
| 面部 | ≤12h |

口诀：一天洗1次头（头部24h），一天洗2次脸（面部12h），一天吃3次饭（普通8h）

处理均为清创后一期缝合。

## 损伤类型

| 类型 | 特点 |
|------|------|
| 刺伤 | 伤口边缘整齐，小而深，深部神经、血管损伤 |
| 利器伤 | 伤口边缘整齐，线状，切断的血管多无明显收缩，出血较多 |
| 挫伤 | 软组织挫伤常见，肿胀、疼痛、瘀斑，局部无伤口，压痛明显 |
| 撕裂伤 | 钝性暴力，伤口形态各异，"藕断丝连" |
| 撕脱伤 | 组织损伤广泛，血管、神经、肌腱从不同平面撕脱 |
| 坠落伤 | 软组织伤，一处或多处骨折，甚至内脏损伤 |
| 闭合伤 | 皮肤和黏膜完整无伤口 |

## 止血方法

- 指压法：骨骼表面
- 填塞法：肌肉、骨骼
- 止血带法：四肢
- 加压包扎法：小动脉和静脉

## 优先抢救顺序

心脏呼吸骤停 > 窒息 > 威胁生命的大出血 > 张力性气胸 > 颅脑损伤

## 其他

- 出现骨折和大出血先止血
- 四肢乱动 → 没有骨折；步行入院 → 没有心梗
- 创伤前期分解代谢为主，修复前期合成代谢为主
- 糖皮质激素促进蛋白质分解，不利于伤口愈合
- 精神因素是诱因：胃十二指肠溃疡；精神因素是病因：急性胃炎、肠应激综合征；应激性溃疡
- 绿走黄看红救黑埋（创伤分诊）
