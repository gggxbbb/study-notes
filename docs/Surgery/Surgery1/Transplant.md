# 移植

## 排斥反应

| 类型 | 时间 |
|------|------|
| 超急性排斥 | 数分钟至数小时（<24h），无法逆转 |
| 加速急性排斥 | 3-5天 |
| 急性排斥 | 3月内（4-90天），可以逆转 |
| 慢性排斥 | >3个月 |

- 移植疗效最好的器官：肾（慢性排斥反应是最大的障碍）
- 排斥反应发生率最高、最易并发感染：小肠

## 器官灌洗

三个水柱：中心静脉 CVP 4-12 cmH₂O，脑脊液 8-18 cmH₂O，器官灌洗液 60-100 cmH₂O

（外科 CVP 5-10 cmH₂O，生理 4-12 cmH₂O）
