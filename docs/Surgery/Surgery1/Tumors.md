# 肿瘤

## 体表肿瘤

| 肿瘤 | 特点 |
|------|------|
| 脂肪瘤 | 无黑头 |
| 皮脂囊肿 | 有黑头，头面部小黑点豆渣物 |
| 皮样囊肿 | 眉梢颅骨骨缝哑铃状 |

## 黑痣与黑色素瘤

- 皮内痣：少见恶变
- 交界痣：外伤或感染后易恶变
- 混合痣：有恶变可能
- 黑色素瘤：高度恶性，应广泛切除治疗，会出现卫星结节及转移

## 年龄特点

- 儿童：胚胎性肿瘤
- 青少年：肉瘤
- 老年人：发展速度慢

## 放射性核素诊断

阳性率：骨肿瘤 > 甲状腺肿瘤 > 脑肿瘤 > 肝肿瘤 > 大肠癌（空腔脏器最低）

## 化疗可治愈的肿瘤

"大伯培养小林上高中"：大细胞淋巴瘤、伯基特淋巴瘤、胚胎性横纹肌肉瘤、侵蚀性葡萄胎/绒癌、小细胞肺癌、急性淋巴细胞白血病、睾丸精原细胞瘤、中枢神经系统淋巴瘤

## 首选放疗的肿瘤

"圣母多灵性"：肾母细胞瘤、多发性骨髓瘤、淋巴造血系统肿瘤、性腺肿瘤

## 遗传相关

常显：肾母细胞瘤(WT)、结肠腺瘤性息肉病(APC)、遗传性非息肉性结肠癌(MSH)、视网膜母细胞瘤(RB)、神经纤维瘤病(NF)

常隐：Li-Fraumeni综合征、Bloom综合征、Fanconi贫血、着色性干皮病(XP)、毛细血管扩张性共济失调症(ATM)
