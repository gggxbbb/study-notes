# 器官功能不全与衰竭

## 多器官功能不全综合征（MODS）

**定义**：机体遭受一种/多种严重应激（严重感染、创伤、休克），两个/两个以上原来健康的器官同时/序贯发生功能不全/衰竭的临床综合征；是危重病病人的严重并发症和重要死因之一。

**病因**：绝大多数由感染引起

**发病机制**：全身炎症反应综合征（SIRS）、代偿性抗炎反应综合征（CARS）、肠屏障功能障碍

### 临床分型

| 分型 | 特点 |
|------|------|
| **一期速发型** | 原发急症发病24h内两个/以上器官同时发生功能不全 |
| **二期迟发型** | 一个重要器官先发生功能不全（多为心、肾、肺），经过稳定期后更多器官功能不全 |

**诊断**：SIRS同时伴有器官功能不全

## 急性呼吸窘迫综合征（ARDS）

**定义**：感染、创伤等危重病时并发的一种急性呼吸衰竭，以严重低氧血症、弥散性肺部浸润和肺顺应性下降为特征。

**临床表现**：
- 初期：呼吸加快、呼吸窘迫感（一般吸氧不能缓解）
- 进展期：明显呼吸困难和发绀，X线示广泛点片状阴影
- 末期：昏迷

**治疗**：一般治疗、维持循环、**呼吸治疗（PEEP）**

## 急性肾衰竭（ARF）

**定义**：各种原因引起的急性肾功能损害及所致的氮质血症、水电解质平衡紊乱。

**病因**：肾前性、肾性、肾后性

**临床表现**：少尿/无尿期 → 多尿期 → 恢复期（也有非少尿型ARF）

**治疗**：
- 少尿期：利尿剂、限制水分和电解质、营养支持、维持内环境稳定、防治感染、**透析**
- 多尿期：注意水电解质平衡

## 急性胃肠功能障碍（AGD）

**定义**：继发于创伤、烧伤、休克和其他全身性病变的胃肠道急性病理改变，以胃肠道黏膜损害以及运动和屏障功能障碍为主要特点。

**举例**：应激性溃疡

## 急性肝衰竭（AHF）

**定义**：各种原因引起的急性肝功能严重损害。

## 肿瘤概述

### 体表肿瘤

| 肿瘤 | 特点 |
|------|------|
| **脂肪瘤** | 无黑头 |
| **皮脂囊肿** | 有黑头，头面部小黑点豆渣物 |
| **皮样囊肿** | 眉梢颅骨骨缝哑铃状 |

### 黑痣与黑色素瘤

| 类型 | 恶变风险 |
|------|---------|
| **皮内痣** | 少见恶变 |
| **交界痣** | 外伤或感染后易恶变 |
| **混合痣** | 有恶变可能 |
| **黑色素瘤** | 高度恶性，广泛切除治疗，可出现卫星结节及转移 |

### 年龄特点

- 儿童：胚胎性肿瘤
- 青少年：肉瘤
- 老年人：发展速度慢

### 放射性核素诊断阳性率

**骨肿瘤 > 甲状腺肿瘤 > 脑肿瘤 > 肝肿瘤 > 大肠癌**（空腔脏器最低）

### 化疗可治愈的肿瘤

口诀：**"大伯培养小林上高中"**
大细胞淋巴瘤、伯基特淋巴瘤、胚胎性横纹肌肉瘤、侵蚀性葡萄胎/绒癌、小细胞肺癌、急性淋巴细胞白血病、睾丸精原细胞瘤、中枢神经系统淋巴瘤

### 首选放疗的肿瘤

口诀：**"圣母多灵性"**
肾母细胞瘤、多发性骨髓瘤、淋巴造血系统肿瘤、性腺肿瘤

### 遗传相关

| 遗传方式 | 肿瘤（基因） |
|---------|-------------|
| **常显** | 肾母细胞瘤(WT)、结肠腺瘤性息肉病(APC)、遗传性非息肉性结肠癌(MSH)、视网膜母细胞瘤(RB)、神经纤维瘤病(NF) |
| **常隐** | Li-Fraumeni综合征、Bloom综合征、Fanconi贫血、着色性干皮病(XP)、毛细血管扩张性共济失调症(ATM) |
