# 休克

## 微循环变化

| 分期 | 别称 | 特点 |
|------|------|------|
| 休克早期 | 微循环收缩期/代偿期/缺血缺氧期 | 少灌少流，灌少于流 |
| 休克进展期 | 微循环扩张期/失代偿期/淤血缺氧期 | 灌而少流，灌大于流 |
| 休克难治期 | 微循环衰竭期/DIC期 | 不灌不流 |

休克早期"只出不进"：进不去（外周阻力增加），出去（回心血量增加）。

大失血时维持心脑供血，先牺牲肾。

## 休克分度

| 分度 | 失血量 | 收缩压 | 神智 |
|------|--------|--------|------|
| 轻度 | <20%（<800ml） | >90 mmHg | 清楚，痛苦 |
| 中度 | 20%-40%（800-1600ml） | 70-90 mmHg | 尚清楚，表情淡漠 |
| 重度 | >40%（>1600ml） | <70 mmHg | 意识模糊，昏迷 |

## 休克监测

- 早期休克敏感指标：脉搏
- 最重要的一般监测：尿量（"肾好才是真的好"）
- 最重要的特殊监测：中心静脉压（CVP）
- 最敏感指标：休克指数（脉率/收缩压）
  - 0.5：无休克
  - >1.0-1.5：有休克
  - >2.0：严重休克

## CVP 判断

CVP 正常值 5-10 cmH₂O：

- CVP < 5：血容量不足 → 补液
- CVP > 15：心功能不全、静脉血管床过度收缩或肺循环阻力增加
- CVP > 20：充血性心力衰竭
- CVP 正常：做补液试验

CVP 高的一定是扩张血管，CVP 低的一定是补液，CVP 正常的是补液试验。

## 休克用药

| 休克类型 | 首选药物 |
|---------|---------|
| 过敏性休克 | 肾上腺素 |
| 心源性休克 | 多巴胺（心率慢时用多巴酚丁胺） |
| 心肺复苏 | 肾上腺素 |
| 顽固性休克 | 糖皮质激素（早期、大剂量、单次冲击，不超过48h） |
| 改善微循环 | 酚妥拉明 |
| 扩充血容量 | 低分子右旋糖酐 |

- 先盐后糖，先晶后胶
- 糖皮质激素连续使用不超过48小时，用量为正常的10-20倍

## 创伤性休克

先出现呼吸性碱中毒（喊救命呼出大量CO₂），后出现代谢性酸中毒（体力消耗酸性代谢产物增加，血流量不够无法带走）。

## 感染性休克

- 暖休克：看起来很正常
- 冷休克：看起来就是休克
- 易迅速感染中毒性休克的外科急腹症：急性出血坏死性胰腺炎、绞窄性肠梗阻、结直肠瘘、消化道穿孔、坏疽穿孔性阑尾炎

## 其他

- 早期休克主要体征：脉率增快
- 晚期休克主要体征：昏迷、末梢发绀、血压下降
- 早期和晚期均可出现脉压变小
- 失代偿期：收缩压 < 90 mmHg
- 最佳抽血时间：败血症/急性血源性骨髓炎→寒战高热时；感染性心内膜炎→任何时
