# 外科感染

## 常见致病菌

| 致病菌 | 特点 |
|--------|------|
| 金葡菌 | 稠黄色脓液，不臭，转移性脓肿，经典化脓症状 |
| 溶血性链球菌 | 脓液淡红稀薄，无转移性脓肿 |
| 大肠杆菌 | 胃肠道症状 |
| 绿脓杆菌 | 甜腥臭味，多见于大面积烧伤，院感 |
| 变形杆菌 | 特殊恶臭味 |
| 拟杆菌 | 恶臭发黑，普通细菌培养阴性 |
| 破伤风杆菌 | 痉挛（厌氧芽孢杆菌） |
| 产气荚膜杆菌 | 气性坏疽（梭状芽孢杆菌） |
| 草绿色链球菌 | 心内膜炎赘生物 |

治疗最关键：尽早处理原发病灶。

## 感染分类

- 急性：3周内（"人有三急"）
- 亚急性：3周-2月
- 慢性：>2月
- 特异性感染：破伤风、结核、真菌、气性坏疽、炭疽、念珠菌（"风姐真坏有贪念"）
- 条件/机会性感染：趁你病要你命，如院内感染
- 二重感染/菌群交替：长期用抗生素不敏感菌趁机繁殖

## 浅部感染

| 疾病 | 特点 |
|------|------|
| 疖 | 单个毛囊 |
| 痈 | 多个毛囊，糖尿病人易在背部并发 |
| 丹毒 | 网状淋巴管炎，一片红，不切开引流（很少组织坏死化脓） |
| 蜂窝织炎 | 肌间隙 |

- 口鼻疖、颈背痈、下肢丹毒、肌间隙蜂窝织炎
- 浅部脓肿：波动感
- 深部脓肿：深压痛 + 凹陷性水肿

## 手部感染

- 鱼刺木刺各种刺 → 脓性指头炎 → 末节指骨缺血坏死 → 骨髓炎
- 剪指甲、撕倒刺 → 甲沟炎
- 化脓性腱鞘炎 → 掌中间隙感染

## 切开引流时机

| 情况 | 处理 |
|------|------|
| 破伤风、气性坏疽 | 早期切开引流（预防发作） |
| 颌下急性蜂窝织炎 | 早期切开引流（预防窒息，小儿多见） |
| 脓性指头炎、化脓性腱鞘炎 | 肿痛明显时切开引流 |
| 丹毒 | 不切开引流 |
| 寒性/冷脓肿 | 尽量不切开引流（仅混合感染、全身中毒症状明显时） |

不需要切开：寒性脓肿、丹毒

出现波动感：疖、痈、蜂窝织炎、甲沟炎

肿痛明显：脓性指头炎、掌深间隙感染、化脓性腱鞘炎、化脓性滑囊炎

需早期切开：颌下蜂窝织炎、破伤风、气性坏疽

## 破伤风

- 预防：注射破伤风抗毒素 1500-3000 IU
- 治疗：注射破伤风抗毒素 10000-60000 IU
- 消毒：过氧化氢

## 气性坏疽

- 消毒：过氧化氢 + 1:1000 高锰酸钾
- 特点：感同身受（感、痛、深、臭）

## 抗生素

- 首选青霉素，次选甲硝唑

## 炎症细胞

- 中性粒细胞：急性炎症
- 淋巴细胞：慢性炎症
- 巨噬细胞：病变晚期

## 淋巴管炎

- 管状淋巴管炎：一条或多条红线
- 网状淋巴管炎（丹毒）：一片红
