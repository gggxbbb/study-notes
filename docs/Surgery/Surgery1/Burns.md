# 烧伤与冻伤

## 烧伤面积计算（中国九分法）

口诀：三三三五六七，十三十三二十一，屁股占五吊占一，小腿十三脚占七

| 部位 | 占比 |
|------|------|
| 头颈面 | 9%（3+3+3） |
| 双上肢 | 18%（5+6+7 = 9%×2） |
| 躯干+会阴 | 27%（13+13+1 = 9%×3） |
| 双下肢（含臀） | 46%（5+7+13+21 = 9%×5+1%） |

儿童：头面颈 = 21 - 年龄，双下肢 = 34 + 年龄

## 烧伤深度

| 分度 | 创面表现 | 愈合时间 |
|------|---------|---------|
| Ⅰ度 | 表面红斑状、干燥 | <1周 |
| 浅Ⅱ度 | 创面红润、潮湿，水疱 | 10天-2周 |
| 深Ⅱ度 | 创面微湿，红白相间 | 3-4周 |
| Ⅲ度 | 创面蜡白或焦黄，甚至炭化，硬如皮革，干燥无渗液 | 需植皮 |

浅Ⅱ度处理：有水疱→穿刺抽；水疱已撕脱→油纱；感染→清除感染分泌物→定期更换纱布

## 烧伤分期

- 伤后1-2天：体液渗出期 → 抗休克
- 伤后3-10天：急性感染期 → 抗感染

## 特殊烧伤

| 类型 | 特点 |
|------|------|
| 酸烧伤 | 一般无水疱，迅速成痂，不向深度侵犯 |
| 碱烧伤 | 向深部穿透 |
| 磷烧伤 | 接触空气自燃 |
| 氢氟酸烧伤 | 深部侵蚀，可溶解脂肪，骨质脱钙，深及骨骼 |
| 电接触烧伤 | 入口较出口重，损伤范围外小内大，没有明显坏死层面，进行性坏死 |

## 特殊溃疡

- Curling 溃疡：烧伤（r像小火苗）
- Cushing 溃疡：脑部疾病（h = head）

## 处理原则

- 呼吸道烧伤：未等呼吸困难先切开气管
- 脓性指头炎：未等形成波动先切开引流
- 成人补液系数 1.5，儿童 1.8，婴儿 2.0
