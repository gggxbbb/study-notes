# 外科2 体征汇总

## 颈部与乳腺

| 体征 | 疾病 | 说明 |
|------|------|------|
| **甲状腺肿大+震颤+血管杂音** | 原发性甲亢（Graves） | 弥漫性、对称 |
| **甲状腺结节** | 结节性甲状腺肿/腺瘤/癌 | 癌→质硬、不平、固定 |
| **Horner综合征** | 颈交感神经受压 | 瞳孔缩小+眼睑下垂+眼球内陷+无汗 |
| **声音嘶哑** | 喉返神经损伤/受压 | 甲状腺术后/晚期甲癌/主动脉瘤 |
| **饮水呛咳** | 喉上神经内侧支损伤 | 甲状腺术后 |
| **音调降低** | 喉上神经外侧支损伤 | 环甲肌麻痹 |
| **手足抽搐** | 甲状旁腺误切→低钙 | 术后1~3天 |
| **呼吸困难/三凹征** | 术后血肿/气管塌陷/双侧喉返损伤 | 术后48h内最危急 |
| **酒窝征** | 乳腺癌 | Cooper韧带受侵→皮肤内陷 |
| **橘皮样变** | 乳腺癌 | 皮下淋巴管阻塞→真皮水肿 |
| **乳头内陷** | 乳腺癌 | 乳管受侵缩短 |
| **卫星结节** | 晚期乳腺癌 | 癌细胞皮肤播散 |
| **铠甲状癌** | 晚期乳腺癌 | 累及胸肌筋膜→固定胸壁 |
| **炎性乳癌** | 红肿热厚糙（不痛） | 预后最差，不宜手术 |
| **Trousseau征（恶性）** | 胃癌/胰腺癌 | 游走性血栓性静脉炎（副癌综合征） |
| **Chvostek征** | 低钙血症 | 轻叩面神经→面部肌肉抽动 |
| **Trousseau征（低钙）** | 低钙血症 | 束臂加压→手部痉挛（助产士手） |

## 腹部 — 腹膜刺激征

| 体征 | 意义 |
|------|------|
| **腹膜刺激征**（压痛+反跳痛+肌紧张） | 腹膜炎症的标志 |
| **板状腹** | 胃十二指肠溃疡穿孔（胃酸刺激→腹肌强直） |
| **腹肌紧张程度=炎症程度** | 轻→轻度炎症/出血；明显→化脓性炎症；板状→穿孔 |
| **肝浊音界消失** | 消化道穿孔（膈下游离气体） |
| **移动性浊音** | 腹水/腹腔积血 |
| **肠鸣音亢进+气过水声** | 机械性肠梗阻早期 |
| **肠鸣音减弱/消失** | 麻痹性肠梗阻/腹膜炎/低血钾 |
| **腹胀加重** | 腹膜炎恶化标志 |
| **振水音** | 幽门梗阻 |
| **液波震颤** | 大量腹水（≥3000~4000ml） |
| **肠型+蠕动波** | 肠梗阻/幽门梗阻 |
| **舟状腹** | 消化性溃疡穿孔 |
| **Grey-Turner征** | 重症胰腺炎（腰部/季肋部青紫瘀斑） |
| **Cullen征** | 重症胰腺炎（脐周青紫瘀斑） |
| **Courvoisier征** | 胰头癌（黄疸+无痛性肿大胆囊） |
| **Murphy征(+)** | 急性胆囊炎（吸气时右上腹触痛） |

## 腹部 — 器官特异性体征

| 体征/试验 | 对应疾病 | 说明 |
|-----------|---------|------|
| **转移性右下腹痛** | 急性阑尾炎 | 脐周→右下腹（内脏→躯体神经） |
| **McBurney点固定压痛** | 急性阑尾炎 | 最有意义体征 |
| **Rovsing征**（罗氏征） | 急性阑尾炎 | 按压左下腹→右下腹痛（间接压痛） |
| **Blumberg征** | 腹膜炎症 | 反跳痛（按压后突然抬手→疼痛加剧） |
| **结肠充气试验** | 阑尾炎 | 仅提示阑尾炎症，不定位置 |
| **腰大肌试验** | 盲肠后位/腹膜后位阑尾炎 | 左侧卧位 |
| **闭孔内肌试验** | 低位阑尾炎（靠近闭孔内肌） | 仰卧位 |
| **直肠指诊触痛** | 盆腔位阑尾炎 | |
| **Boas征** | 急性胆囊炎 | 右肩胛下区感觉过敏/放射痛 |
| **Charcot三联征** | 急性胆管炎 | 腹痛+寒战高热+黄疸 |
| **Reynolds五联征** | AOSC（急性梗阻性化脓性胆管炎） | Charcot三联征+休克+神经抑制 |
| **胆道出血三联征** | 胆道出血 | 胆绞痛+黄疸+消化道出血（周期性） |
| **症征不符** | 胆道蛔虫症 / 肠系膜血管栓塞 | 症状重但体征轻 |
| **Whipple三联征** | 胰岛素瘤 | 空腹低血糖+血糖<2.8+葡萄糖缓解 |
| **Courvoisier征** | 胰头癌 | 黄疸+无痛性肿大光滑胆囊 |
| **咳嗽冲击感明显** | 腹股沟斜疝 | 位于精索前，表浅 |
| **还纳压内环试验** | 斜疝vs直疝鉴别 | 压住内环→斜疝不突出，直疝仍突出 |
| **透光试验** | 鞘膜积液vs斜疝 | 透光(+)→鞘膜积液，(-)→斜疝 |
| **腹膜后捻发音** | 十二指肠腹膜后破裂 | 横结肠系膜根部/直肠指诊骶前 |
| **腰大肌轮廓模糊** | 十二指肠破裂（腹膜后积气） | 花斑状积气 |
| **腰大肌影消失** | 腹膜后血肿 | |
| **Kehr征** | 脾破裂 | 左肩放射痛（膈肌刺激） |
| **Balance征** | 脾破裂 | 左侧卧位→左腰固定浊音（脾周血凝块） |
| **Dance征** | 肠套叠 | 右下腹空虚感（回盲部肠管蜷入升结肠） |
| **腹部搏动性肿块** | 腹主动脉瘤 | |
| **Caput medusae（海蛇头）** | 门静脉高压症 | 脐周腹壁静脉曲张 |
| **红色征** | 门脉高压食管静脉曲张 | 内镜下见，提示程度重→可预防性手术 |
| **脾大** | 门静脉高压（最早表现） | |

## 腹部 — 直肠指诊（DRE）

| 触摸特点 | 对应疾病 |
|----------|---------|
| 不规则肿物，质硬固定 | 直肠癌 |
| 剧烈疼痛（禁指检） | 肛裂 |
| 条索状物/瘘内口小硬结 | 肛瘘 |
| 一般扪不及，多柔软 | 内痔 |
| 质软可推动圆形肿块 | 直肠息肉 |
| 前壁触痛+波动感肿块 | 盆腔脓肿 |
| 骶前捻发音 | 十二指肠腹膜后破裂 |

## 血管疾病体征

### 动脉性

| 体征 | 疾病 | 说明 |
|------|------|------|
| **5P征** | 急性动脉栓塞 | Pain+Pallor+Pulselessness+Paresthesia+Paralysis |
| **间歇性跛行** | ASO / Buerger病 | 行走一段后疼痛，休息缓解 |
| **静息痛** | 严重肢体缺血 | 不活动也痛 |
| **肢体苍白/发凉** | 动脉供血不足 | |
| **Buerger试验(+)** | 动脉闭塞 | 抬高70°~80°→60s后足底苍白 |
| **动脉搏动减弱/消失** | ASO/Buerger病/动脉栓塞 | |
| **血管杂音** | ASO（有）、Buerger病（无）、大动脉炎（可有） | |
| **Adson试验** | 胸廓出口综合征 | 转头深吸气→桡动脉搏动减弱 |
| **Branham征** | 动静脉瘘 | 压迫瘘口→心率下降 |
| **无脉症** | 多发性大动脉炎（Takayasu） | 青年女性 |
| **双上肢血压差** | 主动脉夹层/大动脉炎 | |
| **雷诺三色变** | 雷诺综合征 | 苍白→青紫→潮红（寒冷/情绪诱发） |
| **ABI** | 正常≥1.0；<0.8跛行；<0.4静息痛 | |

### 静脉性

| 体征 | 疾病 | 说明 |
|------|------|------|
| **Homans征(+)** | DVT | 踝关节背屈→小腿剧痛 |
| **单侧肢体肿胀+暗红** | DVT | |
| **股青肿** | DVT最严重型 | 极度肿胀+青紫+动脉搏动消失 |
| **浅静脉曲张** | 单纯性下肢静脉曲张 | 左下肢大隐静脉最常见 |
| **Perthes试验** | 深静脉通畅度 | |
| **Pratt试验** | 交通静脉瓣膜功能 | |
| **Trendelenburg试验** | 大隐静脉瓣膜功能 | |
| **足靴区色素沉着** | 慢性静脉功能不全 | |

## 骨科 — 骨折/脱位特有体征

| 体征 | 对应损伤 |
|------|---------|
| **前脱位→外展外旋** | 髋关节前脱位（张开腿有钱） |
| **后脱位→内收内旋** | 髋关节后脱位（夹紧腿没钱） |
| **脂肪栓塞三联征** | 长骨骨折后24~72h：呼吸窘迫+神经异常+瘀斑 |
| **骨筋膜室综合征5P** | Pain+Pallor+Pulselessness+Paralysis+Paresthesia |
| **肛裂三联征** | 肛乳头肥大+肛裂+前哨痔 |
