# 外科2 最高频考点 — "最"

## 颈部与乳腺

| "最" | 内容 |
|------|------|
| 甲状腺癌**最常见** | 乳头状癌（60%） |
| 甲状腺癌**预后最好** | 乳头状癌（5年生存>90%） |
| 甲状腺癌**预后最差** | 未分化癌（存活3~6个月） |
| 甲状腺**良性肿瘤最常见** | 滤泡型腺瘤 |
| 甲亢术后**最危急**并发症 | 呼吸困难/窒息 |
| 乳腺癌**最常见**病理 | 浸润性导管癌 |
| 乳腺癌**最常见**转移 | 淋巴转移（腋窝） |
| **炎性乳癌** | 预后最差，不宜手术 |
| **Paget病** | 发展缓慢，预后好 |
