# 阑尾疾病

## 解剖要点
- 长度6~8cm，直径0.5~0.7cm
- **阑尾动脉**：回结肠动脉分支，无侧支终末动脉→易坏疽穿孔
- **阑尾静脉**→回结肠静脉→门静脉→可致门静脉炎、肝脓肿
- **神经支配**：T10~11→转移性右下腹痛

## 急性阑尾炎

### 病因
- **阑尾管腔阻塞**（最常见）：淋巴滤泡增生（60%）、肠石（35%）
- 细菌入侵（G⁻杆菌、厌氧菌）

### 病理分型
| 类型 | 特点 |
|------|------|
| 急性单纯性 | 局限于黏膜和黏膜下层 |
| 急性化脓性/蜂窝织性 | 累及全层，腔内有积脓 |
| 坏疽性/穿孔性 | 阑尾壁坏死，穿孔多在根部或近端系膜缘对侧 |
| 阑尾周围脓肿 | 大网膜/邻近肠管包裹 |

### 临床表现
- **转移性右下腹痛**（最有诊断意义）：上腹→脐周→右下腹
- **右下腹固定性压痛/反跳痛**（最有意义体征）：McBurney点
- 诊断性试验：腰大肌试验（盲肠后位）、闭孔内肌试验（盆位）、结肠充气试验

### 鉴别诊断
- 消化性溃疡穿孔（有溃疡病史、膈下气体）
- 异位妊娠破裂（停经史、宫颈举痛）
- 右侧输尿管结石（绞痛、血尿）
- 急性肠系膜淋巴结炎（上感史）

### 治疗
- **原则**：一经确诊尽早手术切除
- **手术**：麦氏切口，沿结肠带寻找阑尾
- **术后并发症**：腹腔内出血（最严重）、切口感染（最常见）、粘连性肠梗阻

## 特殊类型阑尾炎
| 类型 | 特点 |
|------|------|
| **新生儿** | 不典型，穿孔率高，死亡率高 |
| **小儿** | 早期高热呕吐，体征不明显，穿孔早 |
| **老年** | 症状不典型，体征不明显，穿孔率高 |
| **妊娠** | 压痛点抬高，穿孔后不易局限，母子危险 |

## 慢性阑尾炎
- 多由急性转变而来
- 确诊后手术切除

## 阑尾肿瘤
- **阑尾类癌**（90%）：消化道类癌最常见部位
