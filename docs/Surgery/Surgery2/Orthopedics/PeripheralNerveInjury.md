# 周围神经损伤

## 概述

### 神经损伤分类（Seddon分类）
| 类型 | 病理 | 预后 |
|------|------|------|
| **神经麻痹**（neurapraxia） | 神经纤维结构无改变 | 数日~数周自行恢复 |
| **轴索中断**（axonotmesis） | 轴索断裂，神经内膜管完整 | 多可自行恢复 |
| **神经断裂**（neurotmesis） | 神经完全断裂 | 需手术修复 |

### 病理与再生
- **沃勒变性**：远端轴索及髓鞘伤后数小时改变→2~3天分解→5~6天吞噬细胞清除
- **施万细胞增生**：伤后3天达峰，形成中空管道引导再生
- **再生速度**：约1~2mm/天
- **假性神经瘤**：近端再生纤维迂曲呈球形膨大（断端不连接时）

### 临床表现
| 功能 | 表现 |
|------|------|
| **运动功能障碍** | 弛缓性瘫痪，肌张力↓、腱反射消失、特殊畸形（垂腕、爪形手） |
| **感觉功能障碍** | 以绝对支配区为准（正中神经→示中指远节；尺神经→小指） |
| **自主神经功能障碍** | 早期→潮红、干燥无汗；晚期→苍白、皮温低 |
| **Tinel征** | 叩击神经干→远端放射痛→神经损伤/恢复的标志 |
| **神经电生理** | 肌电图+体感诱发电位 |

### 治疗原则
- **闭合性损伤**：观察3个月，无恢复→手术探查
- **开放性损伤**：一期修复（伤后6~8h）、延期修复（2~4周）、二期修复（2~4个月）
- **手术方法**：神经松解术、神经缝合术（外膜/束膜）、神经移植术（自体腓肠神经）、神经移位术、神经植入术

## 上肢神经损伤

### 臂丛神经损伤
| 类型 | 损伤 | 表现 |
|------|------|------|
| **上臂丛**（C5~C6/上干） | 牵拉 | 肩外展+屈肘障碍 |
| **下臂丛**（C8~T1/下干） | 牵拉 | 尺神经支配肌麻痹 |
| **全臂丛** | 严重牵拉 | 整个上肢弛缓性麻痹 |
| **根性撕脱伤** | — | Horner征（睑下垂+瞳孔缩小+面颈无汗） |

### 正中神经损伤
- 常见原因：儿童肱骨髁上骨折、腕部切割伤
- **腕部损伤**：拇指对掌障碍、桡侧半感觉障碍（示中指远节）
- **肘上损伤**：+拇指/示指/中指屈曲功能障碍

### 尺神经损伤
- 易损伤部位：腕部、肘部
- **爪形手**（环小指）、Froment征、小指感觉消失
- 肘上损伤：+环小指末节屈曲功能障碍
- 手内肌功能恢复差，需尽早显微修复

### 桡神经损伤
- 常见原因：肱骨中下1/3骨折
- **垂腕**畸形、伸腕/伸拇/伸指障碍、虎口区感觉异常
- 桡骨小头脱位→深支损伤→伸腕功能基本正常（桡偏）、无感觉障碍

## 下肢神经损伤

| 神经 | 损伤表现 |
|------|---------|
| **股神经** | 膝关节伸直障碍、股前+小腿内侧感觉障碍 |
| **坐骨神经** | 高位→膝不能屈+踝足功能全失+足下垂；低位→保留屈膝 |
| **胫神经** | 踝跖屈/内收/内翻障碍 |
| **腓总神经** | **足内翻下垂畸形**（踝背伸/外翻障碍），小腿前外侧+足背感觉障碍 |

## 周围神经卡压综合征

### 腕管综合征（正中神经）
- **最常见**的周围神经卡压综合征
- 中年女性多见，桡侧三指麻木/疼痛，夜间/清晨最重
- **Phalen试验**（屈腕试验）(+)；Tinel征(+)
- 治疗：早期制动+腕管注射；手术→腕横韧带切开减压

### 肘管综合征（尺神经）
- 肘外翻最常见病因→迟发性尺神经炎
- 手背尺侧+小指+环指尺侧感觉异常→爪形手→夹纸试验(+)
- 治疗：尺神经松解+移出尺神经沟

### 旋后肌综合征（桡神经深支）
- 前臂伸肌功能障碍：拇指外展/伸直障碍、掌指关节不能伸
- 无虎口区感觉异常（区别于桡神经主干损伤）
- 治疗：切开旋后肌腱弓减压

### 梨状肌综合征（坐骨神经）
- 下肢神经慢性损伤中最常见
- 坐骨神经痛（臀→大腿后→小腿→足），行走困难
- 需与腰椎间盘突出症鉴别
- 早期保守治疗，无效则手术
