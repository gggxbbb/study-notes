# 脊柱、脊髓损伤和骨盆、髋臼骨折

## 脊柱骨折
- **三柱理论**（Ferguson）：前柱（椎体前2/3+前纵韧带）、中柱（后1/3+后纵韧带）、后柱（关节突关节+韧带）
- **好发**：胸腰段（T10~L1）
- **分类**：
  | 损伤机制 | 特点 |
  |---------|------|
  | 压缩性骨折 | 仅前柱骨折；屈曲压缩（楔形变）/垂直压缩（爆裂骨折→不稳定） |
  | 屈曲-分离骨折 | 安全带损伤；Chance骨折 |
  | 旋转骨折 | 伴屈曲/压缩，常并肋骨骨折 |

### 上颈椎损伤
- **Jefferson骨折**：寰椎前后弓双骨折，不压迫脊髓，Halo架固定12w
- **寰枢椎脱位**：韧带断裂所致，可压迫脊髓，需融合术
- **齿突骨折**：II型移位>4mm需手术
- **绞刑者骨折**：枢椎椎弓垂直状骨折，不压迫脊髓

## 脊髓损伤
### 分类
| 类型 | 特点 |
|-----|------|
| 脊髓震荡 | 最轻微，数分钟~数天恢复，不留后遗症 |
| 脊髓休克 | 断面以下软瘫，1~6w恢复，球海绵体反射先恢复 |
| 不完全性损伤 | 保留部分感觉/运动，有球海绵体反射 |
| 完全性损伤 | 横贯性损害，无球海绵体反射，2~4w后→硬瘫 |

### 不完全性损伤类型
- **前脊髓综合征**：运动丧失+痛温觉消失，轻触觉深感觉存在（预后最差）
- **脊髓中央管综合征**：上肢瘫重于下肢，多见于颈椎过伸伤
- **Brown-Séquard综合征**：同侧运动/深感觉障碍+对侧痛温觉消失

### Frankel/ASIA分级（A~E级）
- A级：完全瘫痪/完全性损害
- E级：正常

## 骨盆骨折
- 高能量损伤，易致失血性休克（500~5000ml）
- 可合并膀胱尿道损伤、直肠损伤
- **腹膜后巨大血肿**：出血藏在腹膜后间隙，体表看不到——出血量可达 5000ml
- **不可引流**：腹膜后血肿不可切开引流，防止解除压迫后发生大出血
