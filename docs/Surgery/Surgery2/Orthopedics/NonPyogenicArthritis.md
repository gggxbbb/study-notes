# 非化脓性关节炎

## 骨关节炎（OA）
- **定义**：关节软骨退行性变+继发性骨质增生；年龄是主要高危因素
- **病理**：软骨变性（最早最重要）→软骨下骨象牙样硬化+骨赘形成

### 临床表现
| 症状 | 特点 |
|-----|------|
| 关节疼痛 | 主要症状，持续钝痛，休息缓解 |
| 晨僵 | <30min |
| 骨摩擦音 | 膝关节多见 |
| Heberden结节 | 远端指间关节 |
| Bouchard结节 | 近端指间关节 |

### X线表现（重要检查）
- 关节间隙变窄、软骨下骨硬化/囊性变、骨赘形成
- 晚期：间隙消失、关节内/外翻畸形

### 治疗
- **NSAIDs**：控制症状首选
- 透明质酸、氨基葡萄糖：改善病情/软骨保护剂
- 手术：保守治疗无效→游离体摘除、截骨、关节融合/成形

## 痛风性关节炎
- **急性期**：午夜起病，第一跖趾关节刀割样剧痛，红肿热痛，数天~数周自行缓解
- **慢性期**：痛风石形成，关节僵硬畸形
- **诊断金标准**：关节腔穿刺见针状尿酸盐结晶
- **治疗**：秋水仙碱、NSAIDs、非布司他、苯溴马隆

## 大骨节病（Kaschin-Beck病）
- 地方性软骨坏死性疾病
- X线：生长期骨骺过早闭合（重要征象）

- 见《内科学》；多累及骶髂关节，HLA-B27多阳性
