# 非化脓性关节炎

## 骨关节炎（OA）
- **定义**：关节软骨退行性变+继发性骨质增生；年龄是主要高危因素
- **好发**：**女性**发病率高，尤其绝经后（可能与关节软骨中雌激素受体有关）→ 通俗："胖老太太"
- **好发关节**：**非对称性**、**远端指间关节**（Heberden结节）、**膝、髋**（负重大关节）
- **病理**：软骨变性消失→软骨下骨象牙样硬化+骨赘形成
- **早期**：疼痛（主要症状）
- **急性发作期**：疼痛剧烈，先用 **NSAIDs**（如**双氯芬酸钠**）止痛
- **改善病情**：氨基葡萄糖

> **鉴别：OA vs RA — 远端 vs 近端指间关节**

| 关节炎类型 | 好发关节 | 好发人群 | 本质 |
|------|------|------|------|
| **骨关节炎（OA）** | **远端**指间关节、膝、髋 | 胖老太太 | 退行性病变 |
| **类风湿关节炎（RA）** | **近端**指间关节、掌指关节 | 20~45岁女性 | 滑膜炎 |
| **强直性脊柱炎（AS）** | 骶髂关节→脊柱→髋关节 | 青壮年男性 | 附着点炎 |
| **风湿性关节炎** | 四肢大关节（游走性） | 青少年 | — |

### 临床表现
| 症状 | 特点 |
|-----|------|
| 关节疼痛 | 主要症状，持续钝痛，休息缓解 |
| 晨僵 | <30min |
| 骨摩擦音 | 膝关节多见 |
| Heberden结节 | 远端指间关节 |
| Bouchard结节 | 近端指间关节 |

### X线表现（重要检查）
- 关节间隙变窄、软骨下骨硬化/囊性变、骨赘形成
- 晚期：间隙消失、关节内/外翻畸形

### 治疗

> 病理改变**不可逆**，治疗主要目的：缓解症状，最大限度保证和恢复生活。

| 级别 | 方法 | 内容 |
|------|------|------|
| **1. 非药物治疗（首选）** | 减少不合理运动、减轻关节负重、支具改变负重力线 | 用于早期 |
| **2. 药物治疗** | NSAIDs、氨基葡萄糖、关节腔内注射**透明质酸钠** | 用于早期 |
| **3. 手术治疗** | 关节镜下游离体摘除、关节镜清理术、**人工关节置换术** | 保守无效→手术 |
| | **人工关节置换术适应证**：严重关节骨破坏、关节炎性疼痛、关节畸形、关节功能受限 | |

> ⚠️ **糖皮质激素**：长期使用加剧关节软骨损伤、加重症状。**不主张**随意关节腔内注射，更**反对**多次反复使用。

## 痛风性关节炎
- **急性期**：午夜起病，第一跖趾关节刀割样剧痛，红肿热痛，数天~数周自行缓解
- **慢性期**：痛风石形成，关节僵硬畸形
- **诊断金标准**：关节腔穿刺见针状尿酸盐结晶
- **治疗**：秋水仙碱、NSAIDs、非布司他、苯溴马隆

## 大骨节病（Kaschin-Beck病）
- 地方性软骨坏死性疾病
- X线：生长期骨骺过早闭合（重要征象）

## 强直性脊柱炎（AS）
- **好发**：青壮年**男性**
- **本质**：下肢大关节**附着点炎**
- **起源**：骶髂关节 → 常累及脊柱、髋关节 → 少累及膝关节 → **不累及**手脚小关节
- **影像**：**竹节样改变**（脊柱融合）
- **体征**：4字试验（+），HLA-**B27** 阳性
- **止痛**：NSAIDs（如阿司匹林）
- **改善病情**：**柳氮磺吡啶**

## 类风湿关节炎（RA）
- **好发**：20~45岁**女性**，HLA-**DR4**（"4湿=死尸"）
- **本质**：**滑膜炎** → 可致 IPF（肺间质纤维化）
- **RF**：IgM 型 RF（+）
- **早期**：**对称性**、双侧小关节 → **近端**指间关节、掌指关节（腕部上下）
- **晚期**：累及髋、膝关节，出现**畸形**
- **影像**：肿→疏→窄→虫蚀→纤维强直融合
- **治疗**：改善病情 **MTX（甲氨蝶呤）**、对症抗炎 **NSAIDs**
- ⚠️ 关节畸形是**晚期表现**，非诊断/活动标准 → **类风湿结节**才是
- **特征性畸形**：纽扣花、鹅颈瓶、尺侧畸形

## 风湿性关节炎
- **好发**：青少年，病前常有**扁桃体炎**史
- **实验室**：**抗O**（ASO）阳性
- **特点**：**游走性**四肢**大关节**疼痛
- **心畸关节不畸**：心脏畸形（纤维素性渗出）→ 与 RA 相同机制

## 系统性红斑狼疮（SLE）
- **本质**：多系统病变，小血管炎
- **首选**：糖皮质激素（**泼尼松**）
- **诱导缓解期**：**环磷酰胺**
- **背景基础用药**：**羟氯喹**
