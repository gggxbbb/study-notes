# 常考畸形、特殊检查试验及体征的临床意义

## 常考畸形

| 畸形 | 临床意义 |
|------|---------|
| **爪形手** | 尺神经损伤、缺血性肌挛缩 |
| **垂腕** | 桡神经损伤 |
| **猿掌** | 正中神经损伤（拇指对掌障碍） |
| **枪刺样畸形** | Colles 骨折（正面观） |
| **银叉样畸形** | Colles 骨折（侧面观）、Barton 骨折 |
| **钩状足** | 胫神经损伤（足趾只能背屈） |
| **马蹄足** | 腓总神经损伤 |
| **方肩** | 肩关节脱位、腋神经损伤 |
| **翼状肩** | 胸长神经 + 副神经损伤 |
| **肘后三角关系改变** | 肘关节脱位 |
| **外旋畸形（30~45°）** | 股骨颈骨折 |
| **外旋畸形（90°）** | 股骨转子间骨折 |
| **增长畸形** | 髋关节前脱位 |
| **短缩畸形** | 股骨颈骨折、股骨转子间骨折、髋关节后脱位 |

## 常考检查试验

| 检查试验 | 临床意义 |
|---------|---------|
| **Tinel 征**（叩击试验） | 周围神经损伤的诊断/恢复标志 |
| **Mills 征**（伸肌牵拉试验） | 网球肘（肱骨外上髁炎） |
| **Lachman 试验** | 前交叉韧带损伤 |
| **McMurray 试验** | 内侧/外侧半月板损伤 |
| **Apley 试验** | 内侧半月板损伤 或 内侧副韧带损伤 |
| **Finkelstein 试验** | 桡骨茎突狭窄性腱鞘炎 |
| **Thomas 征** | 髋关节结核 |
| **4 字试验** | 髋关节结核 |
| **髋关节过伸试验** | 髋关节结核 |
| **Dugas 征**（杜加征） | 肩关节脱位 |
| **Froment 征** | 尺神经运动功能障碍 |
| **Hoffmann 征** | 脊髓型颈椎病 |
| **Babinski 征** | 脊髓型颈椎病 |
| **拾物试验** | 腰椎结核 |
| **浮髌试验** | 膝关节化脓性关节炎 |
| **Eaton 试验**（臂丛神经牵拉试验） | 神经根型颈椎病 |
| **Spurling 征**（压头试验） | 神经根型颈椎病 |
| **Allen 试验** | 尺、桡动脉吻合通畅性 |

> **口诀助记**
>
> - **民**（Mills 征）→ **工**（肱骨外上髁炎/网球肘）
> - **搭**（Dugas 征）→ **肩**（肩关节脱位）
> - **E 神**（Eaton 试验）→ **神经根**（神经根型颈椎病）
> - **同**（Thomas 征）→ **款**（髋关节结核）
> - **方**（Froment 征）→ **尺**（尺神经损伤）
> - **Hip 肩膀像 H** → Hoffmann 征 → 脊髓型颈椎病
> - **Allis 髋部穿 A 字裙** → Allis 征 → 髋关节
