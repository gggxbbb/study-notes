# 颈腰椎退变性疾病

## 颈椎病
**定义**：颈椎间盘退变→刺激/压迫神经根、脊髓、血管的综合征。C5~6最常见。

### 神经根型颈椎病（最常见，50~60%）
- **病因**：椎间盘侧后方突出、钩椎关节增生→压迫神经根
- **体征**：Spurling征（压头试验）（+）、Eaton试验（臂丛牵拉）（+）
- **治疗**：非手术为主（颌枕带牵引、推拿按摩）

### 脊髓型颈椎病（最严重）
- **表现**：双足踩棉花感、双手精细动作笨拙、后期大小便障碍、病理反射（+）
- **治疗**：禁用颌枕带牵引和推拿按摩；确诊后立即手术

### 交感神经型
- 中年妇女多见；兴奋症状（头痛、心慌、血压↑）或抑制症状（头晕、心动过缓）
- 无明确神经定位体征

### 椎动脉型
- 眩晕、猝倒、偏头痛；神经检查正常

## 腰椎间盘突出症（LDH）
- **好发**：L4~5、L5~S1椎间隙
- **病理类型**：膨出→凸出→突出→脱出→游离型→Schmorl结节

### 临床表现
| 神经根 | L4（L3~4椎间盘） | L5（L4~5椎间盘） | S1（L5~S1椎间盘） |
|-------|-----------------|-----------------|-----------------|
| 感觉麻木 | 小腿前内侧 | 小腿外侧+足背 | 小腿和足外侧 |
| 肌力下降 | 伸膝无力 | 踇趾背伸无力 | 足跖屈无力 |
| 反射异常 | 膝反射减弱 | 无 | 踝反射减弱 |

### 重要体征
- **直腿抬高试验（Lasègue征）（+）**：<60°即放射痛
- **加强试验（Bragard征）（+）**：背屈踝关节又放射痛
- **马尾综合征**：急骤起病，需急诊手术

### 治疗
- 非手术：初次发作、病程短者；卧床3w+腰围
- 手术：半年非手术无效、马尾综合征（急诊）、单根神经麻痹

## 腰椎管狭窄症
- **三联征**：下腰痛+神经源性间歇性跛行+症状重体征轻（直腿抬高试验阴性）
- **CT诊断**：椎管矢状径<10mm，侧隐窝<3mm

## 腰椎滑脱症
- 先天性椎弓崩裂（12~16岁）vs 退行性（中年）
- X线45°斜位片示"狗征"（椎弓崩裂）
