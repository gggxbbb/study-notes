# 骨肿瘤

## 概述
- **确诊唯一可靠检查**：病理检查（穿刺活检首选，切开活检金标准）
- **Enneking外科分期**：G（组织学分级：G0良性/G1低恶/G2高恶）+ T（范围：T0囊内/T1间室内/T2间室外）+ M（转移）

### 良恶性骨肿瘤鉴别
| 特征 | 良性 | 恶性 |
|-----|------|------|
| 发病率 | 多见 | 少见 |
| 年龄 | 成人 | 年轻人 |
| 最常见 | 骨软骨瘤 | 骨肉瘤 |
| 疼痛 | 多无 | 多有（持续性剧痛、夜间痛） |
| 生长 | 缓慢 | 迅速，表面血管怒张、皮温高 |
| 病理性骨折 | 少见 | 常有 |
| 转移 | 无 | 有远处转移 |
| X线 | 边界清楚、密度均匀、无骨膜反应 | 边界不清、虫蛀样、Codman三角/葱皮现象/日光放射征 |

## 良性骨肿瘤
| 类型 | 特点 | 治疗 |
|-----|------|------|
| 骨瘤 | 颅骨下颌骨，Gardner综合征 | 无症状不处理 |
| 骨样骨瘤 | 5~20岁，夜间痛，NSAIDs可缓解，CT最有价值 | 手术切除 |
| 内生软骨瘤 | 手部短管状骨，Ollier病易恶变 | 无症状不处理 |
| 骨软骨瘤（最常见） | 青少年，长骨干骺端，皮质相续 | 一般不需手术（7条适应证） |

## 骨巨细胞瘤（交界性肿瘤）
- **好发**：20~40岁，长骨末端（股骨下端、胫骨上端、桡骨远端）
- **X线**：膨胀性偏心性溶骨性破坏，"肥皂泡样"改变
- **治疗**：手术为主（复发率高），化疗不敏感

## 恶性骨肿瘤
### 骨肉瘤（最常见原发性恶性骨肿瘤）
- **好发**：10~20岁，四肢长骨干骺端（股骨远端、胫骨近端）
- **X线特征**：Codman三角、日光放射征、怒发冲冠征
- **化验**：ALP↑、LDH↑
- **治疗**：术前化疗→病灶切除→术后化疗；肺转移最常见

### 软骨肉瘤
- 中老年人，好发骨盆（髂骨）
- 治疗：首选手术，放化疗不敏感

### Ewing肉瘤
- 10~20岁，长骨骨干/干骺端
- X线：葱皮现象
- 基因：t(11;12)，EWS/ETS融合基因
- 治疗：放化疗常规，可手术切除

## 转移性骨肿瘤
- **好发**：躯干骨（脊椎）
- **常见原发**：乳腺癌、前列腺癌、肺癌、甲状腺癌、肾癌
- X线：溶骨性（最常见，甲状腺癌/肾癌）/成骨性（前列腺癌）
- 放射性核素骨扫描：早期筛查敏感方法

## 骨囊肿
- 儿童青少年，干骺端圆形溶骨性病灶，无骨膜反应
- 病理性骨折时"落叶征"
- 静止型→刮除植骨；活动型→保守治疗
