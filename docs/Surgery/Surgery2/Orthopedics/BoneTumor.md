# 骨肿瘤

## 概述
- **最常见**：良性→骨软骨瘤；原发性恶性→骨肉瘤；转移性恶性→乳腺癌
- **原发性骨肿瘤**：良性多见
- **确诊唯一可靠检查**：病理检查（穿刺活检首选，切开活检金标准）
- **Enneking外科分期**：G（组织学分级：G0良性/G1低恶/G2高恶）+ T（范围：T0囊内/T1间室内/T2间室外）+ M（转移）

### 良恶性骨肿瘤鉴别
| 特征 | 良性 | 恶性 |
|-----|------|------|
| 发病率 | 多见 | 少见 |
| 年龄 | 成人 | 年轻人 |
| 最常见 | 骨软骨瘤 | 转移瘤（乳腺癌）；原发性→骨肉瘤 |
| 疼痛 | 多无 | 多有（持续性剧痛、夜间痛） |
| 生长 | 缓慢 | 迅速，表面血管怒张、皮温高 |
| 病理性骨折 | 少见 | 常有 |
| 转移 | 无 | 有远处转移 |
| X线 | 边界清楚、密度均匀、无骨膜反应 | 边界不清、虫蛀样、Codman三角/葱皮现象/日光放射征 |

## 良性骨肿瘤

### 骨软骨瘤（最常见良性）
- **好发**：青少年，长骨干骺端
- **特征**：**骨性突起**、皮质与宿主骨相续
- **生长**：与生长发育有关，**骨骺闭合后停止生长**
- **治疗**：一般不需手术（7条适应证），无症状观察即可

### 骨瘤
- **好发**：颅骨、下颌骨
- **关联**：Gardner综合征
- **治疗**：无症状不处理

### 骨样骨瘤
- **好发**：5~20岁，下肢长骨
- **特征**：少有的以**疼痛为主**的良性肿瘤，**夜间痛**，NSAIDs可缓解
- **检查**：CT最有价值
- **治疗**：手术切除

### 软骨瘤
- **好发**：手足管状骨
- **类型**：内生性软骨瘤多见
- **关联**：Ollier病易恶变
- **治疗**：手术治疗

## 骨巨细胞瘤（交界性肿瘤）
- **好发**：20~40岁，长骨干骺端（股骨下端、胫骨上端、桡骨远端）
- **症状**：疼痛和肿胀
- **X线**：膨胀性偏心性溶骨性破坏，"**肥皂泡样**"改变
- **治疗**：手术为主
- ⚠️ 术后**易复发**、化疗**无效**、放疗**易肉瘤变**

## 恶性骨肿瘤
### 骨肉瘤（最常见原发性恶性骨肿瘤）
- **好发**：10~20岁，四肢长骨干骺端（股骨远端、胫骨近端）
- **X线特征**：**Codman三角**、**日光放射征**、怒发冲冠征
- **化验**：ALP↑、LDH↑
- **治疗**：术前化疗→病灶切除→术后化疗
- **肺转移**：最常见，术前常规**X线**排查

### 软骨肉瘤
- **好发**：成人和老年人，骨盆（髂骨）
- **治疗**：手术为主，放化疗不敏感

### Ewing肉瘤
- **好发**：儿童，长骨骨干
- **X线**：**洋葱皮样**（葱皮现象）
- **基因**：t(11;12)，EWS/ETS融合基因
- **治疗**：手术 + 放疗 + 化疗

### 恶性淋巴瘤
- **X线**：**溶冰征**
- **治疗**：放疗 + 化疗

### 骨髓瘤
- **好发**：脊柱
- 源于骨髓浆细胞，多发性
- **实验室**：本周蛋白（+）
- **治疗**：放疗 + 化疗

### 脊索瘤
- **好发**：脊椎、颅底
- **治疗**：手术为主，可辅助放疗

## 转移性骨肿瘤
- **性质**：绝对晚期，姑息治疗
- **好发**：躯干骨（脊椎）
- **最常见原发**：**乳腺癌**
- **常见原发**：乳腺癌、前列腺癌、肺癌、甲状腺癌、肾癌
- **X线**：溶骨性（最常见，甲状腺癌/肾癌）→ **血Ca²⁺↑**；成骨性（前列腺癌）→ **ALP↑**
- ⚠️ 前列腺癌骨转移：成骨性 → ALP↑ + **酸性磷酸酶↑**
- 放射性核素骨扫描：早期筛查敏感方法

## 骨囊肿
- **好发**：青少年，长管状骨干骺端，尤其**肱骨**
- **X线**：干骺端圆形、边界清楚的溶骨性改变，**骨皮质变薄**
- 病理性骨折时"**落叶征**"
- 静止型→刮除植骨；活动型→保守治疗
