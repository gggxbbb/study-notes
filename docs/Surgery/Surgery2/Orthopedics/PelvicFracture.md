## 骨盆解剖

- **骨盆环** = 髋骨 + 骶尾骨
- **髋骨** = 髂骨 + 耻骨 + 坐骨

## 骨盆骨折

- 高能量损伤，易致失血性休克（500~5000ml）
- 可合并膀胱尿道损伤、直肠损伤
- 脊髓止于 L1 椎体下缘，骨盆骨折**不会损伤脊髓**
- **腹膜后巨大血肿**：骨盆骨折**最严重的并发症**。出血藏在腹膜后间隙，体表看不到——出血量可达 5000ml
- **不可引流**：腹膜后血肿不可切开引流，防止解除压迫后发生大出血

### 闭合性骨折出血机制对比

- **股骨骨折**（闭合性）：周围筋膜压力高，压迫止血 → **不易导致休克**；反而容易把脂肪挤入破裂血管 → 脂肪栓塞综合征
- **骨盆骨折**（闭合性）：形成腹膜后血肿，空间大、压迫弱；若为大动静脉断裂 → 病人**迅速死亡**

### 合并神经损伤

- **坐骨骨折** → 坐骨神经损伤
- **耻骨骨折** → 股神经 + 闭孔神经损伤

### 合并尿道损伤

- **前尿道**（球部）：常因骑跨伤导致
- **后尿道**（膜部）：常因骨盆骨折导致

### 体征

1. **骨盆分离试验与挤压试验阳性**
2. **肢体长度不对称**
3. **会阴部瘀斑**：耻骨和坐骨骨折的特有体征

## 髋臼骨折

- **髋关节后脱位**：复位最佳时间 **24h** 内
- **髋臼骨折手术**：最佳手术时间 **4～7 天**
