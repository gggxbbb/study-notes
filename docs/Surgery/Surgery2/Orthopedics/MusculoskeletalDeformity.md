# 运动系统畸形

## 先天性肌性斜颈（CMT）
- **病因**：一侧胸锁乳突肌挛缩/变性
- **临床表现**：出生后1w出现硬块→头偏向患侧、下颌转向健侧→半年后纤维条索
- **治疗**：1岁以内非手术（热敷、按摩、手法矫正）；1岁以上手术（胸锁乳突肌切断术，最佳年龄1~4岁）

## 先天性髋关节脱位（CDH/DDH）
### 分类
- 单纯型：I型（发育不良）/II型（半脱位）/III型（脱位，最常见）
- 畸形型：双侧脱位+膝伸直僵直+足外旋

### 诊断
| 年龄段 | 体征/info |
|-------|-----------|
| 站立前期 | 皮褶不对称、会阴增宽、患肢缩短、Ortolani弹进试验（+）、Barlow弹出试验（+）、Allis征（+） |
| 脱位期 | 单侧跛行、双侧鸭步、Trendelenburg征（+） |

**影像学**：超声（筛查）；X线（>3个月后）

### 治疗
- 0~6个月：黄金时期，无需手术
- 1~3岁：手法/切开复位
- 3岁以上：Salter截骨（<6岁, 髋臼指数<45°）/ Pemberton截骨（>6岁, 髋臼指数>46°）

## 先天性脊柱侧凸
- **青少年特发性脊柱侧凸（AIS）为主**
- **严重表现**：剃刀背（razor back）
- **Cobb角测量**：头尾端椎上下缘垂线夹角
- **Risser征**：髂嵴骨骺骨化程度

## 其它畸形
| 类型 | 要点 |
|-----|------|
| 马蹄内翻足（CTEV） | 内收+跖屈+内翻+胫骨内旋；1岁内扳正，10岁软组织手术 |
| 平足症 | 足弓扁平/消失，柔韧型 vs 僵硬型 |
| 踇外翻 | 踇趾外斜+第一跖骨内收，跖骨间角>15°为异常 |
