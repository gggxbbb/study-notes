# 运动系统慢性损伤

## 概述
- **分类**：软组织/骨/软骨/周围神经卡压
- **基本特点**：慢性疼痛、特定压痛点、无炎症表现、过度活动史
- **治疗原则**：限制致伤活动（首要）+理疗+激素/NSAIDs

## 狭窄性腱鞘炎

> **弹指一响剑出鞘** → 弹响指/弹响拇 = 腱鞘炎
> **结婚后洗衣劳累** → 戒指在**中指环指**，频繁活动 → 弹响指好发中指环指

- **最常见**：手指频繁活动所致无菌性炎症
- **弹响指/扳机指**：屈肌腱鞘炎，好发中指环指，掌横纹处痛性结节
- **特征**：疼痛 + 活动受限 + 可触及结节
- **桡骨茎突狭窄性腱鞘炎**：Finkelstein试验（握拳尺偏试验）（+）
- **治疗**：局部制动+腱鞘内注射醋酸泼尼松龙（糖皮质激素封闭为**首选**）；手术仅用于多次封闭无效的顽固病例

## 肱骨外上髁炎（网球肘）
- **病理**：伸肌总腱起点处慢性损伤
- **Mills征（+）**：伸肘握拳屈腕→前臂旋前→肘外侧痛
- **注意**：限制腕关节活动（非肘关节！）

> **网球肘 vs 肩周炎治疗对比**：网球肘 → 限制腕关节活动 | 肩周炎 → 主动活动肩关节

## 肩关节周围炎（冻结肩/五十肩）
- **特点**：中老年多见，40%可在6~24个月内自愈
- **病因**：长期慢性劳损
- **X 线**：平片无明显异常
- **表现**：各方向活动受限（外展外旋、内旋后伸）；压痛点以肩袖间隙区为主
- **鉴别**：vs 肩袖损伤（被动活动正常、痛弧、落臂征）、vs 神经根型颈椎病
- **治疗**：主动活动肩关节（每天进行，不引起剧痛为限）；**急性期适当活动，而非制动**

## 腰腿痛
- 疼痛性质：局部痛（固定压痛点、封闭效果好）/牵涉痛（模糊）/放射痛（神经根体征）
- 常见病因：创伤、炎症、退行性变、先天畸形、肿瘤

## 周围神经卡压综合征
| 综合征 | 受压神经 | 关键体征 |
|-------|---------|---------|
| 腕管综合征 | 正中神经（最常见） | 桡侧3指麻木、Phalen征（+）、Tinel征（+） |
| 肘管综合征 | 尺神经 | 肘外翻（最常见病因）、Froment征（+）、屈肘试验（+） |
| 胸廓出口综合征 | 臂丛+锁骨下血管 | Adson试验（+）；神经+血管受压 |
| 旋后肌综合征 | 桡神经深支 | 拇指外展伸直障碍，无虎口区感觉异常 |
| 梨状肌综合征 | 坐骨神经 | 4字试验+拮抗加重 |

## 骨软骨慢性损伤

> 慢性损伤手术一般**不以根治为目的**。

- **髌骨软骨软化症**：青年运动员，髌骨下疼痛（活动开始→缓解→过久后又痛）
  - 手术目的：①改变髌股关节力线、增加髌骨稳定性 ②刮除软骨上小侵蚀病灶促进修复 ③软骨完全破坏→切除髌骨减轻疼痛
- **胫骨结节骨软骨病（Osgood-Schlatter病）**：9~14 岁男孩，剧烈运动史（初中生踢球后胫骨结节痛→典型）
  - 青少年好动 → 骨骺增大、破碎
  - 治疗：先保守；**不打封闭**（激素抑制骨骺）
- **股骨头骨软骨病（Legg-Calve-Perthes病/扁平髋）**：4~10 岁男孩，该时期骨骺血供最差，轻微外伤即可影响软骨。Thomas征（+）

#### 股骨头骨骺缺血分期（Perthes病）

| 分期 | 特征 |
|------|------|
| 缺血期 | 骨骺血供中断 |
| 血供重建期 | 新生血管长入 |
| 愈合期 | 骨组织修复 |
| 畸形残存期 | 遗留扁平髋畸形 |

#### 股骨头坏死分期

| 分期 | 特征 |
|------|------|
| 软骨下溶解期 | **新月征** |
| 股骨头修复期 | 囊性改变 |
| 股骨头塌陷期 | 股骨头**塌陷** |
| 股骨头脱位期 | 关节间隙**变窄** |
