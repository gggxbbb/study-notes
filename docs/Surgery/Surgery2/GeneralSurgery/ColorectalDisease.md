# 结、直肠与肛管疾病

## 解剖要点
- **齿状线**：黏膜（自主神经、无痛）↔皮肤（阴部内神经、有痛）
- **肛管直肠环**：损伤→肛门失禁
- **直肠指检（DRE）**：直肠癌阳性率70%

## 结直肠癌（大肠癌）

### 病理
- 最好发于直肠（壶腹部）
- 大体：溃疡型（最常见）、肿块型（右半结肠）、浸润型（左半结肠）
- 转移：淋巴转移最主要，肝转移最常见

### Dukes分期
| 分期 | 范围 |
|------|------|
| A期 | 局限于肠壁内，无淋巴结转移 |
| B期 | 穿透肠壁，无淋巴结转移 |
| C期 | 肠壁全层+淋巴结转移 |
| D期 | 远处转移/不能切除 |

### 右半 vs 左半结肠癌 vs 直肠癌
| 特征 | 右半结肠癌 | 左半结肠癌 | 直肠癌 |
|------|------------|------------|--------|
| 病理 | 肿块型多见 | 溃疡/浸润型 | 溃疡型 |
| 全身症状 | 重（贫血） | 轻 | 少见 |
| 肠梗阻 | 无 | 多有 | 可有 |
| 便血 | 多无 | 多有 | 多有 |

### 治疗
- **右半结肠切除术**：回肠-横结肠吻合
- **左半结肠切除术**：结肠-结肠/直肠吻合
- **直肠癌**：Miles术（距齿状线<5cm）、Dixon术（>5cm，最常用）、Hartmann术
- 化疗：FOLFOX方案（奥沙利铂+CF+5-FU）

## 其他疾病

### 痔
| 类型 | 特点 |
|------|------|
| 内痔 | 齿状线上，无痛性间歇性便后出血，好发3、7、11点 |
| 外痔 | 齿状线下，血栓性外痔有剧痛 |
| 混合痔 | 内外痔同时存在 |

- 治疗原则：无症状不需治，以非手术为主

### 肛裂
- 周期性疼痛（便时→间歇→便后）
- **肛裂三联征**：肛乳头肥大+肛裂+前哨痔

### 肛瘘
- Goodsall规律：外口在横线后方→弯瘘、内口在后正中
- 挂线疗法：不引起肛门失禁

### 直肠肛管周围脓肿
- 肛门周围脓肿（最常见）→肛瘘
- 治疗：脓肿切开引流

### 直肠脱垂
- 完全脱垂：黏膜同心圆状；不完全脱垂：放射状
