# 腹外疝

## 定义
疝(hernia)：体内脏器/组织离开正常解剖部位，通过薄弱点/缺损进入另一部位。腹外疝是腹腔内脏器经腹壁薄弱点向体表突出。

## 病因
- **腹壁强度降低**：先天发育不全、手术切口愈合不良
- **腹内压力增高**：慢性咳嗽、便秘、婴儿啼哭

## 病理组成
| 结构 | 说明 |
|------|------|
| 疝囊 | 壁腹膜（疝囊颈=疝门，疝囊体） |
| 疝内容物 | 小肠最多见，大网膜次之 |
| 疝外被盖 | 皮肤、皮下组织 |

## 临床分型
| 类型 | 特点 |
|------|------|
| **易复性疝** | 内容物易回纳入腹腔 |
| **难复性疝** | 不能完全回纳；滑动性疝（内容物为疝囊壁一部分） |
| **嵌顿性疝** | 疝囊颈弹性回缩卡住内容物；逆行性嵌顿（Maydl疝）、肠管壁疝（Richter疝）、Littre疝（Meckel憩室） |
| **绞窄性疝** | 嵌顿+血运障碍→坏死；儿童疝环组织柔软，很少绞窄 |

## 腹股沟疝

### 解剖要点
- **腹股沟管**：长4~5cm；深环（腹股沟韧带中点上方2cm）、浅环（耻骨结节外上方）
- **Hesselbach三角（直疝三角）**：腹壁下动脉+腹直肌外侧缘+腹股沟韧带内侧半

### 斜疝 vs 直疝 vs 股疝
| 特征 | 斜疝 | 直疝 | 股疝 |
|------|------|------|------|
| 发病年龄 | 儿童、青壮年 | 老年人 | 40岁以上妇女 |
| 突出途径 | 经腹股沟管 | 经海氏三角 | 经股管 |
| 进入阴囊 | 可 | 很少 | 很少 |
| 嵌顿机会 | 较多 | 极少 | **最易** |
| 压住内环后 | 不再突出 | 仍可突出 | — |

### 治疗
- **非手术**：1岁以下婴儿、年老体弱者（疝带）
- **传统修补术**：Ferguson（加强前壁）、Bassini/Halsted/Shouldice/McVay（加强后壁）
- **无张力修补**：Lichtenstein、Rutkow、Stoppa术
- **腹腔镜**：TAPA、TEA、IPOM
- **嵌顿/绞窄性**：紧急手术，判断肠管活力；绞窄性只结扎不修补

## 股疝
- 疝囊经股环→股管→卵圆窝突出
- 好发于40岁以上妇女，最易嵌顿和绞窄
- 治疗：确诊后及时手术（McVay术）

## 其他腹外疝
- **切口疝**：切口感染最常见病因；多无完整疝囊
- **脐疝**：小儿2岁前非手术，5岁以上手术；成人易嵌顿
- **白线疝**：上腹部多见
