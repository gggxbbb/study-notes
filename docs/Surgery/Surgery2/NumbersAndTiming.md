## 术后并发症时间

### 胃大部切除术后

| 并发症 | 发生时间 |
|--------|---------|
| 十二指肠残端破裂（最严重） | 术后1~2天 |
| 倾倒综合征（早期/晚期） | 30min / 2~4h（朝三暮四） |
| 碱性反流性胃炎 | 术后1~2年 |
| 吻合口溃疡 | 术后2~3年 |
| 残胃癌 | 术后5年以上（多20~25年） |
| 术后出血（术中止血不彻底） | 1天内 |
| 术后出血（吻合口愈合不良） | 1周内 |
| 术后出血（感染） | 近1个月 |

### 甲状腺术后

| 并发症 | 发生时间 |
|--------|---------|
| 甲状腺危象 | 术后12~36h |
| 呼吸困难/窒息（最危急） | 术后48h内 |
| 手足抽搐（误切甲状旁腺） | 术后1~3天 |
| 气管塌陷 | 术后1~3h |
| 切口内出血→血肿 | 术后24h内 |
| 双侧喉返神经损伤 | 手术台上/手术间 |

### 脾切除术后

| 并发症 | 发生时间 |
|--------|---------|
| 腹腔内大出血 | 术后24~48h |
| 膈下感染 | 术后1~2周 |

## 疾病时间节点

### 阑尾炎

| 节点 | 数值 |
|------|------|
| 三天法则 | 3天内切；>3天等3个月 |
| 阑尾周围脓肿→择期手术 | 3个月后 |

### 胰腺炎

| 节点 | 数值 |
|------|------|
| 血淀粉酶开始升高 | 2~12h |
| 血淀粉酶高峰 | 24h |
| 血淀粉酶持续时间 | 3~5d |
| 血脂肪酶开始升高 | 24~72h |
| 血脂肪酶持续时间 | 7~10d |
| 胰腺脓肿形成 | 2~3周后 |
| 假性囊肿形成 | 3~4周后 |
| 假性囊肿囊壁成熟 | 6~8周（≈1个月） |
| 化学性腹膜炎→化脓性腹膜炎 | 6~8h |
| 重症胰腺炎休克早期（低血容量） | ≤6~8h |
| 重症胰腺炎休克后期（感染性） | >6~8h |

### 脾破裂

| 节点 | 数值 |
|------|------|

| 预防延迟性破裂→绝对卧床 | >1周 |
