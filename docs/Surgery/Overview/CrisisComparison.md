# 危象鉴别

## 三种危象对比

| 危象 | 病因 | 表现特点 | 治疗首选 |
|------|------|---------|---------|
| 💙 **甲状腺危象** | 甲状腺激素过量释放（术前准备不充分） | **上吐下泻，高热大汗，谵妄昏迷**（老贺十二字） | 抗甲状腺药+碘剂+普萘洛尔 |
| 💚 **垂体危象** | 垂体功能减退 | 低血糖、高热或低热 | **葡萄糖** |
| 💚 **肾上腺危象** | 肾上腺皮质功能减退 | 低血压、休克、恶心呕吐 | **盐溶液** |

## 记忆要点

- **甲状腺危象**：激素太多 → 代谢亢进 → 高热脉速
- **垂体危象**：缺糖 → 补糖（葡萄糖）
- **肾上腺危象**：缺盐 → 补盐（盐溶液）
