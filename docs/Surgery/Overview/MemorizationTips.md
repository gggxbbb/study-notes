# 外科记忆口诀汇总

- **波动感 = 脓肿 = 切开引流**：外科通用诊断套路（如脓性指头炎、肛周脓肿、乳房脓肿等）
- **周期性发作**（常考汇总）：①乳腺囊性增生病（经前期加重→月过后缓解）②Pel-Ebstein热→霍奇金淋巴瘤 ③脓性指头炎（周期性痛）④肛裂（便时痛→间歇期→括约肌痉挛痛）⑤胃/十二指肠溃疡 ⑥周期性麻痹（低钾型）⑦Buerger病 ⑧胆道出血（1~2周一次）

## 消化道X线征速记
| 疾病 | X线征 | 形状记忆 |
|------|-------|---------|
| 食管静脉曲张 | 虫蚀→蚯蚓→串珠→菊花（由轻到重） | 静脉增粗扭曲 |
| 贲门失弛缓症 | **鸟嘴征**、漏斗征 | 食管下端狭窄如鸟嘴 |
| 肠结核 | **跳跃征**（激惹征） | 钡剂快速通过病变段 |
| 克罗恩病 | **线样征** | 肠管狭窄如线 |
| 溃疡性结肠炎 | **铅管征** | 结肠袋消失、肠管僵硬如铅管 |
| 肠套叠 | **弹簧征**、**杯口征** | 钡剂进入套叠间隙/受阻于头部 |
| 乙状结肠扭转 | **鸟嘴征** | 钡剂在扭转处截断 |
| 胰头癌 | **反3征** | 十二指肠曲扩大呈反"3"形 |
