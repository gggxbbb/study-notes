# 内分泌及内脏系统药物

## 肾上腺皮质激素类药物

### 糖皮质激素（GC）

**作用**：抗炎、抗免疫、抗休克、抗内毒素→"四抗"；代谢作用（升血糖、动员脂肪/蛋白质）

| 药物 | 特点 |
|------|------|
| **氢化可的松** | 短效，静滴→快速控制重症 |
| **泼尼松/泼尼松龙** | 中效，口服→慢性炎症维持治疗 |
| **地塞米松** | 长效，最强效。术后恶心呕吐预防 |
| **布地奈德** | 局部吸入→哮喘（减少全身副作用） |

麻醉注意：**围术期应激剂量**——长期用GC者手术时应补充"应激剂量"（氢化可的松100mg IV）防肾上腺危象

> 口诀：氢（氢化可的松）→静滴快速，泼（泼尼松）→口服慢治，布（布地奈德）→局部安全

## 甲状腺激素及抗甲状腺药

- **左甲状腺素(T₄)**：甲减替代治疗
- 抗甲状腺药：**甲巯咪唑**（首选）、丙硫氧嘧啶（PTU→孕早期可选→不易过胎盘）
- 甲亢危象：PTU（抑制T₄→T₃）+ 碘剂 + β-R阻断药 + GC

## 胰岛素及口服降血糖药

| 类型 | 代表药 | 特点 |
|------|--------|------|
| **胰岛素** | 短效/中效/长效 | 1型糖尿病、2型晚期、围术期血糖控制 |
| **双胍类** | **二甲双胍** | 2型糖尿病**一线药**。不引起低血糖、不增体重。乳酸酸中毒→肾衰/造影剂禁用 |
| 磺酰脲类 | 格列本脲 | 促进胰岛素分泌→低血糖风险 |
| DPP-4抑制剂 | 西格列汀 | 新型口服药 |
| SGLT-2抑制剂 | 达格列净 | 促进尿糖排泄 |

## 抗骨质疏松药

- 双膦酸盐类（阿仑膦酸钠）→抑制骨吸收。口服→刺激食管→空腹大量水服，服后保持直立30min
- 麻醉注意：长期卧床+双膦酸盐→颌骨坏死风险；骨质疏松患者→体位操作小心骨折
