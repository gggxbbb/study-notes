# 常见医学英语

## 一、常见前缀

| 前缀 | 含义 | 例词 |
|------|------|------|
| a-/an- | 无、缺乏 | apnea（呼吸暂停），anemia（贫血），anuria（无尿） |
| anti- | 抗、对抗 | antibiotic（抗生素），antihypertensive（抗高血压药），anticoagulant（抗凝药） |
| brady- | 缓慢 | bradycardia（心动过缓），bradypnea（呼吸过缓） |
| dys- | 异常、困难 | dyspnea（呼吸困难），dysphagia（吞咽困难），dysuria（排尿困难） |
| endo- | 内 | endocardium（心内膜），endoscopy（内镜检查），endocrine（内分泌） |
| epi- | 上、在…之上 | epidermis（表皮），epigastric（上腹部的），epicardium（心外膜） |
| hemi- | 半 | hemiplegia（偏瘫），hemisphere（半球） |
| hyper- | 高、过多、亢进 | hypertension（高血压），hyperglycemia（高血糖），hyperkalemia（高钾血症） |
| hypo- | 低、过少、减退 | hypotension（低血压），hypoglycemia（低血糖），hypokalemia（低钾血症） |
| inter- | 之间 | intercostal（肋间的），interstitial（间质的） |
| intra- | 内、内部 | intravenous（静脉内的），intramuscular（肌肉内的），intrathecal（鞘内的） |
| macro- | 大 | macrophage（巨噬细胞），macroscopic（肉眼可见的） |
| mal- | 不良、异常 | malnutrition（营养不良），malignancy（恶性肿瘤） |
| micro- | 小 | microorganism（微生物），microscopic（显微镜下的） |
| neo- | 新 | neoplasm（新生物、肿瘤），neonate（新生儿），neonatal（新生儿的） |
| peri- | 周围、围绕 | pericardium（心包），perinatal（围产期的），perioperative（围术期的） |
| poly- | 多 | polyuria（多尿），polydipsia（烦渴、多饮），polyphagia（多食） |
| post- | 后 | postoperative（术后），postpartum（产后） |
| pre- | 前 | preoperative（术前），prenatal（产前） |
| pseudo- | 假 | pseudomembrane（假膜），pseudocyst（假性囊肿） |
| sub- | 下、亚、次 | subcutaneous（皮下的），subacute（亚急性），subclinical（亚临床的） |
| supra- | 上、超 | supraglottic（声门上的），supraventricular（室上性的） |
| tachy- | 快速 | tachycardia（心动过速），tachypnea（呼吸过速），tachyarrhythmia（快速性心律失常） |
| trans- | 经、跨、横过 | transdermal（经皮的），transfusion（输血），transplant（移植） |

## 二、常见后缀

| 后缀 | 含义 | 例词 |
|------|------|------|
| -algia | 疼痛 | neuralgia（神经痛），myalgia（肌痛），arthralgia（关节痛） |
| -ectomy | 切除（术） | appendectomy（阑尾切除术），cholecystectomy（胆囊切除术），gastrectomy（胃切除术） |
| -emia | 血症 | anemia（贫血），leukemia（白血病），septicemia（败血症） |
| -genic | 产生、引起 | cardiogenic（心源性），iatrogenic（医源性），pathogenic（致病的） |
| -gram | 图、记录 | electrocardiogram（心电图），angiogram（血管造影图），mammogram（乳腺X线片） |
| -graphy | 造影、描记法 | angiography（血管造影术），radiography（X线摄影术） |
| -ism | 状态、主义 | embolism（栓塞），hyperthyroidism（甲状腺功能亢进），alcoholism（酒精中毒） |
| -itis | 炎症 | hepatitis（肝炎），myocarditis（心肌炎），pancreatitis（胰腺炎），peritonitis（腹膜炎） |
| -lysis | 溶解、分解 | hemolysis（溶血），thrombolysis（溶栓），lysis of adhesion（粘连松解） |
| -megaly | 肿大 | hepatomegaly（肝肿大），splenomegaly（脾肿大），cardiomegaly（心脏肥大） |
| -oma | 肿瘤 | carcinoma（癌），sarcoma（肉瘤），lymphoma（淋巴瘤），melanoma（黑色素瘤） |
| -opathy | 疾病、病变 | neuropathy（神经病变），nephropathy（肾病），encephalopathy（脑病） |
| -osis | 状态、病态 | necrosis（坏死），cirrhosis（硬化），stenosis（狭窄），thrombosis（血栓形成） |
| -ostomy | 造口术 | colostomy（结肠造口术），tracheostomy（气管切开术），gastrostomy（胃造口术） |
| -otomy | 切开术 | laparotomy（剖腹术），craniotomy（开颅术），tracheotomy（气管切开术） |
| -pathy | 疾病、痛苦 | neuropathy（神经病变），myopathy（肌病），psychopathy（精神病变） |
| -penia | 减少、缺乏 | leukopenia（白细胞减少），thrombocytopenia（血小板减少），osteopenia（骨量减少） |
| -phagia | 吞咽 | dysphagia（吞咽困难），polyphagia（多食） |
| -phasia | 言语 | aphasia（失语症），dysphasia（言语困难） |
| -philia | 嗜、亲和 | hemophilia（血友病），eosinophilia（嗜酸性粒细胞增多） |
| -phobia | 恐惧 | hydrophobia（恐水症），claustrophobia（幽闭恐惧症） |
| -plasty | 成形术 | angioplasty（血管成形术），rhinoplasty（鼻成形术） |
| -plegia | 瘫痪 | paraplegia（截瘫），quadriplegia（四肢瘫），hemiplegia（偏瘫） |
| -pnea | 呼吸 | apnea（呼吸暂停），dyspnea（呼吸困难），tachypnea（呼吸急促） |
| -rrhage/-rrhagia | 出血 | hemorrhage（出血），menorrhagia（月经过多） |
| -rrhea | 流、溢 | diarrhea（腹泻），menorrhea（月经），rhinorrhea（鼻漏、流鼻涕） |
| -sclerosis | 硬化 | atherosclerosis（动脉粥样硬化），multiple sclerosis（多发性硬化） |
| -scope | 镜 | endoscope（内镜），bronchoscope（支气管镜），laparoscope（腹腔镜） |
| -scopy | 镜检查 | endoscopy（内镜检查），bronchoscopy（支气管镜检查），laparoscopy（腹腔镜检查） |
| -stasis | 停滞、静止 | homeostasis（稳态），hemostasis（止血），metastasis（转移） |
| -therapy | 治疗 | chemotherapy（化疗），radiotherapy（放疗），physiotherapy（理疗） |
| -tomy | 切开、切断 | anatomy（解剖），appendectomy（阑尾切除术）通常用 -ectomy |
| -trophy | 营养、生长 | atrophy（萎缩），hypertrophy（肥大），dystrophy（营养不良） |
| -uria | 尿 | hematuria（血尿），proteinuria（蛋白尿），polyuria（多尿），anuria（无尿），oliguria（少尿） |
