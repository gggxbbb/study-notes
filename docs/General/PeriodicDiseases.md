# 周期性发作疾病汇总

1. **乳腺囊性增生病** — 乳房周期性胀痛、肿块，经前期加重→月经过后缓解
2. **霍奇金淋巴瘤** — Pel-Ebstein热（周期性发热）
3. **脓性指头炎** — 周期性痛
4. **肛裂** — 便时痛→间歇期→括约肌痉挛痛（疼痛周期性）
5. **胃/十二指肠溃疡** — 周期性发作
6. **周期性麻痹** — 低钾型最常见，与钾盐代谢障碍有关
7. **血栓闭塞性脉管炎（Buerger病）** — 肢体缺血症状周期性发作
8. **胆道出血** — 周期性出血，每隔1~2周发作一次，多反复发作
