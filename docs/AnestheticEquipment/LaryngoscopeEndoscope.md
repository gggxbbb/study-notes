# 麻醉喉镜和光导纤维内镜

## 第一节 人工气道

**定义**：麻醉机或通气机呼吸气路与患者解剖气道之间最后一级管道连接的统称。

**作用**：通气支持、呼吸治疗，是麻醉学和现代急救医学的基本技术之一。

**结构**：
- **患者端**：人工气道前部，与上呼吸道解剖形态相适应，密闭连接
- **气路端**：人工气道后部，与呼吸气路连接，接口统一国际标准（已有完整国际标准系列）

**按侵入解剖位置分类**：
- 面罩、鼻罩 — 不侵入上呼吸道
- 通气道 — 进入上呼吸道、非气管内安置
- 气管内导管 — 安置于气管内
- 支气管内导管 — 安置于支气管内

### 一、面罩和鼻罩

| 类型 | 用途 |
|------|------|
| 解剖型面罩 | 现场急救通气 |
| 一次性面罩 | 日常麻醉使用 |
| 插管面罩 | 可在正压通气下进行纤维内镜操作；有内镜操作口 |
| 吸氧面罩 | 自主呼吸给氧；含射流空混器、呼气孔、接氧气接口 |
| 鼻罩 | 无创通气支持（慢性呼衰、睡眠治疗）；不遮罩口部，清醒患者易于接受 |

**面罩基本结构**：挂钩、接口、主体、气垫。

### 二、通气道

**定义**：进入上呼吸道、非气管内安置的人工气道。

| 类型 | 特点 |
|------|------|
| 口咽通气道 | 解除舌后坠，用于自主呼吸存在、咽反射不活跃的患者；型号根据气道长度划分 |
| 鼻咽通气道 | 刺激小、易固定、可长时间保留 |
| 喉罩（LMA） | 安置于喉咽腔，环型气垫封闭食管和喉咽腔，经喉腔通气 |
| 喉管 | 经口腔插到食管入口，盲探插入，双套囊（口咽套囊+食管套囊）封闭口咽腔和食管，通气口正对喉咽腔 |
| 食管气管联合导管 | 双腔双套囊（同轴两个通气腔和两个套囊），无论进入食管还是气管均可通气。**使用步骤**：口腔盲探插入→双套囊充气→前端进入食管则经外管腔通气，前端进入气管则经内管腔通气。适用于现场急救。

#### 喉罩分类
- **普通喉罩**：硅胶，可反复使用50次以上；隔栅状设计，30°夹角设计
- **插管喉罩**：短而粗，可经导气管完成盲探气管内插管
- **双管喉罩**：有独立的胃液引流通道，更好保护气道安全性

**喉罩密封位置**：近端密封于舌根下，两侧密封于梨状窝，远端密封于食管上括约肌。

### 三、气管内导管

**定义**：通过一定解剖途径安置于患者气管内的人工气道的统称。

**分类**：经口气管导管、经鼻气管导管、口鼻通用气管导管。

**构成**：导气管 + 防漏套囊（cuff）+ 导管接头

**导气管**：材料为医用橡胶或无毒塑料；要求有一定弹性和硬度，保持弧度，不易被曲折或压扁；斜口左向（右鼻腔导管右向），侧孔（Murphy导管）— 侧孔位于斜口对壁，作用是防止斜口贴气管壁引起的气道梗阻。

**防漏套囊**：充气密闭导管与气管间隙，防止正压通气时漏气和误吸（阻止上呼吸道分泌物或胃返流物进入气管）。小儿气管导管可不带套囊。

**导管接头**：气管导管气路端的连接物，材料为金属或塑料；功能为保证通气管道的通畅，实现不同规格气管导管与呼吸气路的通用连接。

**特殊气管导管**：
- 加强气管导管（spiral embedded tube）— 管壁含钢丝/尼龙螺旋骨架，可任意弯曲不折瘪，特别适用于头颈部手术的麻醉
- 导向气管导管 — 壁内盲端隧道穿尼龙线，拉线可使导管头内曲上翘，适用于鼻腔盲探气管插管和口腔插管困难的情况
- 低损伤气管导管
- 呼吸道激光手术用气管导管 — 添加金属合成材料，防激光点燃

**气管切开导管**：经气管切开造口安置，适用于长期通气管理。

**分类**：
- 有内管/无内管：有内管适用于长期或永久性留置导管，清洗方便
- 有套囊/无套囊
- 有侧孔/无侧孔：有侧孔适用于部分喉切除患者的发声重建
- 有套囊且无侧孔：适用于通气管理

**基本结构**：固定带、接头、套囊、充气管。

### 四、支气管内导管

**定义**：安置于左或右总支气管，实施肺隔离和单肺通气的人工气道的统称。

**用途**：肺隔离和单肺通气

**适应症**：单肺隔离、支气管胸膜瘘、大肺囊肿或肺大泡、气管支气管破裂、单肺灌洗、各类肺部及食管手术

**分类**：
- **单腔支气管导管**：细长，套囊短（左/右型）；右型有右肺侧口和上叶侧口
- **支气管堵塞导管**：管壁内有一隧道，内置活动堵塞引流管，通气阻力小于双腔支气管导管
- **双腔支气管导管**：
  - 卡伦斯（Carlens）双腔管 — 有隆突钩
  - 怀特（White）双腔管 — 有右肺上叶侧口
  - 罗伯特肖（Robertshaw）双腔管 — 左型/右型，目前最常用

## 第二节 麻醉喉镜

**定义**：显示喉和声门，以便在明视下完成气管内插管的器械。

**组成**：喉镜片 + 镜柄

### 特殊喉镜

| 类型 | 角度 | 适应症 |
|------|------|--------|
| Alberts喉镜 | 67° | 颈部强直性过伸 |
| Polio喉镜 | 130°钝角 | 颌胸粘连致颈部过曲 |
| McCoy喉镜 | 前端活动镜片 | 困难气道 |

### 间接喉镜
- Bullard光导喉镜
- Wuscope光导喉镜
- 可视喉镜（视频喉镜）
- 光棒

## 第三节 光导纤维支气管镜

**组成**：冷光源 + 纤维内镜

**主要性能指标**：
- 视野角
- 视野方向
- 景深
- 内景光缆外径
- 有效长度
- 弯曲度
- 吸引介入管道内径
