# 诊断思维

## 诊断疾病的步骤

### 1. 收集资料
- **病史**（问诊）→ 症状
- **体格检查** → 体征
- **辅助检查**（实验室/影像/病理）

### 2. 分析综合 → 提出初步诊断
- 归纳突出临床表现
- 确定主要问题
- 鉴别诊断：找出支持/反对证据
- 列出可能性最大的诊断

### 3. 验证诊断
- 完善检查（针对性）
- 治疗试验/诊断性治疗
- 病情随访观察

## 临床思维方法

### 基本原则
- **整体观**：局部 ≠ 全身，不遗漏系统性疾病
- **一元论**：尽量用一个诊断解释全部临床表现
- **优先考虑常见病**：少见病仅在充分排除常见病后考虑
- **优先考虑器质性疾病**：排除器质性后考虑功能性疾病
- **动态思维**：诊断随病情变化而修正

### 常见思维误区
- 先入为主、依赖直觉
- 忽视阴性症状/体征
- 过度依赖检查结果而轻视临床
- 确认偏倚（只找支持自己假设的证据）

## 临床诊断内容
- **病因诊断**：如肺炎链球菌肺炎
- **病理解剖诊断**：如左心室肥厚
- **病理生理诊断**：如心功能Ⅲ级
- **并发症**：如上消化道出血
- **伴发病**：如2型糖尿病

### 诊断书写要求
- 主次分明（主要诊断排第一）
- 病名规范
- 尽量完整（病因+病理+功能）
