# 临床血液学检测

## 血液一般检测

血常规、全血细胞计数 (CBC)

### 红细胞

| 健康人群 | 红细胞计数/($\times10^{12}/L$) | 血红蛋白浓度 ($g/L$) |
| :--: | :-----------------------: | :------------: |
| 成年男性 |          4.3~5.8          |    130~175     |
| 成年女性 |          3.8~5.1          |    115~150     |
| 初生儿  |          6.0~7.0          |    180~190     |
|      |                           |                |

#### 数量异常

1. 增多
	1. 相对增对：血容减少
	2. 绝对增多：红细胞增多症
		1. 继发性
			1. 红细胞生成素代偿性增加：血氧饱和度降低
			2. 红细胞生成素非代偿性增加：肿瘤，肾脏疾病
		2. 真红细胞增多症：骨髓增殖性肿瘤，红细胞持续性显著增多，白细胞血小板增多，总血容增加
2. 减少
	1. 生理性
		1. 婴幼儿、<15 岁儿童，较正常成人低 10%~20%
		2. 老人，妊娠中、晚均减少
	2. 病理性：贫血
