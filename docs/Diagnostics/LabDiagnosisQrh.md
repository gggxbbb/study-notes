# 实验诊断 QRH（精简版）

## 临床血液学

### 红细胞、血红蛋白
| 人群 | RBC (×10¹²/L) | Hb (g/L) |
|:---:|:---:|:---:|
| 成年男性 | 4.3~5.8 | 130~175 |
| 成年女性 | 3.8~5.1 | 115~150 |
| 初生儿 | 6.0~7.0 | 180~190 |

**↑**：相对（血容↓）/ 绝对（PV、继发）  **↓**：贫血
**贫血分类**：MCV/MCH/MCHC → 大细胞性/正细胞性/小细胞低色素性/单纯小细胞性
**形态异常**：球形（遗传性球形红细胞增多症>25%）、靶形（地贫）、镰形（HbS）、缗钱状（多发性骨髓瘤）
**血沉**：男<15mm/h，女<20mm/h ↑→炎症/肿瘤/贫血

### 白细胞
| 人群 | WBC (×10⁹/L) |
|:---:|:---:|
| 成人 | 3.5~9.5 |
| 新生儿 | 15~20 |

| 分类 | % | 绝对值 (×10⁹/L) |
|:---:|:---:|:---:|
| 中性粒 N | 50~70 | 2~7 |
| 嗜酸粒 E | 0.5~5 | 0.05~0.5 |
| 嗜碱粒 B | 0~1 | 0~0.1 |
| 淋巴 L | 20~40 | 0.8~4 |
| 单核 M | 3~8 | 0.12~0.8 |

**N↑**：急性感染（化脓菌）、应激、肿瘤  **N↓**：病毒/伤寒/SLE/再障/化疗
**L↑**：病毒/百日咳/TB/淋巴瘤  **L↓**：免疫缺陷/激素
**E↑**：过敏/寄生虫/皮肤病/嗜酸粒细胞白血病
**核左移**→感染严重；**核右移**→巨幼贫
**类白血病反应**：WBC显著↑+幼稚细胞，但非白血病

### 网织红细胞
成人：0.5%~1.5%（绝对值 24~84×10⁹/L）
**↑**：溶血性贫血、急性失血  **↓**：再障、骨髓病性贫血

### 血小板
| 分级 | PLT (×10⁹/L) |
|:---:|:---:|
| 正常 | 100~300 |
| 减少 | <100 |
| 重度减少 | <20（自发出血风险） |
| 增多 | >400 |

**↓**：再障/ITP/白血病/DIC/脾亢  **↑**：反应性/原发性血小板增多症

### 贫血形态分类
| 类型 | MCV(fl) | MCH(pg) | MCHC(g/L) | 常见病 |
|:---|:---:|:---:|:---:|:---|
| 大细胞性 | >100 | >34 | 正常 | 巨幼贫 |
| 正细胞性 | 80~100 | 27~34 | 正常 | 再障/溶血/急性失血 |
| 小细胞低色素 | <80 | <27 | <320 | 缺铁贫/地贫 |
| 单纯小细胞 | <80 | <27 | 正常 | 慢性病贫血 |

## 血栓与止血检测

| 项目 | 参考值 | ↑/↓意义 |
|:---|:---|:---|
| BT（出血时间） | 1~3min | ↑→血管壁/血小板异常 |
| PT（凝血酶原时间） | 11~13s | ↑→外源途径缺陷/肝病/VitK缺乏 |
| APTT | 25~35s | ↑→内源途径缺陷/血友病 |
| TT（凝血酶时间） | 16~18s | ↑→纤溶亢进/肝素治疗 |
| FIB（纤维蛋白原） | 2~4g/L | ↑→急性期；↓→DIC/肝病 |
| D-二聚体 | <0.5mg/L | ↑→DIC/血栓/纤溶亢进 |
| 3P试验 | 阴性 | 阳性→DIC（FDP增多） |

**DIC诊断**：PLT↓+PT↑/APTT↑+FIB↓+D-二聚体↑

## 尿液检测
| 项目 | 正常 | 异常意义 |
|:---|:---|:---|
| 尿量 | 1000~2000mL/d | <400→少尿；>2500→多尿 |
| 颜色 | 淡黄~深黄 | 红→血尿/Hb；深茶→黄疸；乳白→乳糜尿 |
| 比重 | 1.015~1.025 | 固定→肾功不全 |
| pH | 4.5~8.0 | 酸性→代谢/呼吸性酸中毒 |
| 蛋白 | 阴性 | 阳性→肾炎/肾病/感染 |
| 糖 | 阴性 | 阳性→DM/肾性糖尿 |
| 酮体 | 阴性 | 阳性→DKA/饥饿 |
| 红细胞 | 0~3/HP | >3→血尿（结石/感染/肿瘤/肾炎） |
| 白细胞 | 0~5/HP | ↑→泌尿系感染 |
| 管型 | 少见 | 透明→肾实质损伤；颗粒→肾小管病；蜡样→慢性肾病 |
