# 辅助检查

## 心电图

### 正常心电图
| 指标 | 正常值 |
|:---|:---|
| P波 | ≤0.11s，≤0.25mV，Ⅰ/Ⅱ/aVF直立，aVR倒置 |
| PR间期 | 0.12~0.20s |
| QRS波 | 0.06~0.10s，胸导R波递增 |
| ST段 | 等电位线，下移≤0.05mV，抬高≤0.1~0.3mV |
| T波 | 方向与QRS一致，≥1/10R |
| QT间期 | <0.44s（心率60时） |
| 心电轴 | -30°~+90° |

### 房室肥大
- **左房大**：P波增宽≥0.12s，双峰（二尖瓣P波）
- **右房大**：P波高尖≥0.25mV（肺性P波）
- **左室大**：RV5+SV1>4.0mV（男）/3.5mV（女）+ ST-T改变
- **右室大**：V1 R/S>1，电轴右偏>+90°

### 心肌缺血/梗死
- **心肌缺血**：ST段下移>0.05mV，T波低平/倒置
- **心梗演变**：
  - 超急性期：T波高耸
  - 急性期（小时~天）：ST弓背抬高→T波倒置+病理性Q波
  - 陈旧期（数周~永久）：Q波持续
- **定位**：前壁（V1~V4）、下壁（Ⅱ/Ⅲ/aVF）、侧壁（Ⅰ/aVL/V5/V6）

### 常见心律失常

| 类型 | 心电图特征 |
|:---|:---|
| 窦性心动过速 | P规律，>100次/分 |
| 房性早搏 | 提早P'波（形态与窦P不同），代偿间歇不完全 |
| 室性早搏 | 宽大畸形QRS≥0.12s，无提前P波，完全代偿间歇 |
| 房颤 | P波消失→f波（350~600次/分），R-R绝对不等 |
| 室上速 | 规则快速QRS（150~250次/分），P波常不可见 |
| 室速 | QRS宽大畸形≥0.12s，规则/略不规则 |
| Ⅰ°AVB | PR>0.20s |
| Ⅱ°AVB I型 | PR逐次延长→QRS脱落 |
| Ⅱ°AVB II型 | PR固定，间歇QRS脱落 |
| Ⅲ°AVB | P与QRS完全无关，房率>室率 |

## 肺功能检查

### 通气功能
- **VC（肺活量）**：男~3500mL，女~2500mL
- **FEV1/FVC**：正常>80%，<70%→阻塞性通气障碍
- **MVV（最大通气量）**: 反映通气储备

### 换气功能
- **DLCO（弥散量）** ↓：肺气肿、间质肺病
- **通气/血流(V/Q)比值**：0.8左右，失调→低氧血症

## 血气分析

| 指标 | 正常值 | 临床意义 |
|:---|:---|:---|
| pH | 7.35~7.45 | <7.35→酸中毒；>7.45→碱中毒 |
| PaO₂ | 80~100 mmHg | <60→呼衰 |
| PaCO₂ | 35~45 mmHg | >50→Ⅱ型呼衰 |
| HCO₃⁻ | 22~27 mmol/L | 反映代谢因素 |
| BE | -3~+3 mmol/L | 代谢性酸碱失衡 |
| SaO₂ | 95~100% | 下降提示缺氧 |

### 酸碱失衡类型
- **呼酸**：pH↓ + PaCO₂↑（通气不足）
- **呼碱**：pH↑ + PaCO₂↓（过度通气）
- **代酸**：pH↓ + HCO₃⁻↓（乳酸↑/酮体↑）
- **代碱**：pH↑ + HCO₃⁻↑（呕吐/利尿剂）
