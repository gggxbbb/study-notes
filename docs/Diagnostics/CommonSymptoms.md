# 常见症状

## 发热

### 正常体温与生理变异

正常体温 36~37℃，因测量方式不同略有差异：

| 测量方式 | 正常范围 |
|---------|---------|
| 口测法 | 36.3~37.2℃ |
| 腋测法 | 36~37℃ |
| 肛测法 | 36.5~37.7℃ |

口诀：三两口（口测法）、六七夜（腋测法）、武器抗（肛测法）

- 下午体温稍高，剧烈运动、劳动、进餐后体温可升高，一般波动范围小于 1℃
- 24小时波动在 1℃ 以内
- 妇女月经前及妊娠期体温略高于正常
- 老人代谢率偏低，体温相对低于青壮年
- 体温低于正常值：见于休克、严重营养不良、甲状腺功能减退、低血糖昏迷等

### 发生机制

致热源包括外源性和内源性两类。

**外源性致热源**：各种微生物病原体及其产物（细菌、病毒、真菌及细菌素等）、炎性渗出物及无菌性坏死组织、抗原抗体复合物、某些类固醇物质（肾上腺皮质激素代谢产物原胆烷醇酮）、多糖体成分及多核苷酸、淋巴细胞激活因子等。多为大分子物质，细菌内毒素无法通过血脑屏障，通过激活血液中中性粒细胞、嗜酸性粒细胞和单核巨噬细胞系统，使其产生并释放内源性致热物质。

**内源性致热源**（白细胞致热源）：IL-1、TNF 和干扰素等。一方面通过血-脑屏障直接作用于体温调节中枢的体温调定点（温阈），使调定点上升；另一方面通过交感神经使皮肤血管及竖毛肌收缩，停止排汗，散热减少。使产热大于散热，体温升高引起发热。

### 热型

| 热型 | 特点 | 常见疾病 |
|------|------|---------|
| 稽留热 | 波动小，持续在 39~40℃ | 大叶性肺炎、伤寒高热期、斑疹伤寒 |
| 弛张热 | 波动大，>2℃，最低仍高于正常 | 败血症、风湿热、重症肺结核、化脓性炎 |
| 间歇热 | 骤升骤降，高热与无热交替 | 疟疾、肾盂肾炎 |
| 波状热 | 缓升渐降 | 布氏杆菌病 |
| 回归热 | 高热数日后降至正常，数日后又升高 | 回归热、霍奇金病 |
| 不规则热 | 无规律 | 结核病（非重症）、支气管肺炎、风湿热、渗出性胸膜炎 |

口诀：
- 稽留热：稽大爷（大叶性肺炎）上（伤寒）班（斑疹伤寒）
- 弛张热：张飞拜（败血症）师（风湿热）结（重症肺结核）盟（化脓性炎）
- 间歇热：急（疟疾）于（肾盂肾炎）歇息
- 回归热：回家挥霍（霍奇金病）
- 不规则热：截（结核病）肢（支气管肺炎）封（风湿热）神（渗出性胸膜炎）

### 伴随症状

- 寒战：大叶性肺炎、败血症、疟疾、钩端螺旋体、流行性脑脊髓膜炎、急性溶血、输血反应、急性胆囊炎、药物热
- 结膜充血：麻疹、斑疹伤寒、流行性出血热、钩端螺旋体
- 单纯疱疹：大叶性肺炎、疟疾、流脑
- 先发热后昏迷：流行性乙型脑炎、斑疹伤寒、流脑、中毒性菌痢、中暑
- 先昏迷后发热：脑出血、巴比妥类药物中毒
- 发热 + 疱疹：感染性疾病
- 低热、盗汗、乏力：结核

## 咳嗽与咳痰

### 痰液特征

| 痰液 | 提示疾病 |
|------|---------|
| 铁锈色痰 | 大叶性肺炎（肺炎链球菌） |
| 砖红色胶冻样痰 | 克雷伯杆菌肺炎 |
| 金黄色脓痰 | 金黄色葡萄球菌 |
| 黄绿色痰 | 绿脓杆菌（铜绿假单胞菌） |
| 粉红色泡沫痰 | 肺水肿（左心衰） |
| 痰白黏稠拉丝 | 真菌感染 |
| 浆液痰含粉皮样物 | 棘球蚴病（包虫病） |
| 恶臭痰 | 厌氧菌感染 |
| 大量脓痰 | 支气管扩张、肺脓肿 |

### 咳嗽特征

- 声音嘶哑：声带炎症、肿瘤压迫喉返神经
- 鸡鸣样咳嗽：百日咳、会厌/喉部疾病、气管受压
- 金属音咳嗽：纵隔肿瘤、主动脉瘤、支气管癌压迫气管
- 低微无力：严重肺气肿、声带麻痹、极度衰弱者
- 咳嗽伴胸痛、咯血和大量脓性痰：肺脓肿
- 咳嗽伴杵状指、咯血和大量脓性痰：支气管扩张或肺脓肿

## 胸痛

| 疾病 | 疼痛特点 |
|------|---------|
| 带状疱疹 | 刀割样/灼烧样剧痛 |
| 食管炎 | 烧灼痛 |
| 肋间神经痛 | 阵发性灼痛或刺痛 |
| 心绞痛 | 绞榨样痛并有重压窒息感，向左肩左前臂放射 |
| 心肌梗死 | 疼痛更剧烈，恐惧/濒死感，休息或含服硝酸酯类药后不易缓解 |
| 气胸 | 撕裂样疼痛 |
| 胸膜炎 | 隐痛、钝痛、刺痛（多在胸侧部） |
| 夹层动脉瘤 | 突然发生胸背部撕裂样剧痛或锥痛，向下放射至下腹、腰部、腹股沟和下肢 |
| 肺栓塞 | 突然发生胸部剧痛/绞痛，常伴呼吸困难 |
| 急性心包炎 | 隐痛、钝痛、刺痛 |
| 纵隔疾病/食管炎 | 胸骨后痛 |

放射痛：
- 心绞痛/心梗 → 胸骨后或心前区，可向左肩、左臂内侧、无名指与小指放射，也可放射至左颈或面颊部（误认牙痛）
- 夹层动脉瘤 → 胸背部，向下放射至下腹、腰部与两侧腹股沟和下肢

## 腹痛

### 疼痛特点

| 类型 | 特点 |
|------|------|
| 空腔脏器 | 绞痛 |
| 穿孔 | 烧灼样痛 |
| 内脏 | 钝痛 |
| 实质器官 | 胀痛 |

### 常见疾病

| 疾病 | 特点 |
|------|------|
| 胃肠穿孔 | 突发中上腹剧烈刀割样痛/烧灼样痛 |
| 胆道蛔虫症 | 阵发性剑突下钻顶样痛 |
| 消化性溃疡 | 上腹痛伴黑便 |
| 幽门梗阻 | 腹痛伴呕吐、隔夜宿食 |
| 肾绞痛 | 腹痛沿患侧腹直肌边缘向患侧腹部和腹股沟放射 |
| 急性阑尾炎 | 早期脐周/上腹痛→进展期转移至右下腹麦氏点→晚期压痛、腹肌紧张、反跳痛 |
| 急性胆囊炎 | 进食油腻食物史 |
| 急性胰腺炎 | 酗酒或暴饮暴食史 |

### 体位与疼痛

- 反流性食管炎：躯体前屈时烧灼痛明显，直立位减轻
- 胃黏膜脱垂：左侧卧位可使疼痛减轻
- 十二指肠瘀滞症：膝胸卧位或俯卧位可使腹痛及呕吐缓解
- 胰腺体部癌：仰卧位疼痛明显，前倾位或俯卧位减轻
- 胃溃疡：餐后痛，进食后加剧
- 十二指肠溃疡：餐前痛/饥饿痛，进食后缓解

### 病史线索

- 进食油腻食物史：胆囊炎、胆石症
- 酗酒或暴饮暴食史：急性胰腺炎
- 腹部手术史/炎症史：粘连性机械性肠梗阻
- 腹部受暴力作用引起剧痛并有休克：肝、脾破裂
- 心房纤颤史：肠系膜血管缺血性疾病

## 心悸

### 病因

**心脏搏动增强**：
- 生理性：剧烈运动、精神紧张、饮酒、浓茶/咖啡、阿托品/肾上腺素/麻黄碱/咖啡因、妊娠
- 病理性：心室肥大（高血压性心脏病、主动脉瓣关闭不全、二尖瓣关闭不全等）

### 伴随症状

| 伴随症状 | 提示疾病 |
|---------|---------|
| 心前区疼痛 | 冠心病（心绞痛、心梗）、心肌炎、心包炎、心脏神经官能症 |
| 发热 | 急性传染病、风湿热、心肌炎、心包炎、感染性心内膜炎 |
| 晕厥或抽搐 | 窦性停搏、高度房室传导阻滞、室性心动过速、病窦 |
| 贫血 | 急性失血（虚汗、脉弱、血压下降或休克） |

- 青年人不规则心悸常见生理性
- 心悸伴出汗：甲亢
- 心悸可是低血糖

## 呼吸困难

| 类型 | 病变部位 |
|------|---------|
| 吸气性呼吸困难 | 气管、大支气管、喉 |
| 呼气性呼吸困难 | 肺组织弹性减弱、小支气管 |
| 混合性呼吸困难 | 重症肺结核、肺不张、胸腔积液、胸膜增厚、气胸 |

- 深大呼吸：糖尿病酮症酸中毒
- 潮式呼吸（Cheyne-Stokes）：浅慢→深快→浅慢
- 比奥呼吸（Biot）：临终危象
- 抑制性呼吸：痛得不敢吸气（呼气是被动）

## 呕血与便血

- 上消化道出血：柏油样黑便（食管、胃、十二指肠），出血量 50~75ml
- 下消化道出血：鲜血便（空肠、回肠、结肠）
- 隐血试验阳性：出血量 5~10ml
  - 间断性阳性：40%-70% 提示消化性溃疡
  - 持续性阳性：95% 提示消化道恶性肿瘤

## 意识障碍

| 程度 | 特征 |
|------|------|
| 嗜睡 | 可唤醒，能正确回答 |
| 昏睡 | 强刺激可唤醒，答非所问 |
| 轻度昏迷 | 呼之不应，眼球反射存在 |
| 中度昏迷 | 眼球固定，深反射存在 |
| 重度昏迷 | 深反射消失 |

口诀：呼之不应轻，眼球固定中，深反消失重

## 面容

| 面容 | 特征 |
|------|------|
| 急性病容 | 面色潮红，兴奋不安，鼻翼翕动（刚跑完步一样） |
| 慢性病容 | 憔悴、忧虑 |
| 贫血面容 | 面色苍白 |
| 肝病面容 | 褐色色素沉着 |
| 肾病面容 | 颜面部水肿 |
| 甲亢面容 | 眼凸、惊愕、烦躁易怒 |
| 二尖瓣面容 | 双颊紫红、口唇发绀 |
| 伤寒面容 | 淡漠（脑子烧坏了无欲无求） |
| 破伤风/苦笑面容 | 牙关紧闭、面肌痉挛似苦笑 |
| 满月面容 | 脸圆、皮肤红、痤疮+胡须 |
