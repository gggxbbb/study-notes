# 体格检查

## 基本方法

| 方法 | 要点 |
|:---|:---|
| **望诊** | 视观察，自然光下，按序全面 |
| **触诊** | 浅触诊（腹部/压痛）、深触诊（脏器/包块） |
| **叩诊** | 清音/浊音/实音/鼓音，间接叩诊为主 |
| **听诊** | 直接/间接听诊，注意环境安静 |

## 一般检查

### 生命体征
- **体温**：口36.3~37.2℃ / 腋36.0~37.0℃ / 肛36.5~37.7℃
- **脉搏**：60~100次/分
- **呼吸**：12~20次/分
- **血压**：<120/<80 mmHg（理想）；≥140/≥90 mmHg（高血压）

### 皮肤
弹性、颜色（苍白/黄染/发绀）、皮疹、出血点（瘀点/紫癜/瘀斑）、蜘蛛痣、水肿

### 淋巴结
头颈部/腋窝/滑车上/腹股沟顺序触诊：大小、质地、活动度、压痛、融合

## 头部 → 颈部
- **眼**：结膜（苍白/充血）、巩膜（黄染）、瞳孔（大小/对光反射）
- **口咽**：扁桃体肿大分度（I°<腭弓、II°超腭弓、III°近中线）
- **甲状腺**：视诊+触诊（峡部/侧叶），肿大分度（I°<拇指、II°<拳头、III°>拳头）
- **颈静脉**：充盈/怒张（右心衰）、气管偏移

## 胸部

### 肺
- **望**：呼吸类型、节律；桶状胸（COPD）
- **触**：语颤↑（实变）↓（积液/气胸）；胸膜摩擦感
- **叩**：清音→浊音（实变）→鼓音（气胸）→过清音（肺气肿）
- **听**：肺泡呼吸音、支气管呼吸音、啰音（干/湿啰音）；语音共振

### 心脏
- **望**：心尖搏动位置（左第5肋间锁骨中线内0.5~1cm）
- **触**：震颤（器质性心脏病标志）
- **叩**：心浊音界（左界/右界）
- **听**：心率/心律、心音（S1/S2鉴别）、杂音（部位/时期/性质/传导）、心包摩擦音

### 血管
- 脉搏（水冲脉/奇脉/交替脉）
- 周围血管征（枪击音、Duroziez征、毛细血管搏动）

## 腹部
- **望**：外形（膨隆/凹陷）、腹壁静脉（曲张→门脉高压/下腔静脉阻塞）
- **听**：肠鸣音（活跃/亢进/减弱/消失）、血管杂音
- **叩**：移动性浊音（腹水≥1000ml）、肝浊音界
- **触**：腹壁紧张度、压痛/反跳痛、脏器（肝/脾/肾）、腹部包块

## 脊柱四肢
- 脊柱弯曲度/活动度/压痛
- 杵状指、匙状甲、水肿

## 神经系统（重点：反射）

| 反射 | 检查方法 | 临床意义 |
|:---|:---|:---|
| **浅反射** | | |
| 腹壁反射 | 划腹壁两侧 | ↓→锥体束/周围神经损害 |
| 提睾反射 | 划大腿内侧 | ↓→L1-2损害 |
| 跖反射 | 划足底外侧 | 正常→足趾跖屈 |
| **深反射** | | |
| 肱二头肌反射 | 敲肱二头肌腱 | C5-6 |
| 膝腱反射 | 敲髌韧带 | L2-4 |
| 跟腱反射 | 敲跟腱 | S1-2 |
| **病理反射** | | |
| Babinski征 | 划足底外侧 | 阳性→锥体束损害 |
| Hoffmann征 | 弹中指末节 | 阳性→颈髓/锥体束损害 |
| **脑膜刺激征** | | |
| 颈项强直、Kernig征、Brudzinski征 | | 脑膜炎/蛛网膜下腔出血 |
