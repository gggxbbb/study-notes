# 水电解质代谢紊乱

## 脱水类型鉴别

- 低渗：不口渴，失钠表现（手脚麻木、恶心呕吐），尿比重低
- 等渗：不口渴，恶心呕吐乏力等
- 高渗：口渴，眼窝凹陷，尿比重高
- 尿比重正常值：1.015-1.025
- 口渴是高渗的专利，低渗等渗不口渴

## 钾代谢

- 低钾：软瘫腹胀肠麻痹，心率快，反常酸尿。除了心率快，其余兴奋性全降低
- 高钾：皮肤苍白
- 急性肾衰时：高钾、高镁、高磷（"贾玲美高？"）；低钠、低氯、低钙（"那概率低"）

## 酸碱失衡判断

三步法：

1. 一看 PaCO₂（35-45）：>45 呼酸，<35 呼碱
2. 二看 HCO₃⁻（22-27）和 BE（-3 ~ +3）：> 代碱，< 代酸
3. 三看 pH（7.35-7.45）：不在正常范围即失代偿

- 低钾低氯性碱中毒，反常性酸性尿
- 高钾高氯性酸中毒，反常性碱性尿
- 呕吐宿食导致低钾血症（呕吐物有大量胃酸，酸少了就容易碱中毒）
